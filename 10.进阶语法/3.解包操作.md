# 解包操作

## 1. 知识点定义

**解包（Unpacking）** 是将可迭代对象中的元素分配给多个变量的操作。Python 提供了 `*` 和 `**` 两种解包运算符，让代码更简洁优雅。

> 小白理解：
> 解包就像拆快递——一个盒子里有多个东西，你一次性把它们拿出来分别放好。不用一个一个地取，一句代码全部到位。

它主要解决的问题是：
- 一行代码完成多变量赋值
- 灵活提取序列中的部分元素
- 合并字典和列表
- 函数参数的灵活传递

---

## 2. 语法格式

### 2.1 序列解包（基础）

```python
# 列表/元组解包
a, b, c = [1, 2, 3]
x, y = 10, 20              # 等价于 (10, 20)

# 交换变量（Python 独有！）
a, b = b, a

# 遍历时解包
for i, val in enumerate(['a', 'b', 'c']):
    print(i, val)

# 字典遍历
for k, v in {"name": "Alice", "age": 25}.items():
    print(f"{k} = {v}")
```

### 2.2 星号解包 `*` —— 吸收多个元素

```python
# 在开头：first 拿第一个，rest 拿剩下的
first, *rest = [1, 2, 3, 4, 5]
print(first)  # 1
print(rest)   # [2, 3, 4, 5]

# 在结尾：last 拿最后一个，head 拿前面的
*head, last = [1, 2, 3, 4, 5]
print(head)  # [1, 2, 3, 4]
print(last)  # 5

# 在中间：两头各取一个，中间全归 middle
first, *middle, last = [1, 2, 3, 4, 5]
print(first)   # 1
print(middle)  # [2, 3, 4]
print(last)    # 5

# 丢弃不关心的值（约定用 _）
a, *_ = [1, 2, 3, 4]       # 只要第一个
first, *_, last = range(10) # 只要头尾
```

### 2.3 字典解包 `**` —— 合并字典

```python
d1 = {"a": 1, "b": 2}
d2 = {"c": 3, "d": 4}
d3 = {**d1, **d2}      # {'a': 1, 'b': 2, 'c': 3, 'd': 4}

# 覆盖：后者覆盖前者
d1 = {"a": 1, "b": 2}
d2 = {"b": 999, "c": 3}
d3 = {**d1, **d2}      # {'a': 1, 'b': 999, 'c': 3}

# 添加/覆盖单个键
config = {"host": "localhost", "port": 8080}
new_config = {**config, "port": 9090, "debug": True}
```

### 2.4 函数参数解包

```python
# *args：接收任意数量位置参数（打包成元组）
def func(*args):
    print(args)           # (1, 2, 3)

# **kwargs：接收任意数量关键字参数（打包成字典）
def func(**kwargs):
    print(kwargs)         # {'a': 1, 'b': 2}

# 调用时解包
def add(a, b, c):
    return a + b + c

nums = [1, 2, 3]
print(add(*nums))         # 6 —— 等价于 add(1, 2, 3)

kwargs = {"a": 1, "b": 2, "c": 3}
print(add(**kwargs))      # 6 —— 等价于 add(a=1, b=2, c=3)
```

### 2.5 字面量中的解包

```python
# 列表/元组/集合字面量中使用 *
[*iterable1, *iterable2]   # 合并列表
{*set1, *set2}             # 合并集合

# 字典字面量中使用 **
{**d1, **d2}               # 合并字典
```

## 3. 重点说明

### 3.1 `*` 的位置规则

- `*` 在一个赋值语句中**只能出现一次**（`a, *b, *c = ...` 是语法错误）
- `*` 吸收不到元素时会是**空列表**（不是 None，不会报错）

```python
first, *rest = [1]
print(rest)  # [] —— 空列表
```

### 3.2 `*args` 和 `**kwargs` 的参数顺序

```python
def func(a, b, *args, c=10, **kwargs):
    pass

# 参数顺序：必选参数 → *args → 关键字参数 → **kwargs
```

### 3.3 字典解包 `{**d1, **d2}` 是浅合并

```python
d1 = {"db": {"host": "a", "port": 1}}
d2 = {"db": {"port": 2}}
merged = {**d1, **d2}
print(merged)  # {'db': {'port': 2}} —— host 丢失了！
```

> `{**d1, **d2}` 只合并第一层，内层嵌套字典会被整个替换。

## 4. Demo 样例

```python
# ========== 1. 序列解包 ==========
print("=" * 40)
print("1. 序列解包")

a, b, c = [1, 2, 3]
print(f"a={a}, b={b}, c={c}")

# Python 一行交换变量（不需要临时变量）
x, y = 10, 20
x, y = y, x
print(f"交换后: x={x}, y={y}")

# ========== 2. 星号解包 ==========
print("\n" + "=" * 40)
print("2. 星号解包 *")

data = [1, 2, 3, 4, 5]

first, *rest = data
print(f"first={first}, rest={rest}")     # first=1, rest=[2, 3, 4, 5]

*head, last = data
print(f"head={head}, last={last}")       # head=[1, 2, 3, 4], last=5

first, *middle, last = data
print(f"first={first}, middle={middle}, last={last}")

# 丢弃不关心的值
a, *_ = [10, 20, 30, 40]
print(f"只要第一个: {a}")                 # 10

# ========== 3. 字典解包 ==========
print("\n" + "=" * 40)
print("3. 字典解包 **")

base = {"host": "localhost", "port": 8080, "debug": False}
user = {"port": 9090, "debug": True}

# 合并配置（后者覆盖前者）
final = {**base, **user}
print(f"合并后: {final}")
# {'host': 'localhost', 'port': 9090, 'debug': True}

# ========== 4. 函数参数解包 ==========
print("\n" + "=" * 40)
print("4. 函数参数解包")

def greet(greeting, name):
    return f"{greeting}, {name}!"

# 用 * 解包列表为位置参数
args = ["你好", "张三"]
print(greet(*args))                      # 你好, 张三!

# 用 ** 解包字典为关键字参数
kwargs = {"greeting": "Hello", "name": "World"}
print(greet(**kwargs))                   # Hello, World!

# ========== 5. 实际场景 ==========
print("\n" + "=" * 40)
print("5. 实际场景")

# 场景1：拆分路径
path = "/home/user/data/file.txt"
*dirs, filename = path.split("/")
print(f"目录: {dirs}, 文件名: {filename}")

# 场景2：合并多个列表
a, b, c = [1, 2], [3, 4], [5, 6]
combined = [*a, *b, *c]
print(f"合并列表: {combined}")            # [1, 2, 3, 4, 5, 6]

# 场景3：函数参数转发（装饰器中常用）
def log_call(func):
    def wrapper(*args, **kwargs):
        print(f"调用 {func.__name__}, args={args}, kwargs={kwargs}")
        return func(*args, **kwargs)       # 完整转发
    return wrapper

print("\n解包操作演示完成！")
```

**运行结果：**

```

交换后: x=20, y=10

========================================

head=[1, 2, 3, 4], last=5
first=1, middle=[2, 3, 4], last=5
只要第一个: 10

========================================
3. 字典解包 **
合并后: {'host': 'localhost', 'port': 9090, 'debug': True}

========================================
4. 函数参数解包

Hello, World!

========================================
5. 实际场景

解包操作演示完成！
```

## 5. 典型例题（附带答案）

### 例题 1

**题目：**

给一行 CSV 数据 `"Alice,25,Engineer,NY,US"`，用解包提取名字和年龄，其余用 `*rest` 收集。

**答案：**

```python
line = "Alice,25,Engineer,NY,US"
name, age, *rest = line.split(",")

print(f"姓名: {name}")    # Alice
print(f"年龄: {age}")     # 25
print(f"其他: {rest}")    # ['Engineer', 'NY', 'US']
```

### 例题 2

**题目：**

有三个配置字典，按优先级从低到高合并（`defaults < env < user`），验证覆盖行为。

**答案：**

```python
defaults = {"host": "0.0.0.0", "port": 80, "debug": False, "cache": True}
env_config = {"port": 8080, "debug": True}
user_config = {"host": "myhost.com"}

# 优先级: user > env > defaults（后者覆盖前者）
final = {**defaults, **env_config, **user_config}
print(final)
# {'host': 'myhost.com', 'port': 8080, 'debug': True, 'cache': True}
```

## 6. 易错点

### 易错点 1：星号解包变量数量不足

错误写法：
```python
first, *middle, last = [1]   # ValueError: not enough values to unpack
```

错误原因：`first` 和 `last` 各需要 1 个值，至少需要 2 个元素。

正确写法：
```python
first, *middle = [1]   # OK，middle = []
```

### 易错点 2：`**` 只能解包 mapping 对象

错误写法：
```python
lst = [("a", 1), ("b", 2)]
d = {**lst}   # TypeError: 列表不能 ** 解包
```

错误原因：`**` 只能用于字典或其他 mapping 类型。

正确写法：
```python
d = {**dict(lst)}   # 先转为字典再解包
```

### 易错点 3：`{**d1, **d2}` 内层不合并

错误写法：
```python
d1 = {"db": {"host": "a", "port": 1}}
d2 = {"db": {"port": 2}}
merged = {**d1, **d2}  # {'db': {'port': 2}} —— host 丢了！
```

错误原因：`{**d1, **d2}` 是浅合并，嵌套字典被整个替换。

正确写法：需要深度合并时手动递归处理或使用第三方库。

## 7. 课后练习（附带答案）

### 练习 1

**题目：**

给定 `lines = ["=== 开始 ===", "数据1", "数据2", "数据3", "=== 汇总 ==="]`，用解包提取第一行（标题）、最后一行（汇总）和中间的数据行。

**答案：**

```python
lines = ["=== 开始 ===", "数据1", "数据2", "数据3", "=== 汇总 ==="]

first, *data, last = lines
print(f"标题: {first}")
print(f"数据: {data}")
print(f"汇总: {last}")
```

### 练习 2

**题目：**

编写装饰器函数 `log_call(func)`，在调用原函数前打印参数名和值，使用 `*args` 和 `**kwargs` 完整转发。

**答案：**

```python
def log_call(func):
    def wrapper(*args, **kwargs):
        print(f"调用 {func.__name__}, args={args}, kwargs={kwargs}")
        return func(*args, **kwargs)      # 完整转发
    return wrapper

@log_call
def add(a, b):
    return a + b

result = add(3, 5)   # 输出: 调用 add, args=(3, 5), kwargs={}
print(result)        # 8
```

### 练习 3

**题目：**

用 `*` 将多个列表、元组、集合分别合并为新对象。

**答案：**

```python
list1, list2 = [1, 2], [3, 4]
combined_list = [*list1, *list2]     # [1, 2, 3, 4]

t1, t2 = (1, 2), (3, 4)
combined_tuple = (*t1, *t2)          # (1, 2, 3, 4)

s1, s2 = {1, 2}, {2, 3}
combined_set = {*s1, *s2}            # {1, 2, 3}（自动去重）
```

## 8. 答题总结

- 序列解包：`a, b, c = [1, 2, 3]`，一行赋值多个变量。
- 星号 `*` 吸收多个元素为列表，只能在一个赋值中出现一次。
- 字典 `**` 合并字典，后者覆盖前者（浅合并，内层不递归）。
- `*args` 打包位置参数为元组，`**kwargs` 打包关键字参数为字典。
- 调用时 `func(*list)` 解包列表，`func(**dict)` 解包字典。
- 不需要的值用 `_` 或 `*_` 丢弃（约定俗成）。
- `a, b = b, a` 是 Python 一行交换变量的优雅写法。
