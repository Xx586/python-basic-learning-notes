# 循环

## 1. 知识点定义

程序中经常有重复执行的操作——打印 1-100、遍历列表每个元素、读取文件每一行。**循环**让我们用少量代码表达重复逻辑，是程序效率的核心来源。

Python 有两种循环：`for`（遍历可迭代对象）和 `while`（条件驱动）。配合 `break`（跳出）、`continue`（跳过本次）、`pass`（占位）和 `range()`（生成整数序列），能实现各种循环逻辑。

> 小白理解：
> **for 循环** = 老师按名单逐个点名（你知道名单上有多少人）。**while 循环** = 等人——一直等到人来为止（你不知道要等多久）。

## 2. 语法格式

### 2.1 for 循环

```python
# 遍历列表
for fruit in ["苹果", "香蕉", "橘子"]:
    print(f"我喜欢吃{fruit}")

# 遍历字典
for key, value in student.items():
    print(f"{key}: {value}")

# 同时获取索引和值
for index, color in enumerate(["红", "绿", "蓝"], start=1):
    print(f"第{index}个: {color}")

# 同时遍历多个列表
for name, score in zip(["小明","小红"], [95, 88]):
    print(f"{name}: {score}")
```

### 2.2 while 循环

```python
# 条件为 True 就一直执行
count = 1
while count <= 5:
    print(count)
    count += 1  # 必须有改变条件的语句！

# 典型场景：用户输入验证
password = ""
while password != "123456":
    password = input("请输入密码：")
```

### 2.3 for vs while 选择指南

| 场景 | 使用 |
|------|------|
| 遍历已知集合/序列 | `for` |
| 循环次数确定 | `for i in range(n)` |
| 条件驱动（次数不确定） | `while` |
| 永久运行直到外部中断 | `while True` |

### 2.4 break、continue、pass

```python
# break：立即终止整个循环
for num in [1, 3, 5, 7, 8, 9]:
    if num == 8:
        print("找到了！")
        break  # 找到后不再继续

# continue：跳过本次，进入下一次迭代
for i in range(1, 11):
    if i % 2 == 0:
        continue  # 跳过偶数
    print(i)  # 只打印奇数

# pass：什么都不做，占位符
def future_function():
    pass  # TODO: 以后再实现
```

### 2.5 range() 函数

```python
range(5)          # 0, 1, 2, 3, 4（默认从0开始）
range(2, 6)       # 2, 3, 4, 5
range(0, 10, 2)   # 0, 2, 4, 6, 8（步长为2）
range(10, 0, -1)  # 10, 9, 8, ..., 1（倒序）
```

> `range()` 不包含 stop 值，且是惰性求值（不占内存）。

### 2.6 循环嵌套

```python
# 九九乘法表
for i in range(1, 10):
    for j in range(1, i + 1):
        print(f"{j}×{i}={i*j}", end="\t")
    print()  # 换行
```

### 2.7 for-else / while-else

```python
# else 在循环中指"没有被 break 中断"
for item in ["苹果", "香蕉"]:
    if item == "西瓜":
        print("找到了")
        break
else:
    print("没找到西瓜")  # 循环正常结束才执行
```

## 3. 重点说明

### 3.1 for 和 while 的核心区别

| for | while |
|-----|-------|
| 知道要遍历什么 | 知道什么条件停止 |
| 次数通常确定 | 次数通常不确定 |
| 底层靠迭代器 | 底层靠条件判断 |

### 3.2 range() 不包含 stop 值

```python
for i in range(1, 10):  # 实际只到 9！
    print(i)
# 想打印 1-10 用 range(1, 11)
```

### 3.3 循环中修改正在遍历的列表

```python
# ❌ 边遍历边删除（导致跳元素）
for num in numbers:
    if num % 2 == 0:
        numbers.remove(num)

# ✅ 遍历副本或使用列表推导式
numbers = [num for num in numbers if num % 2 != 0]
```

## 4. Demo 样例

```python
# ===== 1. for 循环遍历 =====
fruits = ["苹果", "香蕉", "橘子"]
for i, fruit in enumerate(fruits, 1):
    print(f"{i}. {fruit}")

# ===== 2. break 演示 =====
target = 8
for num in [1, 3, 5, 7, 8, 9, 11]:
    if num == target:
        print(f"找到了 {target}！")
        break

# ===== 3. continue 演示 =====
for i in range(1, 11):
    if i % 2 == 0:
        continue
    print(i, end=" ")  # 只打印奇数: 1 3 5 7 9
print()

# ===== 4. while 循环：猜数字 =====
import random
answer = random.randint(1, 10)
guess = 0
attempts = 0
while guess != answer and attempts < 5:
    guess = int(input("猜 1-10: "))
    attempts += 1
    if guess < answer:    print("太小")
    elif guess > answer:  print("太大")
if guess == answer:
    print(f"猜对了！用了 {attempts} 次")
```

运行结果：

```
1. 苹果
2. 香蕉
3. 橘子
找到了 8！
1 3 5 7 9 
```

## 5. 典型例题（附带答案）

### 例题 1

题目：

用 for 循环计算 1-100 所有整数的和。

答案：

```python
total = 0
for i in range(1, 101):
    total += i
print(f"1-100 的和: {total}")  # 5050
```

### 例题 2

题目：

用 while 循环生成斐波那契数列前 20 项（1, 1, 2, 3, 5, 8, ...）。

答案：

```python
n, a, b = 20, 1, 1
count = 0
while count < n:
    print(a, end=" ")
    a, b = b, a + b
    count += 1
```

## 6. 易错点

### 易错点 1：无限循环

错误写法：`while` 循环中忘记更新条件变量。

```python
count = 0
while count < 5:
    print(count)
    # 忘了 count += 1 → 死循环！
```

### 易错点 2：循环中修改正在遍历的列表

边遍历边删除会导致元素被跳过或报错。正确做法：遍历副本 `for item in list[:]` 或列表推导式。

### 易错点 3：range() 不包含 stop

`range(1, 10)` 只生成 1-9，不是 1-10。要包含 10 用 `range(1, 11)`。

### 易错点 4：缩进导致逻辑错误

```python
total = 0
for i in range(1, 6):
    total += i
    print(total)  # 每次循环都打印（在循环内）
# 如果只想最后打印一次，print 应与 for 对齐
```

### 易错点 5：pass 和 continue 混淆

- `pass`：什么都不做，继续往后执行
- `continue`：跳过本次循环剩余代码，进入下一次迭代

### 练习 1

打印九九乘法表。

```python
for i in range(1, 10):

```

### 练习 2

题目：

猜数字游戏：随机生成 1-100，用户有 7 次机会猜。

答案：

```python
import random
target = random.randint(1, 100)
for attempt in range(1, 8):
    guess = int(input(f"第{attempt}次猜: "))
    if guess == target:
        print(f"对了！用了{attempt}次")
        break
    elif guess < target: print("太小")
    else: print("太大")
else:
    print(f"答案是 {target}")
```

### 练习 3

题目：

统计字符串中每个字符出现次数（不区分大小写，忽略空格），用字典存储。

答案：

```python
text = input("输入文字: ").lower()
char_count = {}
for char in text:
    if char == " ": continue
    char_count[char] = char_count.get(char, 0) + 1
for char, count in sorted(char_count.items(), key=lambda x: x[1], reverse=True):
    print(f"'{char}': {count}次")
```

## 8. 答题总结

- for 用于遍历已知集合，while 用于条件驱动的循环。
- `break` 跳出整个循环，`continue` 跳过本次迭代，`pass` 是占位符（什么都不做）。
- `range(start, stop, step)` 生成整数序列，惰性求值，不占内存。注意不包含 stop 值。
- 循环中不要边遍历边修改列表——遍历副本或使用列表推导式。
- `for-else` 中 else 表示"循环正常结束，没有被 break 中断"。
- 避免死循环：while 循环中确保条件变量最终会变成 False。
