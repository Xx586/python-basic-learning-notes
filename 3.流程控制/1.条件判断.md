# 条件判断

## 1. 知识点定义

程序默认按顺序从上到下逐行执行。但现实中我们经常需要**根据不同情况执行不同的代码**：登录时密码正确跳转首页、错误提示重试；根据分数判断等级；根据用户选择走不同分支。

**条件判断**让程序拥有了"决策能力"，是程序智能化的基础。

> 小白理解：
> 条件判断就像过马路看红绿灯——绿灯走（if）、红灯停（else）、黄灯等一等（elif）。程序也是一样，根据不同条件"决定"接下来做什么。

## 2. 语法格式

### 2.1 if / elif / else 基本结构

```python
# 最简单的 if 语句
age = 20
if age >= 18:
    print("成年了")  # 缩进 4 个空格，表示属于 if 的代码块

# if-else：二选一
age = 16
if age >= 18:
    print("成年")
else:
    print("未成年")

# if-elif-else：多选一
score = 85
if score >= 90:
    print("优秀")
elif score >= 80:
    print("良好")
elif score >= 70:
    print("中等")
elif score >= 60:
    print("及格")
else:
    print("不及格")
```

### 2.2 关键规则

| 规则 | 说明 |
|------|------|
| 条件后必须有冒号 `:` | `if x > 0:` 不能漏冒号 |
| 代码块必须缩进 | 同一个块缩进空格数必须一致（推荐 4 空格） |
| elif 可以有多个 | 只能有一个 if 和一个可选的 else |
| 自上而下匹配 | 一旦条件为 True，执行后跳出整个结构 |

### 2.3 嵌套 if

```python
has_ticket = True
age = 8
if has_ticket:
    if age < 12:
        print("儿童票，免费")
    elif age < 18:
        print("学生票，半价")
    else:
        print("全票")
else:
    print("请先购票")
```

> 嵌套不宜超过 3 层，否则难读。可将内层逻辑抽取为函数。

### 2.4 三元表达式（条件表达式）

```python
# 语法：值1 if 条件 else 值2
result = "成年" if age >= 18 else "未成年"
max_value = a if a > b else b
```

### 2.5 match-case（Python 3.10+，了解即可）

```python
code = 200
match code:
    case 200:
        print("成功")
    case 404:
        print("未找到")
    case _:           # _ 是通配符，相当于 default
        print("未知状态码")

# 解构匹配
point = (0, 5)
match point:
    case (0, 0):   print("原点")
    case (0, y):   print(f"Y轴上，y={y}")
    case (x, y):   print(f"坐标({x},{y})")
```

## 3. 重点说明

### 3.1 真值测试：哪些值被视为 False

| 假值 | 类型 |
|------|------|
| `False`、`None`、`0`、`0.0` | 基本类型 |
| `""`、`[]`、`()`、`{}`、`set()` | 空容器 |

**除此之外所有值都是 True。**

```python
# 利用真值测试写简洁代码
items = []
if items:             # ✅ 简洁，等价于 if len(items) > 0
    print("有数据")
if not username:      # ✅ 检查是否为空
    print("用户名不能为空")
```

### 3.2 if-elif 的匹配顺序很重要

```python
score = 85
# ❌ 错误顺序：永远不会到达 "良好"
if score >= 60:
    print("及格")
elif score >= 80:
    print("良好")  # 永远不会执行

# ✅ 正确顺序：从高到低排列
if score >= 90:
    print("优秀")
elif score >= 80:
    print("良好")
```

## 4. Demo 样例

```python
# ===== 1. 成绩等级判断 =====
score = 85
if score >= 90:
    grade = "优秀"
elif score >= 80:
    grade = "良好"
elif score >= 70:
    grade = "中等"
elif score >= 60:
    grade = "及格"
else:
    grade = "不及格"

# ===== 2. 用户登录模拟 =====
username = "admin"
password = "123456"
input_user = "admin"
input_pass = "123456"

    print("登录成功！")
elif input_user != username:
    print("用户名不存在")
else:
    print("密码错误")

# ===== 3. 真值测试 =====
data = []
if data:
    print(f"有 {len(data)} 条数据")
else:
    print("数据为空")
```

运行结果：

```
分数 85，等级: 良好
登录成功！

```

## 5. 典型例题（附带答案）

### 例题 1

题目：

输入三个数，用条件判断找出最大值（不用 `max()` 函数）。

答案：

```python
a, b, c = 15, 8, 22
if a >= b and a >= c:
    max_num = a
elif b >= a and b >= c:
    max_num = b
else:
    max_num = c
print(f"最大值: {max_num}")  # 22
```

### 例题 2

题目：

输入年份，判断是否为闰年。规则：能被 4 整除但不能被 100 整除，或能被 400 整除。

答案：

```python
year = 2024
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print(f"{year} 是闰年")
else:
    print(f"{year} 不是闰年")
```

## 6. 易错点

### 易错点 1：`=` 和 `==` 混淆

错误写法：`if age = 18:`（赋值，SyntaxError！）
正确写法：`if age == 18:`（比较）

### 易错点 2：少写冒号

错误写法：`if score >= 60`（少了 `:`）
正确写法：`if score >= 60:`

### 易错点 3：缩进不一致

```python
if age >= 18:
    print("成年")
  print("这句话缩进不对")  # IndentationError!
```

正确：同一代码块缩进必须完全一致（都 4 个空格）。

### 易错点 4：elif 拼写错误

Python 中是 `elif`，不是 `else if`（那是 C/Java 的写法）。

### 易错点 5：浮点数比较

```python
if 0.1 + 0.2 == 0.3:  # False！精度问题
    pass
# 正确：用 math.isclose() 或容差比较
```

## 7. 课后练习（附带答案）

### 练习 1

题目：

输入分数，输出等级（>=90 优秀，>=80 良好，>=70 中等，>=60 及格，<60 不及格）。

答案：

```python
score = float(input("请输入分数: "))
if score >= 90:    grade = "优秀"
elif score >= 80:  grade = "良好"
elif score >= 70:  grade = "中等"
elif score >= 60:  grade = "及格"
else:              grade = "不及格"

```

### 练习 2

题目：

使用 match-case 编写：输入数字 1-7，输出星期名称。

答案：

```python
day = int(input("输入数字 1-7: "))
match day:
    case 1: print("星期一")
    case 2: print("星期二")
    case 3: print("星期三")
    case 4: print("星期四")
    case 5: print("星期五")
    case 6: print("星期六")
    case 7: print("星期日")
    case _: print("无效输入")
```

## 8. 答题总结

- 条件判断让程序拥有"决策能力"，关键语法：`if` / `elif` / `else`。
- 条件表达式后必须有冒号 `:`，代码块必须缩进（推荐 4 空格）。
- Python 中空容器、0、None、False 在条件中为假值，可直接用于判断。
- `=` 是赋值，`==` 是比较！if 条件中混淆二者是新手最常见错误。
- elif 条件从高到低排列，避免逻辑错误。
- match-case（3.10+）是 if-elif 的替代方案，适合多值匹配场景。
- 浮点数不要直接用 `==` 比较，用 `math.isclose()`。
