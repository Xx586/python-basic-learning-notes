# 简单菜单程序

## 1. 知识点定义

**简单菜单程序**是综合运用**顺序结构、条件判断和循环**编写的交互式控制台程序。它通常显示一个选项列表（菜单），让用户通过输入数字选择功能，反复执行直到用户选择退出。

它主要解决的问题是：
- 把之前学的顺序、条件、循环三个知识点组合起来
- 学习如何编写真正的"交互式程序"
- 体验一个完整小程序的编写流程

在实际写 Python 代码时，菜单程序是最常见的命令行交互模式，很多实用工具（学生管理系统、记账本、通讯录等）都是这种结构。

> 小白理解：
> 菜单程序就像你去餐厅点菜。服务员给你菜单（显示选项），你选 1 号菜（输入数字），厨房就做那道菜（执行对应功能）。吃完后服务员又问你要不要加点别的（循环回到菜单），你说不要了（输入退出选项），就结账走人（程序结束）。

---

## 2. 语法格式

菜单程序没有固定的语法，而是一种**编程模式**，由以下结构组合而成：

```python
# 菜单程序的标准套路
while True:                        # 无限循环，让菜单反复显示
    print("===== 菜单 =====")       # 显示菜单选项
    print("1. 功能A")
    print("2. 功能B")
    print("3. 退出")
    
    choice = input("请选择：")       # 获取用户选择
    
    if choice == "1":               # 根据选择执行不同功能
        功能A的代码
    elif choice == "2":
        功能B的代码
    elif choice == "3":
        print("再见！")
        break                       # 跳出循环，结束程序
    else:
        print("输入有误，请重新选择")  # 处理无效输入
```

说明：
- `while True`：让菜单反复出现，用户选"退出"时才停止。
- `print`：显示菜单选项给用户看。
- `input`：接收用户的选项编号。
- `if-elif-else`：根据用户选择走不同的分支。
- `break`：退出循环，结束程序。

## 3. 重点说明

### 3.1 菜单程序的三大组成部分

| 组成部分 | 使用的知识点 | 作用 |
|----------|-------------|------|
| **显示菜单** | `print` + 顺序结构 | 把选项展示给用户 |
| **接收选择** | `input` | 获取用户输入的选项编号 |
| **执行功能** | `if-elif-else` + `break` | 根据选择跳转到对应功能 |

### 3.2 为什么需要 while True？

```python
# 不用循环：菜单只出现一次，选完就没了
print("1. 功能A")
choice = input("请选择：")
# 用完一次就结束，用户想继续用怎么办？

# 用 while True：菜单反复出现
while True:
    print("1. 功能A")
    choice = input("请选择：")
    # 执行完后，循环回到开头，再次显示菜单
    # 只有用户选"退出"时 break 才结束
```

### 3.3 注意 input 返回的是字符串

```python
choice = input("请选择：")  # 用户输入 1，choice 是 "1"（字符串）
# 判断时要用字符串比较
if choice == "1":          # ✅ 正确：和字符串 "1" 比较
    ...
# if choice == 1:          # ❌ 错误：和整数 1 比较，永远不成立
```

### 3.4 菜单程序是综合练习

写一个菜单程序会把前几章学的知识点全部用上：

| 章节 | 知识点 | 在菜单程序中的使用 |
|------|--------|-------------------|
| 第 1 章 | `print`、`input`、变量 | 显示菜单、获取输入、保存数据 |
| 第 2 章 | 数据类型、运算符 | 存储选项编号、比较判断 |
| 第 3 章 | `if-elif-else`、`while`、`break` | 条件分支、循环控制 |

## 4. Demo 样例

### Demo 1：最简单的三选项菜单

```python
# 一个最简单的菜单程序：问候、计算、退出

while True:                                   # 无限循环，反复显示菜单
    print("\n========== 主菜单 ==========")     # \n 是换行，让输出更美观
    print("1. 打招呼")                         # 选项1
    print("2. 计算加法")                       # 选项2
    print("3. 退出程序")                       # 选项3：退出
    print("================================")
    
    choice = input("请输入你的选择（1/2/3）：")  # 获取用户输入的选项编号
    
    if choice == "1":                          # 用户选了"打招呼"
        name = input("你叫什么名字？")          # 获取名字
        print("你好，", name, "！欢迎使用本程序。")  # 打招呼
        
    elif choice == "2":                        # 用户选了"计算加法"
        a = int(input("请输入第一个数："))      # 获取第一个数
        b = int(input("请输入第二个数："))      # 获取第二个数
        print(a, "+", b, "=", a + b)           # 输出计算结果
        
    elif choice == "3":                        # 用户选了"退出"
        print("谢谢使用，再见！")              # 告别
        break                                  # 跳出 while 循环，程序结束
        
    else:                                      # 用户输入了 1/2/3 以外的内容
        print("输入有误，请输入 1、2 或 3！")   # 提示重新输入
```

运行结果：

### Demo 2：带状态记录的计算器菜单

```python
# 一个稍微复杂一点的菜单：计算器，能记住上一次的结果

result = 0  # 用一个变量保存计算结果，在循环外面定义

while True:
    print("\n===== 简易计算器 =====")
    print("当前结果：", result)    # 每次显示当前保存的结果
    print("1. 加法（当前结果 + 输入数）")
    print("2. 减法（当前结果 - 输入数）")
    print("3. 清零")
    print("4. 退出")
    
    choice = input("请选择操作：")
    
    if choice == "1":                          # 加法
        num = float(input("请输入要加的数："))
        result = result + num                  # 更新结果
        print("计算完毕！")
        
    elif choice == "2":                        # 减法
        num = float(input("请输入要减的数："))
        result = result - num                  # 更新结果
        print("计算完毕！")
        
    elif choice == "3":                        # 清零
        result = 0                             # 把结果重置为 0
        print("结果已清零！")
        
    elif choice == "4":                        # 退出
        print("最终结果：", result)
        print("再见！")
        break                                  # 结束循环
        
    else:
        print("输入错误，请输入 1-4！")
```

运行结果：

```text
===== 简易计算器 =====
当前结果： 0
1. 加法（当前结果 + 输入数）
2. 减法（当前结果 - 输入数）
3. 清零
4. 退出
请选择操作：1
请输入要加的数：100
计算完毕！

===== 简易计算器 =====
当前结果： 100.0
1. 加法（当前结果 + 输入数）
2. 减法（当前结果 - 输入数）
3. 清零
4. 退出
请选择操作：2
请输入要减的数：30
计算完毕！

===== 简易计算器 =====
当前结果： 70.0
...
请选择操作：4
最终结果： 70.0
再见！
```

> 注意：`result` 变量定义在 `while` 循环**外面**，所以才能在多次循环之间保持数据不丢失。如果放在循环里面，每次循环都会被重置为 0。

## 5. 典型例题（附带答案）

### 例题 1

题目：

请编写一个"个人名片"菜单程序，要求：

1. 菜单有 3 个选项：1.查看名片、2.修改名片、3.退出
2. 程序初始时名片信息为空
3. 选 1 时：如果名片已设置则显示，否则提示"名片信息为空"
4. 选 2 时：让用户依次输入姓名、电话、邮箱
5. 选 3 时：退出程序

答案：

```python
# 初始时名片为空
name = ""
phone = ""
email = ""

while True:
    print("\n===== 个人名片 =====")
    print("1. 查看名片")
    print("2. 修改名片")
    print("3. 退出")
    
    choice = input("请选择：")
    
    if choice == "1":
        # 查看名片：先判断是否为空
        if name == "":
            print("名片信息为空，请先设置名片！")
        else:
            print("姓名：", name)
            print("电话：", phone)
            print("邮箱：", email)
            
    elif choice == "2":
        # 修改名片：依次输入信息
        name = input("请输入姓名：")
        phone = input("请输入电话：")
        email = input("请输入邮箱：")
        print("名片设置成功！")
        
    elif choice == "3":
        print("再见！")
        break
        
    else:
        print("输入有误，请重新选择！")
```

解析：

关键点：
- 变量 `name`、`phone`、`email` 定义在循环外面，修改后能保持。
- 查看名片时先检查是否为空（`if name == ""`），这是良好的用户体验。
- `while True` 配合 `break` 实现反复显示菜单直到用户退出。

### 例题 2

题目：

以下菜单程序有一个 bug，请找出并修复：

```python
while True:
    print("1. 开始游戏")
    print("2. 退出")
    choice = input("请选择：")
    
    if choice == 1:          # ← 这里有问题
        print("游戏开始！")
    elif choice == 2:
        print("再见！")
        break
```

答案：

Bug 在第 5 行：`if choice == 1` 比较的是整数 `1`，但 `input` 返回的是字符串 `"1"`，所以这个条件永远不会成立。

修复：把 `1` 和 `2` 改成 `"1"` 和 `"2"`：

```python
if choice == "1":           # 和字符串比较
    print("游戏开始！")
elif choice == "2":
    print("再见！")
    break
```

## 6. 易错点

### 易错点 1：忘记 break 导致死循环

错误写法：

```python
while True:
    print("1. 退出")
    choice = input("请选择：")
    if choice == "1":
        print("再见！")
        # 忘了写 break！程序永远不会结束
```

错误原因：`while True` 是无限循环，只有 `break` 能跳出。忘了写 `break`，程序就停不下来。

正确写法：

```python
if choice == "1":
    print("再见！")
    break       # 必须加 break
```

### 易错点 2：用整数比较 input 返回值

错误写法：

```python
choice = input("请选择：")
if choice == 1:      # input 返回 "1"（字符串），不会等于 1（整数）
    ...
```

错误原因：`input` 返回值永远是字符串。

正确写法：

```python
choice = input("请选择：")
if choice == "1":    # 和字符串比较
    ...
```

### 易错点 3：变量放在循环里面导致数据丢失

错误写法：

```python
while True:
    total = 0         # 每次循环都重置为 0
    ...
    if choice == "1":
        total = total + 100  # total 永远从 0 开始加
```

错误原因：`total = 0` 放在循环里面，每轮循环都重新赋值，之前的数据就丢失了。

正确写法：

```python
total = 0             # 放在循环外面，只初始化一次
while True:
    ...
    if choice == "1":
        total = total + 100  # 在上次结果基础上累加
```

## 7. 课后练习（附带答案）

### 练习 1

题目：

编写一个简单的"猜数字"菜单程序。菜单有两个选项：1.开始游戏、2.退出。选 1 时随机生成一个 1-10 的数字，让用户猜一次，告诉用户猜对还是猜错。选 2 退出。

提示：使用 `import random` 后用 `random.randint(1, 10)` 生成随机数。

答案：

```python
import random  # 导入随机数模块

while True:
    print("\n===== 猜数字游戏 =====")
    print("1. 开始游戏")
    print("2. 退出")
    
    choice = input("请选择：")
    
    if choice == "1":
        answer = random.randint(1, 10)       # 随机生成 1-10 的数字
        guess = int(input("猜一个 1-10 的数字："))  # 获取用户猜测
        if guess == answer:
            print("恭喜你，猜对了！答案是", answer)
        else:
            print("很遗憾，猜错了！答案是", answer)
            
    elif choice == "2":
        print("再见！")
        break
    else:
        print("输入有误，请选 1 或 2！")
```

说明：`random.randint(1, 10)` 会随机生成 1 到 10 之间的整数（包含 1 和 10）。

### 练习 2

题目：

请为 Demo 1 的菜单程序增加第 4 个选项："4.显示历史"。每次用户选 2 做加法时，把计算记录保存到一个列表中。选 4 时显示所有历史记录。

提示：用一个列表 `history = []` 放在循环外面，每次计算后 `history.append(记录)`。

答案：

```python
history = []  # 放在循环外面，存储历史记录

while True:
    print("\n========== 主菜单 ==========")
    print("1. 打招呼")
    print("2. 计算加法")
    print("3. 显示历史")
    print("4. 退出程序")
    print("================================")
    
    choice = input("请输入你的选择（1/2/3/4）：")
    
    if choice == "1":
        name = input("你叫什么名字？")
        print("你好，", name, "！")
        
    elif choice == "2":
        a = int(input("请输入第一个数："))
        b = int(input("请输入第二个数："))
        result = a + b
        print(a, "+", b, "=", result)
        # 把本次计算记录保存到历史列表
        record = str(a) + "+" + str(b) + "=" + str(result)
        history.append(record)  # 添加到列表末尾
        print("（已保存到历史记录）")
        
    elif choice == "3":
        # 显示历史记录
        if len(history) == 0:
            print("暂无历史记录")
        else:
            print("历史记录：")
            for i, record in enumerate(history):
                print(i + 1, ".", record)
                
    elif choice == "4":
        print("谢谢使用，再见！")
        break
    else:
        print("输入有误！")
```

说明：`history` 列表放在循环外面不会被重置。`history.append()` 把新记录追加进去。

### 练习 3

题目：

请设计并编写一个"学生成绩管理"菜单程序，包含以下功能：

1. 添加学生成绩（输入姓名和三科成绩）
2. 查看所有学生成绩
3. 计算平均分
4. 退出

提示：用一个字典列表存储学生信息，如 `students = [{"name": "张三", "score": 85}, ...]`。

答案：

```python
students = []  # 存储所有学生信息，放在循环外面

while True:
    print("\n===== 学生成绩管理 =====")
    print("1. 添加学生成绩")
    print("2. 查看所有成绩")
    print("3. 计算平均分")
    print("4. 退出")
    
    choice = input("请选择：")
    
    if choice == "1":
        # 添加学生
        name = input("请输入学生姓名：")
        chinese = float(input("语文成绩："))
        math = float(input("数学成绩："))
        english = float(input("英语成绩："))
        # 把学生信息保存为字典
        total = chinese + math + english
        student = {
            "name": name,
            "chinese": chinese,
            "math": math,
            "english": english,
            "total": total
        }
        students.append(student)  # 添加到列表
        print("添加成功！")
        
    elif choice == "2":
        # 查看所有成绩
        if len(students) == 0:
            print("还没有学生记录")
        else:
            print("姓名\t语文\t数学\t英语\t总分")
            for stu in students:
                print(stu["name"], "\t", 
                      stu["chinese"], "\t", 
                      stu["math"], "\t", 
                      stu["english"], "\t", 
                      stu["total"])
                      
    elif choice == "3":
        # 计算平均分
        if len(students) == 0:
            print("还没有学生记录")
        else:
            all_total = 0
            for stu in students:
                all_total += stu["total"]
            average = all_total / len(students)
            print("共有", len(students), "名学生")
            print("总分平均分：", average)
            
    elif choice == "4":
        print("再见！")
        break
    else:
        print("输入有误，请选 1-4！")
```

说明：这道题综合了变量、列表、字典、循环和条件判断多个知识点，是一个完整的实战练习。

## 8. 答题总结

- 简单菜单程序是**顺序+条件+循环**的综合应用，是控制台交互程序的基本模式。
- 核心套路：`while True`（循环显示菜单） + `print`（显示选项） + `input`（获取选择） + `if-elif-else`（分支执行） + `break`（退出）。
- 需要持久保存的数据（如历史记录、累加结果）放在循环**外面**定义。
- `input` 返回值是字符串，判断选项时必须和字符串比较（`if choice == "1"`）。
- 别忘了 `break`！否则 `while True` 永远不会停止。
- 好的菜单程序要考虑用户体验：输入无效时友好提示、数据为空时给出说明。
