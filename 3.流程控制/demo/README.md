# 3.流程控制 Demo 学习说明

## 1. 推荐学习顺序

按照理论文档顺序学习：

1. 先阅读知识点的 .md 理论文档
2. 再查看其中的 Demo 样例
3. 理解代码注释
4. 复制或手敲 Demo 代码运行
5. 修改变量、条件、数据，观察结果变化
6. 再进入下一个知识点

## 3. 如何运行 Demo

如果文档中有代码块，可以复制到当前章节的 demo/ 文件夹中保存为 .py 文件运行。

运行方式：

```bash
python 文件名.py
```

如果电脑同时安装了 Python 2 和 Python 3，则使用：

```bash
python3 文件名.py
```

## 4. 学习建议

- 先看理论，再看 Demo
- 不要只复制代码，建议手敲一遍
- 每运行一次 Demo，都要观察输出结果
- 可以修改变量值、列表内容、条件表达式等，看看结果如何变化
- 遇到报错时，优先检查：拼写、缩进、中文符号、变量名和文件路径
- 完成当前知识点的练习和例题后再进入下一个
