# 闭包

## 1. 知识点定义

**闭包（Closure）** 是指内层函数引用了外层函数的变量，并且外层函数返回了内层函数。闭包让内层函数"记住"外层函数的变量，即使外层函数已经执行完毕。

满足三个条件就是闭包：① 有嵌套函数；② 内层函数**引用了**外层函数的变量；③ 外层函数将内层函数**作为返回值返回**。

> 小白理解：
> 你去露营搭好帐篷（外层函数），帐篷里有睡袋（外层变量）。你离开了（外层执行完），但帐篷还在原地。下次回来，睡袋还在等你。闭包就是让内层函数"记住"创建时的环境。

## 2. 语法格式

```python
def outer():
    msg = "Hello"         # 外层函数的变量
    def inner():          # 内层函数
        print(msg)        # 引用了外层的 msg
    return inner          # 返回内层函数本身

f = outer()               # outer() 执行完毕，但 msg 没被销毁
f()                       # 输出: Hello（inner 记住了 msg）

# 更实用的例子：创建"记忆"参数的函数
def make_multiplier(n):
    def multiply(x):
        return x * n      # n 被闭包记住
    return multiply

double = make_multiplier(2)
print(double(5))          # 10
```

## 3. 重点说明

### 3.1 闭包的核心价值

闭包实现了**数据封装**和**状态保持**——不需要全局变量、不需要类，就能让函数"记住"状态。

```python
def make_counter():
    count = 0                   # 私有变量
    def counter():
        nonlocal count          # 修改外层变量
        count += 1
        return count
    return counter

c = make_counter()
print(c(), c(), c())  # 1 2 3
```

### 3.2 nonlocal 的作用

在闭包内对外层变量**赋值**，必须用 `nonlocal` 声明。**只读取不需要**。

### 3.3 闭包是装饰器的基础

装饰器本质上就是一个接收函数、返回新函数的闭包。理解闭包是学习装饰器的前提。

## 4. Demo 样例

```python
# ===== 1. 基本闭包：配置化函数 =====
def make_greeting(greeting):
    """创建个性化的问候函数"""
    def greet(name):
        return f"{greeting}, {name}!"
    return greet

say_hi = make_greeting("你好")
say_hello = make_greeting("Hello")
print(say_hi("小明"))    # 你好, 小明!
print(say_hello("Tom"))  # Hello, Tom!

# ===== 2. 验证闭包 =====
double = make_multiplier(2)
print(double.__closure__)                      # 有值，是闭包
print(double.__closure__[0].cell_contents)     # 2（闭包记住的值）

# ===== 3. 计数器（状态保持） =====
c = make_counter()
for _ in range(5):
    print(c(), end=" ")  # 1 2 3 4 5
```

## 5. 典型例题（附带答案）

### 例题 1

题目：用闭包实现 `make_power(n)`，返回计算 x^n 的函数。

答案：

```python
def make_power(n):
    def power(x):
        return x ** n
    return power

square = make_power(2)
print(square(4))  # 16
```

### 例题 2

题目：以下代码输出什么？为什么？

```python
def outer():
    funcs = []
    for i in range(3):
        def inner():
            return i
        funcs.append(inner)
    return funcs

for f in outer():
    print(f())
```

答案：输出 `2 2 2`。闭包捕获的是变量名 `i`，循环结束后 `i=2`，所有 inner 都引用同一个 `i`。修复：`def inner(i=i): return i`。

## 6. 易错点

### 易错点 1：循环中创建闭包变量绑定问题

闭包记住的是变量名，不是值。循环中用默认参数绑定当前值。

### 易错点 2：修改外层变量忘用 nonlocal

在闭包内对外层变量赋值，必须用 `nonlocal`；只读取不需要。

### 易错点 3：闭包和嵌套函数混淆

只有引用了外层变量且被返回的才是闭包。简单嵌套不是闭包。

## 7. 课后练习（附带答案）

### 练习 1

题目：用闭包实现 `make_adder(n)`，返回加 n 的函数。

答案：`def make_adder(n): return lambda x: x + n`

### 练习 2

题目：用闭包实现一次缓存 `once(func)`，func 只执行一次。

答案：

```python
def once(func):
    result, called = None, False
    def wrapper(*args, **kwargs):
        nonlocal result, called
        if not called:
            result = func(*args, **kwargs)
            called = True
        return result
    return wrapper
```

## 8. 答题总结

- 闭包 = 内层函数 + 引用外层变量 + 外层返回内层函数。用 `.__closure__` 可验证。
- 核心价值：函数"记住"状态，无需全局变量或类。闭包是装饰器的前置知识。
- 修改外层变量需 `nonlocal`；循环中注意变量绑定问题。
- 闭包适合简单的状态保持场景；复杂状态管理建议用类。
