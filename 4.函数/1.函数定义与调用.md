# 函数定义与调用

## 1. 知识点定义

函数是组织代码、提高复用性的基本单位。把重复使用的逻辑封装成函数，需要时"喊一声"就行——一次编写，多次调用。

函数让代码：**模块化**（拆分为小问题逐一解决）、**可复用**（一次编写多次调用）、**可读性强**（通过函数名知道做什么）、**便于测试**（独立测试每个函数）。

> 小白理解：
> 函数就像微波炉的"一键加热"按钮。你不需要知道微波炉怎么产生微波，只需要按一下按钮（调用函数），它就会自动完成加热。不同的菜可能有不同的加热程序（不同的函数），你根据需要选择调用哪个。

## 2. 语法格式

### 2.1 def 定义函数

```python
def 函数名(参数列表):
    """文档字符串 (docstring)"""
    函数体
    return 返回值  # 可选
```

### 2.2 参数类型（按顺序排列）

```
必选参数 → 默认参数 → *args → **kwargs
```

**位置参数（必选）：**

```python
def introduce(name, age):
    print(f"我叫{name}，今年{age}岁")

introduce("小明", 18)  # 正确
```

**默认参数：**

```python
def greet(name, greeting="你好"):
    print(f"{greeting}，{name}！")

greet("小明")                  # 你好，小明！
greet("Alice", greeting="Hi")  # Hi，Alice！
```

**可变参数 *args（打包成元组）：**

```python
def my_sum(*args):
    return sum(args)

print(my_sum(1, 2, 3, 4))  # 10
```

**关键字参数 **kwargs（打包成字典）：**

```python
def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"  {key}: {value}")

print_info(name="小明", age=18, city="北京")
```

### 2.3 return 返回值

```python
def square(x):
    return x * x

# 多值返回（本质是返回元组）
def divide_and_remainder(a, b):
    return a // b, a % b

quotient, remainder = divide_and_remainder(10, 3)  # 解包接收
```

## 3. 重点说明

### 3.1 不要用可变对象做默认值！（重要⭐）

```python
# ❌ 错误：默认值 [] 只创建一次，多次调用共享
def add_item(item, items=[]):
    items.append(item)
    return items
print(add_item(1))  # [1]
print(add_item(2))  # [1, 2] ← bug! 上一次的 1 还在

# ✅ 正确：用 None 代替
def add_item(item, items=None):
    if items is None:
        items = []
    items.append(item)
    return items
```

### 3.2 参数传递本质

Python 参数传的是**对象引用**：
- 不可变对象（int/str/tuple）：函数内修改不影响外部
- 可变对象（list/dict）：函数内修改会影响外部

### 3.3 没有 return 默认返回 None

```python
def no_return():
    print("没有 return")
result = no_return()
print(result)  # None
```

## 4. Demo 样例

```python
# 四种参数混用完整示例
def full_demo(required, default="默认值", *args, **kwargs):
    """参数顺序：必选 → 默认 → *args → **kwargs"""
    print(f"必选参数: {required}")
    print(f"默认参数: {default}")
    print(f"可变参数 *args: {args}")
    print(f"关键字参数 **kwargs: {kwargs}")

# 调用演示
full_demo("必须传", "改一下", 1, 2, 3, name="小明", age=18)
# 输出:
# 必选参数: 必须传
# 默认参数: 改一下
# 可变参数 *args: (1, 2, 3)
# 关键字参数 **kwargs: {'name': '小明', 'age': 18}

# 可变对象参数传递演示
def modify_list(lst):
    lst.append(4)  # 修改了原对象！

my_list = [1, 2, 3]
modify_list(my_list)
print(f"函数外: {my_list}")  # [1, 2, 3, 4] — 被改变了！
```

## 5. 典型例题（附带答案）

### 例题 1

题目：

编写函数 `calculate(a, b, operator='+')`，支持加减乘除。

答案：

```python
def calculate(a, b, operator='+'):
    if operator == '+': return a + b
    elif operator == '-': return a - b
    elif operator == '*': return a * b
    elif operator == '/':
        return "错误：除数不能为0" if b == 0 else a / b
    return "不支持的运算符"

print(calculate(10, 5))       # 15
print(calculate(10, 5, '-'))  # 5
```

### 例题 2

题目：

以下代码有 bug，请找出并修复：

```python
def append_user(name, user_list=[]):
    user_list.append(name)
    return user_list
```

答案：

Bug：可变对象做默认参数。修复：

```python
def append_user(name, user_list=None):
    if user_list is None:
        user_list = []
    user_list.append(name)
    return user_list
```

## 6. 易错点

### 易错点 1：可变默认参数

`def f(lst=[])` 每次调用共享同一个列表。用 `None` + 函数内判断替代。

### 易错点 2：忘记 return

没有 return 的函数返回 None。需要返回值的函数务必写 return。

### 易错点 3：参数顺序错误

`*args` 必须在默认参数**之后**，`**kwargs` 必须在最后。
错误：`def f(*args, default=1)` → SyntaxError！

### 易错点 4：可变对象被意外修改

传入列表/字典后在函数内被修改，影响外部。需要时先拷贝 `lst.copy()`。

## 7. 课后练习（附带答案）

### 练习 1

题目：编写函数 `min_max_avg(numbers)`，返回最小值、最大值和平均值。

答案：

```python
def min_max_avg(numbers):
    return min(numbers), max(numbers), sum(numbers) / len(numbers)

print(min_max_avg([1, 2, 3, 4, 5]))  # (1, 5, 3.0)
```

### 练习 2

题目：编写 `char_count(text, *letters)`，返回 text 中所有指定字母的总出现次数。

答案：

```python
def char_count(text, *letters):
    return sum(1 for ch in text if ch in letters)

print(char_count("hello world", 'l', 'o'))  # 5
```

### 练习 3

题目：编写 `format_print(template, **kwargs)`，替换模板中的占位符。

答案：

```python
def format_print(template, **kwargs):
    print(template.format(**kwargs))

format_print("我叫{name}，今年{age}岁", name="小明", age=18)
# 我叫小明，今年18岁
```

## 8. 答题总结

- 函数用 `def` 定义，是代码复用的基本单位。`return` 返回值，无 return 返回 None。
- 参数顺序：必选 → 默认 → `*args` → `**kwargs`。
- **默认参数绝不能用可变对象**（list/dict），用 `None` + 函数内判断替代。
- `*args` 将多参数打包成元组，`**kwargs` 打包成字典。
- Python 参数传递的是对象引用：不可变对象不影外部，可变对象会影响外部。
- 多值返回本质是返回元组，可通过解包接收。
