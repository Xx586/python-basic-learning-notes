# lambda 匿名函数

## 1. 知识点定义

lambda 是一种**匿名函数**——不需要 `def` 定义、没有函数名、只包含单行表达式的函数。它的核心价值是"一次性的小函数，用完就扔"。

语法：`lambda 参数1, 参数2, ... : 表达式`

和 def 最大的区别：**只有表达式，不能有语句**（如 if、for、return）。lambda 的返回值就是表达式的结果，**不需要写 return**。

> 小白理解：
> def 定义的函数像你家里的多功能电器，有名字、可以反复用。lambda 就像一次性筷子——用完就扔，不用起名字，非常轻便。

## 2. 语法格式

```python
# def 写法
def add(x, y):
    return x + y

# lambda 写法（一行搞定）
add_lambda = lambda x, y: x + y

print(add(3, 5))        # 8
print(add_lambda(3, 5))  # 8

# 最常用场景：作为参数直接传给高阶函数
# sorted 的 key 参数
students = [("张三", 85), ("李四", 92), ("王五", 78)]
students.sort(key=lambda s: s[1])  # 按分数排序
print(students)  # [('王五', 78), ('张三', 85), ('李四', 92)]

# map + lambda
nums = [1, 2, 3, 4]
squared = list(map(lambda x: x ** 2, nums))  # [1, 4, 9, 16]

# filter + lambda
evens = list(filter(lambda x: x % 2 == 0, range(1, 11)))  # [2, 4, 6, 8, 10]
```

## 3. 重点说明

### 3.1 lambda 能做什么、不能做什么

| ✅ 可以 | ❌ 不可以 |
|---------|----------|
| 单行表达式 | 多行语句（if/for/while） |
| 数学运算、字符串操作 | 赋值语句（x = 1） |
| 条件表达式（三元） | return/break/continue |
| 作为 map/filter/sorted 的参数 | 复杂的业务逻辑 |

### 3.2 lambda vs def

| 维度 | lambda | def |
|------|--------|-----|
| 名称 | 匿名（可选赋给变量） | 有名称 |
| 行数 | 单行 | 多行 |
| 复杂度 | 简单表达式 | 任意复杂 |
| 典型用途 | sorted key, map, filter | 主要业务逻辑 |

## 4. Demo 样例

```python
# ===== 1. 基本用法 =====
square = lambda x: x ** 2
print(f"5 的平方: {square(5)}")  # 25

# ===== 2. sorted 按 key 排序 =====
words = ["banana", "apple", "Cherry", "date"]
# 按长度排序
print(sorted(words, key=lambda w: len(w)))  # ['date', 'apple', 'banana', 'Cherry']
# 按最后字母排序
print(sorted(words, key=lambda w: w[-1]))   # ['banana', 'apple', 'date', 'Cherry']

# ===== 3. map + filter 组合 =====
nums = [1, 2, 3, 4, 5, 6]
# 先过滤偶数，再乘以 10
result = list(map(lambda x: x * 10, filter(lambda x: x % 2 == 0, nums)))
print(result)  # [20, 40, 60]
# 用列表推导式更直观：
result2 = [x * 10 for x in nums if x % 2 == 0]  # 推荐！
```

## 5. 典型例题（附带答案）

### 例题 1

题目：用 lambda 和 sorted 对以下列表按绝对值排序。

```python
nums = [5, -3, 1, -8, 4, -2]
```

答案：

```python
sorted_nums = sorted(nums, key=lambda x: abs(x))
print(sorted_nums)  # [1, -2, -3, 4, 5, -8]
```

### 例题 2

题目：用 lambda 和 filter 过滤出列表中的回文串。

答案：

```python
words = ["radar", "hello", "level", "world", "madam"]
palindromes = list(filter(lambda w: w == w[::-1], words))
print(palindromes)  # ['radar', 'level', 'madam']
# 用列表推导式更推荐：
palindromes2 = [w for w in words if w == w[::-1]]
```

## 6. 易错点

### 易错点 1：lambda 不能包含语句

错误：`lambda x: if x > 0: x else -x`
正确：`lambda x: x if x > 0 else -x`（用三元表达式）

### 易错点 2：lambda 适合简单场景

过度复杂的 lambda 应该改成 def 函数，可读性更重要。

### 易错点 3：lambda 中变量捕获

lambda 捕获的是变量名，不是值。在循环中创建 lambda 要小心：

```python
# ❌ 错误：所有 lambda 都引用同一个变量 i
funcs = [lambda x: x + i for i in range(3)]
print([f(0) for f in funcs])  # [2, 2, 2]，不是 [0, 1, 2]！

# ✅ 正确：用默认参数绑定当前值
funcs = [lambda x, i=i: x + i for i in range(3)]
print([f(0) for f in funcs])  # [0, 1, 2]
```

## 7. 课后练习（附带答案）

### 练习 1

题目：用 lambda 将列表中每个元素乘以 2。

答案：`list(map(lambda x: x * 2, [1, 2, 3, 4]))  # [2, 4, 6, 8]`

### 练习 2

题目：对字典列表按 age 排序。

```python
users = [{"name": "张三", "age": 25}, {"name": "李四", "age": 18}]
```

答案：`sorted(users, key=lambda u: u["age"])`

### 练习 3

题目：过滤出 1-20 中能被 3 整除的数字。

答案：`list(filter(lambda x: x % 3 == 0, range(1, 21)))  # [3, 6, 9, 12, 15, 18]`

## 8. 答题总结

- lambda 是匿名函数，语法 `lambda 参数: 表达式`，只能写单行表达式。
- 常配合 `sorted(key=...)`、`map()`、`filter()` 使用。
- lambda 不能包含语句（if/for/return），需要时用三元表达式替代。
- 简单场景用 lambda，复杂逻辑用 def。**可读性优先**。
- 在循环中创建 lambda 要注意变量捕获问题，可用默认参数绑定。
