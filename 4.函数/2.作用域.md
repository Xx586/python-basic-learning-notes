# 作用域

## 1. 知识点定义

作用域（Scope）决定了变量在哪里可以被访问。Python 中，变量在哪个位置被赋值，决定了它的可见范围。

Python 使用 **LEGB 规则**查找变量：**L**ocal（局部）→ **E**nclosing（闭包外层）→ **G**lobal（全局）→ **B**uilt-in（内置）。

> 小白理解：
> 作用域就像公司层级——你工位上的东西（局部变量）只有你能用，部门公共区的东西（闭包变量）部门内都能用，公司大厅的东西（全局变量）全公司都能用，社会公共设施（内置函数如 `print`）任何人都能用。

## 2. 语法格式

### 2.1 局部变量

```python
def my_func():
    local_var = "我是局部变量"  # 只能在函数内部使用
    print(local_var)

my_func()
# print(local_var)  # ❌ NameError: 函数外无法访问
```

### 2.2 全局变量与 global 关键字

```python
global_var = "我是全局变量"

def my_func():
    print(global_var)  # ✅ 可以读取

def change_global():
    global global_var           # 声明要修改全局变量
    global_var = "被修改了"

change_global()
print(global_var)  # "被修改了"
```

### 2.3 nonlocal 关键字

```python
def outer():
    count = 0
    def inner():
        nonlocal count  # 声明要修改外层（非全局）变量
        count += 1
        return count
    return inner

f = outer()
print(f())  # 1
print(f())  # 2
```

### 2.4 LEGB 查找规则

```python
x = "global"          # Global

def outer():
    x = "enclosing"   # Enclosing
    def inner():
        x = "local"   # Local
        print(x)      # 优先找最近的作用域
    inner()

outer()  # 输出: "local"
```

## 3. 重点说明

### 3.1 三个关键字对比

| 关键字 | 作用 | 何时用 |
|--------|------|--------|
| 无（默认） | 读取变量 | 只要不赋值，可以读取外层变量 |
| `global` | 在函数内修改全局变量 | 需要在函数内改变全局变量的值 |
| `nonlocal` | 在嵌套函数内修改外层变量 | 闭包中修改外层函数的变量 |

### 3.2 LEGB 查找顺序

Python 查找变量时按 LEGB 顺序：先找 Local，找不到找 Enclosing，再找 Global，最后找 Built-in（如 `print`、`len`）。

```python
# Built-in 示例
print(len)  # <built-in function len> — 来自 Built-in 作用域
```

## 4. Demo 样例

```python
# ===== 1. 局部 vs 全局 =====
total = 0  # 全局变量

def add_to_total(n):
    global total        # 声明要修改全局变量
    total += n          # 修改全局变量
    local_temp = n * 2  # 局部变量，函数外不可见
    print(f"加了 {n}，当前全局 total = {total}")

add_to_total(10)  # 加了 10，当前全局 total = 10
add_to_total(5)   # 加了 5，当前全局 total = 15

# ===== 2. nonlocal 演示：计数器 =====
def make_counter():
    count = 0
    def counter():
        nonlocal count  # 修改外层函数的 count
        count += 1
        return count
    return counter

counter1 = make_counter()
print(counter1())  # 1
print(counter1())  # 2
print(counter1())  # 3
```

## 5. 典型例题（附带答案）

### 例题 1

题目：以下代码输出什么？

```python
x = 10
def func():
    x = 20
    print(x)
func()
print(x)
```

答案：`20` 和 `10`。函数内 `x=20` 创建的是局部变量，不影响全局 `x=10`。

### 例题 2

题目：用 nonlocal 实现一个累加器，每次调用在上次结果基础上加 1。

答案：

```python
def make_accumulator():
    total = 0
    def add():
        nonlocal total
        total += 1
        return total
    return add
```

## 6. 易错点

### 易错点 1：函数内修改全局变量忘记 global

```python
count = 0
def increment():
    # count += 1  # ❌ UnboundLocalError! 没声明 global
    pass
```

正确：`def increment(): global count; count += 1`

### 易错点 2：混淆 global 和 nonlocal

- `global`：修改**全局**变量（模块级别）
- `nonlocal`：修改**外层函数**的变量（嵌套函数中）

### 易错点 3：认为函数内可以随意修改全局变量

函数内可以**读取**全局变量，但**修改（赋值）**必须用 `global` 声明。

## 7. 课后练习（附带答案）

### 练习 1

题目：写出以下代码的输出。

```python
a = 1
def f():
    a = 2
    def g():
        global a
        a = 3
    g()
    print(a)
f()
print(a)
```

答案：`2` 和 `3`。`g()` 中的 `global a` 修改的是全局的 `a`，不是 `f()` 里的局部 `a`。

### 练习 2

题目：实现一个函数 `create_multiplier(n)`，返回一个函数，该函数接收参数 x，返回 x * n。

答案：

```python
def create_multiplier(n):
    def multiply(x):
        return x * n  # n 来自外层作用域（闭包）
    return multiply

double = create_multiplier(2)
print(double(5))  # 10
```

## 8. 答题总结

- 作用域决定变量的可见范围，Python 按 LEGB 顺序查找变量。
- 函数内可以读取全局变量，但**修改**必须用 `global` 声明。
- `nonlocal` 用于嵌套函数中修改外层函数的变量。
- LEGB：Local → Enclosing → Global → Built-in。
- 同名变量按就近原则：内层覆盖外层，查找不到时报 `NameError`。
