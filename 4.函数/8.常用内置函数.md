# 常用内置函数

## 1. 知识点定义

Python 提供了大量**内置函数**（built-in functions），不需要导入任何模块就能直接使用。这些函数经过 C 语言优化，效率极高，是 Python 编程的"瑞士军刀"。

> 小白理解：
> 内置函数就像手机预装的基础 App——电话、短信、相机。你不需要去应用商店下载就能用，而且经过厂商优化，性能很好。

## 2. 语法格式

### 2.1 数学相关

```python
len(seq)          # 获取序列长度
max(iterable)     # 最大值；max(a, b, c)
min(iterable)     # 最小值
sum(iterable)     # 求和；sum([1,2,3]) → 6
abs(x)            # 绝对值
round(x, n)       # 四舍五入到n位小数
pow(x, y)         # x的y次方，等价于 x**y
divmod(a, b)      # 同时返回商和余数 → (a//b, a%b)
```

### 2.2 类型判断与转换

```python
type(x)              # 获取精确类型
isinstance(x, int)   # 判断是否为某类型（支持子类）
int(x) / float(x) / str(x) / bool(x)
list(x) / tuple(x) / set(x) / dict(x)
```

### 2.3 迭代相关

```python
range(start, stop, step)  # 生成整数序列（惰性）
enumerate(seq, start=0)   # 同时获取索引和值
zip(a, b)                 # 将多个序列"拉链式"配对
map(func, iterable)       # 对每个元素应用函数
filter(func, iterable)    # 过滤元素
sorted(iterable, key=..)  # 排序，返回新列表
reversed(seq)             # 反转序列
```

### 2.4 逻辑判断

```python
all(iterable)   # 全部为 True 返回 True
any(iterable)   # 任意一个为 True 返回 True
```

## 3. 重点说明

### 3.1 常用函数速查

| 函数 | 作用 | 示例 |
|------|------|------|
| `len` | 获取长度 | `len([1,2,3])` → 3 |
| `max/min` | 最大/最小值 | `max(1,5,3)` → 5 |
| `sum` | 求和 | `sum([1,2,3])` → 6 |
| `zip` | 并行迭代 | `zip([1,2],['a','b'])` → (1,'a'),(2,'b') |
| `map` | 函数映射 | `map(str, [1,2])` → ['1','2'] |
| `filter` | 过滤 | `filter(lambda x:x>0, [-1,0,1])` |
| `enumerate` | 索引+值 | `enumerate(['a','b'])` → (0,'a'),(1,'b') |
| `sorted` | 排序返回新列表 | `sorted([3,1,2])` → [1,2,3] |
| `any/all` | 逻辑判断 | `any([False,True])` → True |

### 3.2 map/filter vs 推导式

```python
# map + filter：函数式风格
list(map(lambda x: x*2, filter(lambda x: x>0, nums)))

# 列表推导式：更推荐，可读性更好
[x*2 for x in nums if x > 0]
```

### 3.3 sorted 的 key 参数

```python
# 按长度排序
sorted(words, key=len)
# 按字典某字段排序
sorted(users, key=lambda u: u["age"])
```

## 4. Demo 样例

```python
# ===== 1. enumerate + zip =====
names = ["张三", "李四", "王五"]
scores = [85, 92, 78]
for i, (name, score) in enumerate(zip(names, scores), 1):
    print(f"{i}. {name}: {score}分")
# 1. 张三: 85分  2. 李四: 92分  3. 王五: 78分

# ===== 2. sorted 自定义排序 =====
words = ["banana", "apple", "Cherry"]
print(sorted(words, key=str.lower))  # 忽略大小写
print(sorted(words, key=len))        # 按长度

# ===== 3. any / all =====
scores = [85, 92, 55, 78]
print(all(s >= 60 for s in scores))  # False (不是全部及格)
print(any(s >= 90 for s in scores))  # True  (有人90+)

# ===== 4. map 配合内置函数 =====
nums = ["1", "2", "3"]
print(list(map(int, nums)))  # [1, 2, 3]
```

## 5. 典型例题（附带答案）

### 例题 1

题目：用内置函数判断列表中是否所有元素都大于 0。

答案：`all(x > 0 for x in [1, 2, 3, -1])  # False`

### 例题 2

题目：对字典按值排序。

答案：

```python
scores = {"张三": 85, "李四": 92, "王五": 78}
sorted_items = sorted(scores.items(), key=lambda x: x[1])
print(sorted_items)  # [('王五',78), ('张三',85), ('李四',92)]
```

## 6. 易错点

### 易错点 1：sorted 返回新列表 vs .sort() 原地排序

`sorted()` 返回新列表（不改变原列表），`.sort()` 原地修改（不返回）。

### 易错点 2：max/min 对空序列报错

`max([])` 报 ValueError。安全做法：`max(lst, default=0)`。

### 易错点 3：map/filter 返回迭代器

`map(int, ["1","2"])` 返回迭代器，需 `list()` 转换。直接 print 看到的是对象地址。

## 7. 课后练习（附带答案）

### 练习 1

题目：用 `enumerate` 打印带编号的水果列表。

答案：

```python
fruits = ["苹果", "香蕉", "橘子"]
for i, f in enumerate(fruits, 1):
    print(f"{i}. {f}")
```

### 练习 2

题目：用 `zip` 合并两个列表为字典。

答案：`dict(zip(["name","age"], ["张三", 25]))`

### 练习 3

题目：用 `filter` 筛选出列表中的正整数。

答案：`list(filter(lambda x: x > 0, [-1, 0, 2, -3, 5]))  # [2, 5]`

## 8. 答题总结

- Python 内置函数是"瑞士军刀"：`len/max/min/sum`（计算）、`zip/map/filter/enumerate/sorted`（迭代）、`any/all`（逻辑）、`isinstance/type`（类型）。
- `sorted()` 返回新列表，`.sort()` 原地排序。`sorted` 支持 `key` 自定义规则。
- `map/filter` 返回迭代器，用 `list()` 转换。但推导式 `[...]` 更推荐。
- `zip` 并行迭代；`enumerate` 同时获取索引和值。`any/all` 配合生成器表达式省内存。
- `max/min/sum` 对空序列报错，可用 `default=` 参数安全处理。
