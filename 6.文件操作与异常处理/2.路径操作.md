# 路径操作

## 1. 知识点定义

路径操作是指在 Python 程序中对文件和文件夹的路径进行**拼接、判断、查找、创建**等操作。

> 小白理解：
> 路径就是文件在电脑上的"地址"。就像快递需要收货地址一样，程序读写文件也需要知道文件在哪里。路径操作就是帮程序找到正确的"地址"。

它主要解决的问题是：
- 拼接文件路径（配置文件路径、保存目录等）
- 判断文件/文件夹是否存在
- 遍历文件夹中的所有文件
- 获取文件名、扩展名、文件大小等元信息
- 创建多层目录

在实际写 Python 代码时，它常用于：
- 读取指定目录下的所有文件
- 批量重命名文件
- 创建项目目录结构
- 判断配置文件是否存在

Python 提供了两个主要模块：传统的 `os.path` 和现代化的 `pathlib`（推荐）。

## 2. 语法格式

### 2.1 绝对路径 vs 相对路径

```python
# 绝对路径：从盘符或根目录开始的完整路径
# Windows: C:\Users\name\Documents\file.txt
# Mac/Linux: /home/name/Documents/file.txt

# 相对路径：相对于当前工作目录的路径
# . 表示当前目录
# .. 表示上一级目录
# ./data/file.txt 表示当前目录下的 data 文件夹中的 file.txt
```

### 2.2 os.path 模块（传统方式）

```python
import os

# 路径拼接（自动处理不同系统的分隔符）
path = os.path.join('folder', 'subfolder', 'file.txt')

# 判断是否存在
os.path.exists('file.txt')      # 文件或目录是否存在
os.path.isfile('file.txt')      # 是否是文件
os.path.isdir('folder')         # 是否是目录

# 拆分路径
os.path.basename('/a/b/c.txt')  # 'c.txt' —— 文件名
os.path.dirname('/a/b/c.txt')   # '/a/b'  —— 目录部分
os.path.splitext('a.txt')       # ('a', '.txt') —— 文件名和扩展名

# 文件信息
os.path.getsize('file.txt')     # 文件大小（字节）
os.path.abspath('file.txt')     # 获取绝对路径
```

### 2.3 pathlib 模块（推荐方式）

```python
from pathlib import Path

# 创建 Path 对象
p1 = Path('folder/subfolder/file.txt')   # 相对路径
p2 = Path('/Users/name/data')            # 绝对路径
p3 = Path.home()                          # 用户主目录
p4 = Path.cwd()                           # 当前工作目录

# / 运算符拼接路径（pathlib 的核心特性！）
base = Path('/Users/name/project')
data_file = base / 'data' / 'results.csv'  # 用 / 拼接，非常直观

# 常用属性和方法
p = Path('/Users/name/project/data.csv')
p.name           # 'data.csv' —— 完整文件名
p.stem           # 'data' —— 文件名（不含扩展名）
p.suffix         # '.csv' —— 扩展名
p.parent         # Path('/Users/name/project') —— 父目录
p.exists()       # 路径是否存在
p.is_file()      # 是否是文件
p.is_dir()       # 是否是目录

# 创建目录
p.mkdir(parents=True, exist_ok=True)
# parents=True：递归创建父目录（类似 mkdir -p）
# exist_ok=True：目录已存在时不报错

# 读写文件（适合小型文件）
Path('hello.txt').write_text('你好，世界！', encoding='utf-8')
content = Path('hello.txt').read_text(encoding='utf-8')

# 遍历目录
for item in Path('.').iterdir():   # 列出当前目录内容（不递归）
    print(item)
for py_file in Path('.').rglob('*.py'):  # 递归搜索所有 .py 文件
    print(py_file)
```

### 2.4 获取脚本自身所在目录

```python
# 获取脚本文件所在的目录（而不是当前工作目录！）
script_dir = Path(__file__).parent
data_file = script_dir / 'data.txt'
```

> **重要**：相对路径是相对于**当前工作目录**（运行命令的地方），不是相对于脚本文件所在目录。

## 3. 重点说明

### 3.1 os.path vs pathlib 对比

| 功能 | os.path | pathlib |
|------|---------|---------|
| 路径拼接 | `os.path.join('a', 'b')` | `Path('a') / 'b'` |
| 判断存在 | `os.path.exists('a')` | `Path('a').exists()` |
| 获取文件名 | `os.path.basename(p)` | `Path(p).name` |
| 获取扩展名 | `os.path.splitext(p)[1]` | `Path(p).suffix` |
| 遍历目录 | `os.listdir('.')` | `Path('.').iterdir()` |
| 递归搜索 | `glob.glob('**/*.py')` | `Path().rglob('*.py')` |

> **推荐**：新项目优先使用 `pathlib`，它更面向对象、代码更易读。但维护老项目时需要熟悉 `os.path`。

### 3.2 相对路径的基准是什么？

这是一个非常容易踩坑的问题。相对路径是相对于**当前工作目录（CWD）**，不是相对于脚本文件所在目录。

```python
# 假设脚本在 /home/user/project/script.py
# 但用户在 /home/user/ 目录下运行: python project/script.py
# 此时：当前工作目录是 /home/user/，不是 /home/user/project/

# 安全做法：以脚本自身所在目录为基准
script_dir = Path(__file__).parent    # /home/user/project/
data_file = script_dir / 'data.txt'   # /home/user/project/data.txt ✅
```

### 3.3 glob 模式说明

| 模式 | 含义 |
|------|------|
| `*` | 匹配任意字符（不含路径分隔符） |
| `**` | 匹配任意层级的目录 |
| `?` | 匹配单个字符 |
| `[abc]` | 匹配 a、b 或 c |

```python
# glob() —— 不递归，只匹配当前目录
for f in Path('.').glob('*.py'):      # 当前目录下的 .py 文件
    print(f)

# rglob() —— 递归匹配所有子目录
for f in Path('.').rglob('*.py'):     # 所有子目录中的 .py 文件
    print(f)
```

### 3.4 mkdir 的 parents 和 exist_ok 参数

```python
# ❌ 错误：父目录不存在时会报错
Path('a/b/c').mkdir()

# ✅ 正确：递归创建所有父目录，已存在也不报错
Path('a/b/c').mkdir(parents=True, exist_ok=True)
```

## 4. Demo 样例

```python
from pathlib import Path

# ========== 路径拼接 ==========
# 用 / 运算符拼接路径，代码像写文件路径一样自然
project_dir = Path.home() / 'my_project'   # 在家目录下创建项目文件夹
data_dir = project_dir / 'data'            # 项目下的 data 子目录
print(f"数据目录: {data_dir}")

# ========== 创建目录 ==========
# parents=True 递归创建所有父目录，exist_ok=True 已存在时不报错
data_dir.mkdir(parents=True, exist_ok=True)
print(f"目录已创建: {data_dir}")

# ========== 写入和读取文件 ==========
note_file = data_dir / '笔记.txt'          # 拼接出文件路径
# write_text 一次性写入文本，适合小文件
note_file.write_text('今天学习了 pathlib\n非常方便！\n', encoding='utf-8')
# read_text 一次性读取全部文本
content = note_file.read_text(encoding='utf-8')
print(f"读取内容: {content}")

# ========== 路径信息查看 ==========
print(f"文件名: {note_file.name}")          # 笔记.txt
print(f"不含扩展名: {note_file.stem}")      # 笔记
print(f"扩展名: {note_file.suffix}")        # .txt
print(f"父目录: {note_file.parent}")        # .../my_project/data
print(f"文件大小: {note_file.stat().st_size} 字节")

# ========== 遍历当前目录的 Python 文件 ==========
print("\n当前目录下的 Python 文件:")
for py_file in Path('.').glob('*.py'):     # glob 不递归子目录
    print(f"  {py_file}")

# ========== os.path 方式（作为对比） ==========
import os
current_file = os.path.abspath(__file__)    # 获取当前脚本的绝对路径
print(f"\n当前文件路径: {current_file}")
print(f"文件名: {os.path.basename(current_file)}")
name, ext = os.path.splitext('学习笔记.txt')  # 分离文件名和扩展名
print(f"os.path.splitext: 名称='{name}', 扩展名='{ext}'")
```

**运行结果：**

```
数据目录: /Users/用户名/my_project/data
目录已创建: /Users/用户名/my_project/data
读取内容: 今天学习了 pathlib
非常方便！

文件名: 笔记.txt
不含扩展名: 笔记
扩展名: .txt
父目录: /Users/用户名/my_project/data
文件大小: 30 字节

当前目录下的 Python 文件:
  demo/demo_path.py

当前文件路径: /Users/用户名/project/script.py
文件名: script.py
os.path.splitext: 名称='学习笔记', 扩展名='.txt'
```

## 5. 典型例题（附带答案）

### 例题 1

**题目：**

编写函数 `find_large_files(directory, min_size_mb)`，递归搜索指定目录下所有大于 `min_size_mb` (MB) 的文件，打印它们的路径和大小。

**答案：**

```python
from pathlib import Path

def find_large_files(directory, min_size_mb):
    """查找大于指定大小的文件"""
    base = Path(directory)
    min_bytes = min_size_mb * 1024 * 1024    # 转换为字节

    for f in base.rglob('*'):                # 递归遍历所有文件和目录
        if f.is_file() and f.stat().st_size > min_bytes:
            size_mb = f.stat().st_size / (1024 * 1024)
            print(f"{size_mb:.2f} MB — {f}")

# find_large_files('.', 10)  # 查找大于10MB的文件
```

**解析：**
- `rglob('*')` 递归匹配所有文件和目录
- `f.is_file()` 过滤掉目录，只保留文件
- `f.stat().st_size` 获取文件字节数

### 例题 2

**题目：**

编写函数 `create_project_structure(project_name)`，在工作目录下创建如下项目结构：

```
project_name/
├── src/
├── tests/
├── docs/
└── README.md
```

**答案：**

```python
from pathlib import Path

def create_project_structure(project_name):
    """创建标准项目目录结构"""
    base = Path(project_name)
    dirs = ['src', 'tests', 'docs']

    for d in dirs:
        (base / d).mkdir(parents=True, exist_ok=True)  # 递归创建每个子目录

    # 创建 README.md 并写入初始内容
    (base / 'README.md').write_text(f'# {project_name}\n\n项目说明\n', encoding='utf-8')
    print(f"项目 {project_name} 创建完成！")

# create_project_structure('my_new_project')
```

**解析：**
- `/` 运算符拼接 `base` 和子目录名，自动适配不同操作系统
- `parents=True` 保证父目录不存在时也能成功创建
- `write_text()` 是 pathlib 提供的快捷写文件方法

## 6. 易错点

### 易错点 1：手动拼接路径而不是用 join() 或 /

错误写法：
```python
# 手动拼接容易在不同系统上出错
path = 'folder' + '/' + 'file.txt'   # Windows 上不对
path = 'folder' + '\\' + 'file.txt'  # Mac/Linux 上不对
```

错误原因：不同操作系统的路径分隔符不同（Windows 用 `\`，Mac/Linux 用 `/`），手动拼接无法跨平台。

正确写法：
```python
# os.path 方式
import os
path = os.path.join('folder', 'file.txt')

# pathlib 方式（推荐）
from pathlib import Path
path = Path('folder') / 'file.txt'
```

### 易错点 2：相对路径的基准是当前工作目录，不是脚本目录

错误写法：
```python
# 脚本中写了相对路径，以为相对于脚本所在目录
with open('data.txt', 'r') as f:  # 实际相对于运行命令的目录！
    content = f.read()
```

错误原因：相对路径始终相对于"当前工作目录"（即运行 `python xxx.py` 时所在的目录）。

正确写法：
```python
from pathlib import Path

# 以脚本自身所在目录为基准
script_dir = Path(__file__).parent
data_file = script_dir / 'data.txt'
with open(data_file, 'r', encoding='utf-8') as f:
    content = f.read()
```

### 易错点 3：mkdir 不设置 parents=True

错误写法：
```python
# 父目录 a/b 不存在时会报错！
Path('a/b/c').mkdir()
```

错误原因：`mkdir()` 默认只能创建最后一级目录，父目录不存在时抛出 `FileNotFoundError`。

正确写法：
```python
# 递归创建所有父目录
Path('a/b/c').mkdir(parents=True, exist_ok=True)
```

### 易错点 4：glob 不递归搜索子目录

错误写法：
```python
# glob() 只匹配当前目录，不递归子目录
files = Path('.').glob('*.py')
```

错误原因：`glob()` 不进入子目录搜索，`rglob()` 才是递归搜索。

正确写法：
```python
# rglob() 递归搜索所有子目录
files = Path('.').rglob('*.py')
```

## 7. 课后练习（附带答案）

### 练习 1

**题目：**

编写函数 `batch_rename(directory, old_ext, new_ext)`，将指定目录下所有 `old_ext` 扩展名的文件批量改为 `new_ext` 扩展名。

**答案：**

```python
from pathlib import Path

def batch_rename(directory, old_ext, new_ext):
    """批量修改文件扩展名"""
    count = 0
    for f in Path(directory).glob(f'*{old_ext}'):  # 匹配所有指定扩展名的文件
        new_name = f.with_suffix(new_ext)           # 替换扩展名
        f.rename(new_name)                          # 重命名
        print(f"重命名: {f.name} -> {new_name.name}")
        count += 1
    print(f"共重命名 {count} 个文件")

# batch_rename('./images', '.jpeg', '.jpg')
```

**说明：** `with_suffix()` 是 pathlib 提供的便捷方法，直接替换扩展名部分。

### 练习 2

**题目：**

编写函数 `classify_by_ext(directory)`，遍历目录中的所有文件，按扩展名分类后打印统计结果。

**答案：**

```python
from pathlib import Path
from collections import defaultdict

def classify_by_ext(directory):
    """按扩展名分类文件"""
    ext_groups = defaultdict(list)
    for f in Path(directory).iterdir():
        if f.is_file():
            ext = f.suffix.lower() or '(无扩展名)'
            ext_groups[ext].append(f.name)

    for ext, files in sorted(ext_groups.items()):
        print(f"{ext}: {len(files)} 个文件")
        for name in sorted(files):
            print(f"  - {name}")

# classify_by_ext('.')
```

**说明：** `defaultdict(list)` 自动为不存在的键创建空列表，省去手动判断。

### 练习 3

**题目：**

编写函数 `count_code_lines(directory, ext='.py')`，递归统计指定目录下所有指定扩展名文件的总行数和非空行数。

**答案：**

```python
from pathlib import Path

def count_code_lines(directory, ext='.py'):
    """统计代码行数"""
    total_lines = 0
    non_empty_lines = 0

    for f in Path(directory).rglob(f'*{ext}'):    # 递归搜索所有匹配文件
        with open(f, 'r', encoding='utf-8') as file:
            lines = file.readlines()
            total_lines += len(lines)
            non_empty_lines += sum(1 for line in lines if line.strip())

    print(f"总行数: {total_lines}")
    print(f"非空行数: {non_empty_lines}")
    if total_lines > 0:
        print(f"空行比例: {(1 - non_empty_lines/total_lines)*100:.1f}%")

# count_code_lines('.', '.py')
```

**说明：** `line.strip()` 去掉空白字符后，如果为空字符串，说明该行是空行。`sum(1 for ... if ...)` 是 Python 中高效的计数方式。

## 8. 答题总结

- 绝对路径是从根目录开始的完整路径，相对路径是相对于当前工作目录的路径。
- `os.path.join()` 和 `Path('a') / 'b'` 可以跨平台安全地拼接路径，不要手动用 `+` 拼接。
- `pathlib` 是 Python 3.4+ 推荐的路径操作方式，用 `/` 拼接路径非常直观。
- `rglob()` 递归搜索所有子目录，`glob()` 只搜索当前目录，不要搞混。
- `mkdir(parents=True, exist_ok=True)` 是创建多层目录的安全写法。
- 相对路径的基准是"当前工作目录"而非脚本所在目录，用 `Path(__file__).parent` 获取脚本目录。
- `Path` 对象的 `.name`、`.stem`、`.suffix`、`.parent` 等属性非常实用。
