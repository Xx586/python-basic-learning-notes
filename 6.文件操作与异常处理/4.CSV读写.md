# CSV读写

## 1. 知识点定义

CSV（Comma-Separated Values，逗号分隔值）是一种**表格数据存储格式**，本质上是纯文本文件，每行代表一条记录，字段之间用分隔符隔开。

> 小白理解：
> CSV 就是纯文字版的 Excel 表格。你可以用记事本打开它，看到数据整整齐齐地用逗号分开。它比 Excel 文件更小、更通用，几乎所有数据处理工具都支持。

它主要解决的问题是：
- Excel 数据的导入导出
- 数据分析和机器学习的原始数据集存储
- 不同系统之间的表格数据交换
- 数据库批量导入导出

在实际写 Python 代码时，它常用于：
- 读取实验数据、成绩单、财务报表等表格数据
- 将程序处理结果导出为 Excel 可打开的格式
- 数据清洗和转换

## 2. 语法格式

### 2.1 csv.reader：逐行读取

```python
import csv

# 基本读取（每行返回一个列表）
with open('data.csv', 'r', encoding='utf-8') as f:
    reader = csv.reader(f)
    for row in reader:
        print(row)  # ['张三', '25', '北京', '85']

# 注意：csv.reader 读取的所有字段都是字符串，需要手动转换 int/float
```

### 2.2 csv.writer：写入 CSV

```python
import csv

data = [
    ['姓名', '年龄', '城市'],
    ['张三', 25, '北京'],
    ['李四', 30, '上海'],
]

with open('output.csv', 'w', encoding='utf-8', newline='') as f:
    writer = csv.writer(f)
    writer.writerow(['姓名', '年龄'])      # 写入单行
    writer.writerows(data)                # 一次写入多行
```

> **关键**：`open()` 必须设置 `newline=''`，否则在 Windows 上每行之间会多一个空行。

### 2.3 csv.DictReader：字典方式读取（推荐）

```python
import csv

with open('data.csv', 'r', encoding='utf-8') as f:
    reader = csv.DictReader(f)       # 第一行自动作为表头
    for row in reader:
        # row 是字典，用字段名访问，比索引更直观
        print(f"{row['姓名']} - {row['年龄']}岁")
```

### 2.4 csv.DictWriter：字典方式写入（推荐）

```python
import csv

fieldnames = ['姓名', '年龄', '城市']
data = [
    {'姓名': '张三', '年龄': 25, '城市': '北京'},
    {'姓名': '李四', '年龄': 30, '城市': '上海'},
]

with open('output.csv', 'w', encoding='utf-8', newline='') as f:
    writer = csv.DictWriter(f, fieldnames=fieldnames)
    writer.writeheader()        # 必须先写入表头！
    writer.writerows(data)      # 批量写入数据行
```

### 2.5 编码处理：Excel 兼容

```python
# 写一个 Excel 能正确打开中文的 CSV
with open('output.csv', 'w', encoding='utf-8-sig', newline='') as f:
    writer = csv.writer(f)
    writer.writerow(['姓名', '年龄'])

# 读取 Excel 导出的 gbk 编码 CSV
with open('excel_export.csv', 'r', encoding='gbk') as f:
    reader = csv.DictReader(f)
    for row in reader:
        print(row)
```

## 3. 重点说明

### 3.1 DictReader vs reader 对比

| 对比项 | `csv.reader` | `csv.DictReader` |
|-------|-------------|-----------------|
| 返回类型 | 每行是一个**列表** | 每行是一个**字典** |
| 访问字段 | `row[0]`（用索引） | `row['姓名']`（用字段名） |
| 可读性 | 差（不知道索引对应哪个字段） | 好（一看就知道是什么） |
| 表头处理 | 需要手动跳过第一行 | 自动识别第一行为表头 |

> **推荐**：能用 `DictReader`/`DictWriter` 就尽量用，代码可读性高很多。

### 3.2 newline='' 为什么必须写

```python
# ❌ 不写 newline=''，Windows 上每行间会多一个空行
with open('data.csv', 'w', encoding='utf-8') as f:
    writer = csv.writer(f)
    writer.writerows(data)

# ✅ 正确写法
with open('data.csv', 'w', encoding='utf-8', newline='') as f:
    writer = csv.writer(f)
    writer.writerows(data)
```

> 原因：CSV 模块会自己处理换行符，如果 `open()` 也做换行符转换（Windows 上是 `\n` → `\r\n`），就会导致双倍换行。

### 3.3 编码问题：Excel 导出的 CSV 不是 utf-8

这是 CSV 操作中最常见的坑！Excel 默认用 **gbk** 编码导出 CSV。

```python
# 尝试多种编码读取
encodings = ['utf-8', 'utf-8-sig', 'gbk', 'gb2312']
for enc in encodings:
    try:
        with open('data.csv', 'r', encoding=enc) as f:
            reader = csv.reader(f)
            print(f"✓ 用 {enc} 编码读取成功")
            break
    except UnicodeDecodeError:
        continue
```

> **建议**：如果 CSV 要在 Excel 中打开，写入时用 `encoding='utf-8-sig'`；只在 Python 程序中使用时，统一用 `encoding='utf-8'`。

### 3.4 csv.reader 所有字段都是字符串

```python
# ❌ csv 读出来的数字是字符串，不能直接运算
with open('data.csv', 'r', encoding='utf-8') as f:
    reader = csv.DictReader(f)
    for row in reader:
        # row['年龄'] 是 '25'（字符串），不是 25（整数）
        age = int(row['年龄'])       # 手动转换
        score = float(row['分数'])   # 手动转换
```

## 5. 典型例题（附带答案）

### 例题 1

**题目：**

给定一个 `grades.csv` 文件，内容包含 `姓名,语文,数学,英语`，编写程序读取文件，计算每位学生的总分和平均分，按总分从高到低打印排名。

**答案：**

```python
import csv

# 先创建测试数据
with open('grades.csv', 'w', encoding='utf-8-sig', newline='') as f:
    writer = csv.writer(f)
    writer.writerow(['姓名', '语文', '数学', '英语'])
    writer.writerows([
        ['张三', '85', '92', '78'],
        ['李四', '90', '88', '95'],
        ['王五', '76', '85', '80'],
    ])

# 读取并计算
students = []
with open('grades.csv', 'r', encoding='utf-8-sig') as f:
    reader = csv.DictReader(f)
    for row in reader:
        name = row['姓名']
        scores = [int(row[k]) for k in ['语文', '数学', '英语']]  # 转为整数
        total = sum(scores)
        avg = total / len(scores)
        students.append({'姓名': name, '总分': total, '平均分': avg})

# 按总分降序排序
students.sort(key=lambda x: x['总分'], reverse=True)

print("排名 | 姓名 | 总分 | 平均分")
print("-" * 30)
for i, s in enumerate(students, 1):
    print(f"  {i}  | {s['姓名']} | {s['总分']} | {s['平均分']:.1f}")
```

**解析：** CSV 读取的所有字段都是字符串，做数学运算前必须用 `int()` 或 `float()` 转换。

### 例题 2

**题目：**

编写函数 `csv_to_json(csv_file, json_file)`，将 CSV 文件转换为 JSON 文件。

**答案：**

```python
import csv
import json

def csv_to_json(csv_file, json_file):
    """将 CSV 文件转换为 JSON 文件"""
    data = []
    with open(csv_file, 'r', encoding='utf-8-sig') as f:
        reader = csv.DictReader(f)
        for row in reader:
            data.append(row)               # 每行是一个字典

    with open(json_file, 'w', encoding='utf-8') as f:
        json.dump(data, f, ensure_ascii=False, indent=2)

    print(f"转换完成：{csv_file} → {json_file}，共 {len(data)} 条记录")

# csv_to_json('data.csv', 'data.json')
```

**解析：** `DictReader` 天然返回字典格式，直接收集后 `json.dump()` 即可，非常方便。

## 6. 易错点

### 易错点 1：忘记设置 newline=''

错误写法：
```python
with open('data.csv', 'w', encoding='utf-8') as f:  # 没有 newline=''
    writer = csv.writer(f)
    writer.writerows(data)
```

错误原因：CSV 模块自己处理换行符，`open()` 如果再转换一次（Windows 上），会导致每行之间出现多余空行。

正确写法：
```python
with open('data.csv', 'w', encoding='utf-8', newline='') as f:
    writer = csv.writer(f)
    writer.writerows(data)
```

### 易错点 2：Excel 导出 CSV 编码不是 utf-8

错误写法：
```python
# 用 utf-8 读 Excel 导出的 CSV 可能报 UnicodeDecodeError
with open('excel_data.csv', 'r', encoding='utf-8') as f:
    reader = csv.reader(f)
```

错误原因：Excel 默认使用 gbk 编码导出 CSV，不是 utf-8。

正确写法：
```python
# 尝试多种编码
for enc in ['utf-8', 'utf-8-sig', 'gbk', 'gb2312']:
    try:
        with open('excel_data.csv', 'r', encoding=enc) as f:
            reader = csv.reader(f)
            print(f"✓ 用 {enc} 编码读取成功")
            break
    except UnicodeDecodeError:
        continue

# 写入时用 utf-8-sig，Excel 能正确打开中文
with open('output.csv', 'w', encoding='utf-8-sig', newline='') as f:
    writer = csv.writer(f)
```

### 易错点 3：DictWriter 忘记 writeheader()

错误写法：
```python
with open('data.csv', 'w', encoding='utf-8', newline='') as f:
    writer = csv.DictWriter(f, fieldnames=['姓名', '年龄'])
    writer.writerows([{'姓名': '张三', '年龄': 25}])  # 没有调用 writeheader()！
```

错误原因：`DictWriter` 不会自动写表头，必须手动调用 `writeheader()`。

正确写法：
```python
with open('data.csv', 'w', encoding='utf-8', newline='') as f:
    writer = csv.DictWriter(f, fieldnames=['姓名', '年龄'])
    writer.writeheader()  # ← 这行不能少！
    writer.writerows([{'姓名': '张三', '年龄': 25}])
```

### 易错点 4：csv.reader 的所有字段都是字符串

错误写法：
```python
# row['年龄'] 是字符串 '25'，不能直接做数值运算
avg_age = sum(row['年龄'] for row in reader)  # TypeError!
```

错误原因：CSV 是纯文本格式，没有类型信息，所有字段读出来都是 `str`。

正确写法：
```python
age = int(row['年龄'])       # 手动转为整数
score = float(row['分数'])   # 手动转为浮点数
```

### 练习 1

**题目：**

读取 `data.csv`，筛选出所有年龄大于等于 18 岁的记录，写入 `adults.csv`。要求用 `DictReader` 和 `DictWriter`。

**答案：**

```python
import csv

# 准备测试数据
with open('data.csv', 'w', encoding='utf-8-sig', newline='') as f:
    writer = csv.writer(f)
    writer.writerow(['姓名', '年龄', '城市'])
    writer.writerows([
        ['张三', '25', '北京'],
        ['小明', '12', '上海'],
        ['李四', '30', '广州'],
    ])

# 筛选成年人
adults = []
with open('data.csv', 'r', encoding='utf-8-sig') as f:
    reader = csv.DictReader(f)
    fieldnames = reader.fieldnames          # 保存表头
    for row in reader:
        if int(row['年龄']) >= 18:           # 注意转为 int 再比较
            adults.append(row)

# 导出筛选结果
with open('adults.csv', 'w', encoding='utf-8-sig', newline='') as f:
    writer = csv.DictWriter(f, fieldnames=fieldnames)
    writer.writeheader()
    writer.writerows(adults)

print(f"共筛选出 {len(adults)} 条成年记录，已写入 adults.csv")
```

**说明：** 用 `reader.fieldnames` 获取表头，保证写入时字段顺序一致。

### 练习 2

**题目：**

编写程序，处理一个大 CSV 文件（不能一次性加载到内存），逐行统计总行数（不含表头）和各列缺失值（空值）的比例。

**答案：**

```python
import csv

def analyze_csv(filepath):
    """逐行分析 CSV 文件，不全部加载到内存"""
    with open(filepath, 'r', encoding='utf-8-sig') as f:
        reader = csv.DictReader(f)
        fieldnames = reader.fieldnames

        total = 0
        missing_counts = {field: 0 for field in fieldnames}  # 初始化每列缺失计数

        for row in reader:        # 逐行读取，内存占用始终很小
            total += 1
            for field in fieldnames:
                if not row[field].strip():   # 空字符串表示缺失
                    missing_counts[field] += 1

    print(f"总行数（不含表头）: {total}")
    print(f"\n各列缺失值统计：")
    for field in fieldnames:
        rate = (missing_counts[field] / total * 100) if total > 0 else 0
        print(f"  {field}: {missing_counts[field]} 个缺失 ({rate:.1f}%)")

# analyze_csv('big_file.csv')
```

**说明：** 用 `for row in reader` 逐行读取，无论文件多大，内存占用都很小。

### 练习 3

**题目：**

给定两个 CSV 文件 `a.csv` 和 `b.csv`（表头相同），将它们合并为一个 `merged.csv`，并按第一列去重。

**答案：**

```python
import csv

# 准备测试数据
for filename, rows in [
    ('a.csv', [['ID', '姓名', '分数'], ['001', '张三', '85'], ['002', '李四', '90']]),
    ('b.csv', [['ID', '姓名', '分数'], ['002', '李四', '90'], ['003', '王五', '78']]),
]:
    with open(filename, 'w', encoding='utf-8-sig', newline='') as f:
        csv.writer(f).writerows(rows)

# 合并去重
seen_ids = set()       # 用集合记录已见过的 ID
merged_rows = []

for filename in ['a.csv', 'b.csv']:
    with open(filename, 'r', encoding='utf-8-sig') as f:
        reader = csv.DictReader(f)
        if not merged_rows:
            fieldnames = reader.fieldnames
        for row in reader:
            if row['ID'] not in seen_ids:
                seen_ids.add(row['ID'])
                merged_rows.append(row)

# 导出合并结果
with open('merged.csv', 'w', encoding='utf-8-sig', newline='') as f:
    writer = csv.DictWriter(f, fieldnames=fieldnames)
    writer.writeheader()
    writer.writerows(merged_rows)

print(f"合并完成，共 {len(seen_ids)} 条不重复记录")
```

**说明：** 用 `set` 记录已见过的 ID 来实现去重，效率很高（O(1) 查找）。

## 8. 答题总结

- CSV 是纯文本表格格式，每行一条记录，字段用逗号分隔。
- `csv.reader` 返回列表，`csv.DictReader` 返回字典（推荐后者，代码更可读）。
- `open()` 时必须加 `newline=''`，否则 Windows 上会产生多余空行。
- CSV 读取的所有字段都是**字符串**，做数值运算前必须手动 `int()` 或 `float()`。
- Excel 导出的 CSV 通常是 gbk 编码，读取时要注意编码问题。
- 写入需要 Excel 打开的 CSV 时，使用 `encoding='utf-8-sig'`。
- `DictWriter` 必须先调用 `writeheader()` 写入表头，再写入数据行。
