# JSON读写

## 1. 知识点定义

JSON（JavaScript Object Notation）是一种**轻量级的数据交换格式**，用纯文本表示结构化数据。

> 小白理解：
> JSON 就像一张"通用的数据表格"，不同编程语言都能读写。Python 程序把数据存成 JSON，Java 程序也能读出来——就像大家都说普通话，沟通无障碍。

它主要解决的问题是：
- 前后端数据交互（Web API 最常用的格式）
- 配置文件存储（如 package.json）
- 数据持久化存储（替代简单的文本文件）
- 跨语言数据交换

在实际写 Python 代码时，它常用于：
- 调用 Web API 获取数据并解析
- 保存程序配置到 JSON 文件
- 不同系统之间的数据传输

## 2. 语法格式

### 2.1 JSON 格式示例

```json
{
    "name": "张三",
    "age": 25,
    "skills": ["Python", "SQL"],
    "is_student": false,
    "address": null
}
```

### 2.2 JSON vs Python 字典的区别

| 特性 | JSON | Python dict |
|------|------|-------------|
| 性质 | **字符串**格式 | 内存中的数据结构 |
| 键的类型 | 必须是字符串（双引号） | 任意可哈希类型 |
| 字符串引号 | 必须双引号 `"` | 单引号或双引号均可 |
| 布尔值 | `true`/`false`（小写） | `True`/`False`（首字母大写） |
| 空值 | `null`（小写） | `None` |
| 注释 | 不支持 | 支持 `#` |
| 末尾逗号 | 严格不允许 | 允许 |

### 2.3 字符串转换：loads() 和 dumps()

> 记忆口诀：带 `s` 的是处理**字符串**（string），不带 `s` 的是处理**文件**。

```python
import json

# json.loads()：JSON 字符串 → Python 对象
json_str = '{"name": "张三", "age": 25}'
data = json.loads(json_str)        # 解析为 Python 字典
print(data['name'])                # 张三

# json.dumps()：Python 对象 → JSON 字符串
data = {"name": "李四", "age": 30}
json_str = json.dumps(data)        # 序列化为 JSON 字符串

# dumps() 常用参数
json_str = json.dumps(data, ensure_ascii=False)  # 保留中文，不转义为 Unicode
json_str = json.dumps(data, indent=4)             # 格式化缩进，方便阅读
json_str = json.dumps(data, ensure_ascii=False, indent=2, sort_keys=True)
```

### 2.4 文件读写：load() 和 dump()

```python
import json

# json.dump()：把 Python 对象写入 JSON 文件
data = {"users": [{"name": "张三", "score": 85}]}
with open('data.json', 'w', encoding='utf-8') as f:
    json.dump(data, f, ensure_ascii=False, indent=4)

# json.load()：从 JSON 文件读取为 Python 对象
with open('data.json', 'r', encoding='utf-8') as f:
    data = json.load(f)
print(data['users'][0]['name'])    # 张三
```

### 2.5 JSON ↔ Python 类型映射

| Python 类型 | JSON 类型 |
|-------------|-----------|
| `dict` | `object` |
| `list`、`tuple` | `array` |
| `str` | `string` |
| `int`、`float` | `number` |
| `True` / `False` | `true` / `false` |
| `None` | `null` |

> **注意**：tuple 序列化为 JSON array 后，反序列化回来会变成 list。

## 3. 重点说明

### 3.1 loads/dumps 和 load/dump 怎么记？

| 函数 | 操作对象 | 方向 |
|------|---------|------|
| `loads()` | **字符串**（s = string） | JSON 字符串 → Python 对象 |
| `dumps()` | **字符串**（s = string） | Python 对象 → JSON 字符串 |
| `load()` | **文件** | JSON 文件 → Python 对象 |
| `dump()` | **文件** | Python 对象 → JSON 文件 |

### 3.2 ensure_ascii=False 很重要

```python
# 不加 ensure_ascii=False：中文变成 推荐
json.dumps({"城市": "北京"})
# 输出: '{"城市": "\\u5317\\u4eac"}'

# 加上 ensure_ascii=False：中文正常显示
json.dumps({"城市": "北京"}, ensure_ascii=False)
# 输出: '{"城市": "北京"}'
```

> 处理中文数据时，务必加上 `ensure_ascii=False`。

### 3.3 处理不可直接序列化的类型

`datetime`、`Decimal`、`set`、自定义类等不能直接被 `json.dumps()` 处理：

```python
import json
from datetime import datetime

# 方案1：使用 default 参数
def handle_datetime(obj):
    if isinstance(obj, datetime):
        return obj.strftime('%Y-%m-%d %H:%M:%S')
    raise TypeError(f'Type {type(obj)} not serializable')

json.dumps({"time": datetime.now()}, default=handle_datetime)

# 方案2：提前手动转换（最简单）
data = {"time": datetime.now().isoformat()}
json.dumps(data)
```

## 4. Demo 样例

```python
import json
from datetime import datetime

# ========== 1. 基本读写：字符串转换 ==========
print("=" * 40)
print("1. loads / dumps 字符串转换")

# 这是一段 JSON 格式的字符串（注意外层是 Python 字符串，内层是 JSON）
json_str = '{"name": "张三", "age": 25, "hobbies": ["读书", "游泳"]}'

# loads = load string，把 JSON 字符串解析为 Python 字典
data = json.loads(json_str)
print(f"解析后的类型: {type(data)}")      # <class 'dict'>
print(f"姓名: {data['name']}")            # 张三
print(f"爱好: {data['hobbies']}")         # ['读书', '游泳']

# dumps = dump string，把 Python 字典序列化为 JSON 字符串
data['city'] = '北京'
new_json = json.dumps(data, ensure_ascii=False, indent=2)  # 保留中文，缩进格式化
print(f"\n格式化 JSON:\n{new_json}")

# ========== 2. 文件读写 ==========
print("\n" + "=" * 40)
print("2. load / dump 文件读写")

users = [
    {"name": "Alice", "score": 85},
    {"name": "Bob", "score": 92},
]

# dump：把 Python 对象写入 JSON 文件
with open('users.json', 'w', encoding='utf-8') as f:
    json.dump(users, f, ensure_ascii=False, indent=4)
print("已写入 users.json")

# load：从 JSON 文件读取为 Python 对象
with open('users.json', 'r', encoding='utf-8') as f:
    loaded = json.load(f)
print(f"从文件读取到: {loaded}")

# ========== 3. 处理 datetime 类型 ==========
print("\n" + "=" * 40)
print("3. datetime 序列化处理")

def datetime_serializer(obj):
    """把 datetime 对象转为字符串，其他类型报错"""
    if isinstance(obj, datetime):
        return obj.strftime('%Y-%m-%d %H:%M:%S')
    raise TypeError(f'Type {type(obj)} not serializable')

# 创建包含 datetime 的数据
log = {
    "event": "user_login",
    "time": datetime.now(),        # datetime 类型不能直接序列化
    "user_id": 1001,
}

# 使用 default 参数传入自定义处理函数
json_str = json.dumps(log, ensure_ascii=False, default=datetime_serializer, indent=2)
print(json_str)

print("\nJSON 读写演示完成！")
```

## 5. 典型例题（附带答案）

### 例题 1

**题目：**

编写程序，将一个字典配置（包含数据库连接信息、服务器地址、端口等）写入 `config.json` 文件，然后再读出来打印。

**答案：**

```python
import json

config = {
    "database": {
        "host": "localhost",
        "port": 3306,
        "user": "admin",
        "password": "123456",
    },
    "server": {
        "host": "0.0.0.0",
        "port": 8080,
        "debug": True,
    }
}

# 写入配置文件
with open('config.json', 'w', encoding='utf-8') as f:
    json.dump(config, f, ensure_ascii=False, indent=2)
print("配置文件已保存")

# 读取配置文件
with open('config.json', 'r', encoding='utf-8') as f:
    loaded_config = json.load(f)

print(f"数据库地址: {loaded_config['database']['host']}")
print(f"服务器端口: {loaded_config['server']['port']}")
```

**解析：** 嵌套字典的访问用多级 `[]`，如 `loaded_config['database']['host']`。

### 例题 2

**题目：**

编写函数 `is_valid_json(s)`，判断一个字符串是否是合法的 JSON 格式，如果是则返回解析后的数据，否则返回错误信息。

**答案：**

```python
import json

def is_valid_json(s):
    """校验 JSON 字符串是否合法"""
    try:
        data = json.loads(s)
        return True, data
    except json.JSONDecodeError as e:
        return False, f"JSON格式错误: {e}"

# 测试
test_cases = [
    '{"name": "张三", "age": 25}',       # 合法
    '{"name": "张三", "age": }',         # 不合法
    '{name: "张三"}',                    # 不合法（key 没加引号）
]

for s in test_cases:
    valid, result = is_valid_json(s)
    if valid:
        print(f"✓ 合法 JSON: {result}")
    else:
        print(f"✗ {result}")
```

**解析：** 用 `try/except` 捕获 `json.JSONDecodeError` 来判断字符串是否为合法 JSON 是一种标准做法。

## 6. 易错点

### 易错点 1：混淆 loads/dumps 和 load/dump

错误写法：
```python
import json
data = {"name": "张三"}

# ❌ dump 期望文件对象，传了字符串
json_str = json.dump(data)  # TypeError！

# ❌ dumps 返回字符串，不是写入文件
json.dumps(data, f)          # 不会写入文件
```

错误原因：带 s 的是字符串操作，不带 s 的是文件操作，一定要分清。

正确写法：
```python
json_str = json.dumps(data)          # Python → JSON 字符串
data_back = json.loads(json_str)     # JSON 字符串 → Python
# json.dump(data, file_obj)          # Python → JSON 文件
# data = json.load(file_obj)         # JSON 文件 → Python
```

### 易错点 2：中文变成 Unicode 转义符

错误写法：
```python
data = {"城市": "北京", "美食": "烤鸭"}
json_str = json.dumps(data)  # 中文变成 北京
```

错误原因：`dumps()` 默认 `ensure_ascii=True`，非 ASCII 字符会被转义。

正确写法：
```python
json_str = json.dumps(data, ensure_ascii=False)
print(json_str)  # {"城市": "北京", "美食": "烤鸭"}
```

### 易错点 3：JSON 字符串中布尔值大小写错误

错误写法：
```python
bad_json = '{"active": True, "deleted": False}'  # Python 的 True/False！
```

错误原因：JSON 中的布尔值是 `true`/`false`（小写），不是 Python 的 `True`/`False`。

正确写法：
```python
good_json = '{"active": true, "deleted": false}'
data = json.loads(good_json)  # 解析后变成 Python 的 True/False
```

### 易错点 4：datetime 等类型直接序列化报错

错误写法：
```python
from datetime import datetime
json.dumps({"time": datetime.now()})  # TypeError!
```

错误原因：`datetime` 不是 JSON 可序列化类型，`json.dumps()` 不知道如何处理。

正确写法：
```python
# 方案1：提前转为字符串（最简单）
data = {"time": datetime.now().isoformat()}
json.dumps(data)

# 方案2：使用 default 参数
json.dumps({"time": datetime.now()}, default=lambda o: o.isoformat())
```

## 7. 课后练习（附带答案）

### 练习 1

**题目：**

编写一个简单的用户管理系统，支持添加用户（姓名、年龄、邮箱），将所有用户保存到 `users.json`，以及从 `users.json` 加载所有用户并打印。

**答案：**

```python
import json
import os

USERS_FILE = 'users.json'

def load_users():
    """加载用户列表，文件不存在时返回空列表"""
    if os.path.exists(USERS_FILE):
        with open(USERS_FILE, 'r', encoding='utf-8') as f:
            return json.load(f)
    return []

def save_users(users):
    """保存用户列表到 JSON 文件"""
    with open(USERS_FILE, 'w', encoding='utf-8') as f:
        json.dump(users, f, ensure_ascii=False, indent=2)

def add_user(name, age, email):
    """添加新用户"""
    users = load_users()
    users.append({"name": name, "age": age, "email": email})
    save_users(users)
    print(f"用户 {name} 已添加")

# 使用示例
add_user("张三", 25, "zhangsan@example.com")
add_user("李四", 30, "lisi@example.com")

all_users = load_users()
for user in all_users:
    print(f"{user['name']} | {user['age']}岁 | {user['email']}")
```

**说明：** 先用 `os.path.exists()` 判断文件是否存在，避免第一次运行时 `json.load()` 报错。

### 练习 2

**题目：**

编写日志函数 `write_log(event, detail)`，每次调用时将日志追加写入 `app.log.json` 文件。日志以 JSON 数组形式存储，每条包含 `time`、`event`、`detail` 三个字段。

**答案：**

```python
import json
import os
from datetime import datetime

LOG_FILE = 'app.log.json'

def write_log(event, detail):
    """追加写入 JSON 格式的日志"""
    # 读取已有日志（文件不存在时返回空列表）
    if os.path.exists(LOG_FILE):
        with open(LOG_FILE, 'r', encoding='utf-8') as f:
            logs = json.load(f)
    else:
        logs = []

    # 追加新日志条目
    logs.append({
        "time": datetime.now().strftime('%Y-%m-%d %H:%M:%S'),
        "event": event,
        "detail": detail,
    })

    # 写回文件
    with open(LOG_FILE, 'w', encoding='utf-8') as f:
        json.dump(logs, f, ensure_ascii=False, indent=2)

# 测试
write_log("user_login", "用户 admin 登录成功")
write_log("file_upload", "用户上传了 report.pdf")
print("日志已写入 app.log.json")
```

**说明：** 这个方案适合小型日志系统。如果日志量很大，应考虑逐行写入而非整体重写。

### 练习 3

**题目：**

编写一个通用的 JSON 编码器，支持序列化 `datetime`、`date` 和 `set` 类型（set 转为 list）。

**答案：**

```python
import json
from datetime import datetime, date

class UniversalEncoder(json.JSONEncoder):
    """通用 JSON 编码器，支持 datetime、date、set 类型"""
    def default(self, obj):
        if isinstance(obj, datetime):
            return obj.strftime('%Y-%m-%d %H:%M:%S')
        if isinstance(obj, date):
            return obj.strftime('%Y-%m-%d')
        if isinstance(obj, set):
            return list(obj)           # set 转为 list
        return super().default(obj)    # 其他类型交给父类处理

# 测试
data = {
    "name": "测试",
    "created_at": datetime.now(),
    "birthday": date(2000, 1, 1),
    "tags": {"Python", "JSON", "编码"},
}

json_str = json.dumps(data, ensure_ascii=False, indent=2, cls=UniversalEncoder)
print(json_str)
```

**说明：** 继承 `json.JSONEncoder` 并重写 `default()` 方法，是处理自定义类型的标准做法。

## 8. 答题总结

- JSON 是一种用纯文本表示结构化数据的格式，独立于编程语言。
- `loads()`/`dumps()` 处理字符串，`load()`/`dump()` 处理文件——带 s = string。
- 处理中文数据时务必加上 `ensure_ascii=False`，否则中文会变成 `\uXXXX`。
- JSON 的布尔值是 `true`/`false`（小写），空值是 `null`，与 Python 不同。
- `datetime`、`set` 等类型不能直接序列化，需要提前转换或使用 `default` 参数。
- `json.loads()` 解析非法 JSON 时会抛出 `json.JSONDecodeError`，应该用 `try/except` 捕获。
- `indent` 参数可以让输出的 JSON 格式化缩进，方便人类阅读。
