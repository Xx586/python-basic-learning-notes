# 继承与多态

## 1. 知识点定义

**继承**是子类"获得"父类的所有属性和方法，在此基础上可以添加或覆盖。**多态**是不同对象调用同一方法，表现出不同的行为。两者是面向对象编程的核心特性。

继承 = 复用代码（子类继承父类的成员）。多态 = 同一接口，不同实现。

> 小白理解：继承 = 你继承了父母的房子和财产（属性和方法），还可以自己添置家具（添加新方法）或者重新装修（重写方法）。多态 = 同样的指令"叫"，狗汪汪叫，猫喵喵叫——表现形式不同。

## 2. 语法格式

### 2.1 单继承

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        return f"{self.name} 发出声音"

# Dog 继承 Animal
class Dog(Animal):
    def speak(self):                # 方法重写（override）
        return f"{self.name} 汪汪叫！"

class Cat(Animal):
    def speak(self):
        return f"{self.name} 喵喵叫！"

dog = Dog("旺财")
cat = Cat("咪咪")
print(dog.speak())  # 旺财 汪汪叫！
print(cat.speak())  # 咪咪 喵喵叫！
```

### 2.2 super() 调用父类方法

```python
class Dog(Animal):
    def __init__(self, name, breed):
        super().__init__(name)     # 调用父类的 __init__
        self.breed = breed         # 添加子类自己的属性
```

### 2.3 多继承

```python
class A:
    def method(self): return "A"
class B:
    def method(self): return "B"
class C(A, B):  # 多继承，MRO：C → A → B → object
    pass

c = C()
print(c.method())  # "A"（按 MRO 顺序先找到 A）
```

## 3. 重点说明

### 3.1 方法重写（Override）

子类定义与父类同名的方法，覆盖父类实现。`super().method()` 可以调用父类版本。

### 3.2 MRO（方法解析顺序）

多继承中，Python 使用 C3 线性化确定方法查找顺序。查看：`ClassName.__mro__` 或 `ClassName.mro()`。

### 3.3 多态的核心价值

```python
def animal_sound(animal):   # 不关心具体是什么动物
    print(animal.speak())   # 同一接口，不同实现

animal_sound(Dog("旺财"))   # 旺财 汪汪叫！
animal_sound(Cat("咪咪"))   # 咪咪 喵喵叫！
```

## 4. Demo 样例

```python
class Animal:
    def __init__(self, name):
        self.name = name
    def speak(self):
        return "..."

class Dog(Animal):
    def speak(self):
        return f"{self.name}: 汪汪！"

class Cat(Animal):
    def speak(self):
        return f"{self.name}: 喵喵！"

# 多态：同一接口，不同实现
animals = [Dog("旺财"), Cat("咪咪"), Dog("大黄")]
for a in animals:
    print(a.speak())
# 旺财: 汪汪！ 咪咪: 喵喵！ 大黄: 汪汪！
```

## 5. 典型例题（附带答案）

### 例题 1

题目：创建 `Vehicle` 基类，`Car` 和 `Bike` 子类，每个子类实现 `move()` 方法。

答案：

```python
class Vehicle:
    def move(self): return "移动"
class Car(Vehicle):
    def move(self): return "汽车在公路上行驶"
class Bike(Vehicle):
    def move(self): return "自行车在骑行"
```

### 例题 2

题目：`super()` 的作用是什么？

答案：`super()` 在子类中调用父类的方法，常用于 `__init__` 中复用父类的初始化逻辑。

## 6. 易错点

### 易错点 1：忘记调用 super().__init__

子类覆盖 `__init__` 后，如果忘记 `super().__init__()`，父类的初始化代码不会执行。

### 易错点 2：多继承的菱形继承问题

两个父类有同名方法时，MRO 决定用哪个。避免过于复杂的多继承结构。

### 易错点 3：Python 中所有类默认继承 object

`class MyClass:` 等价于 `class MyClass(object):`。

## 7. 课后练习（附带答案）

### 练习 1

题目：创建 `Shape` 基类，`Circle` 和 `Rectangle` 子类，每个实现 `area()`。

答案：

```python
class Shape:
    def area(self): raise NotImplementedError
class Circle(Shape):
    def __init__(self, r): self.r = r
    def area(self): return 3.14 * self.r ** 2
class Rectangle(Shape):
    def __init__(self, w, h): self.w, self.h = w, h
    def area(self): return self.w * self.h
```

## 8. 答题总结

- 继承：`class Child(Parent)`，复用父类代码。`super()` 调用父类方法。
- 方法重写（Override）：子类定义同名方法覆盖父类。多态：同一接口不同实现。
- MRO 决定多继承中的方法查找顺序。`__mro__` 可查看。
- 继承满足 is-a 关系（Dog is a Animal）；组合满足 has-a 关系（Car has a Engine）。
- 优先使用组合而非深层次继承，继承层次不宜超过 3 层。
