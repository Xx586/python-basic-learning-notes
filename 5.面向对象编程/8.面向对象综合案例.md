# 面向对象综合案例

## 1. 知识点定义

本案例通过一个完整的**学生信息管理系统**，把面向对象的核心概念（类、封装、继承、方法类型、魔术方法、文件操作）串起来实战。

> 小白理解：学了一堆乐高零件（类、继承、多态...），现在用这些零件搭一个完整的房子，看看它们在实际项目中怎么配合使用。

## 2. 语法格式

本案例使用的核心知识点：

| 知识点 | 在案例中的使用 |
|--------|-------------|
| `class` | 定义 Student 类 |
| `__init__` | 初始化学生信息 |
| `self` | 访问实例属性 |
| `@property` | 计算总分和平均分 |
| `__str__` / `__repr__` | 打印学生信息 |
| 文件操作 | 保存/读取学生数据 |

## 3. 重点说明

### 3.1 系统设计思路

```
学生管理系统
├── Student 类（数据模型：姓名、成绩、计算总分）
├── StudentManager 类（业务逻辑：增删改查）
└── 主程序（命令行交互菜单）
```

### 3.2 封装的应用

- Student 类封装了学生数据和计算方法
- StudentManager 类封装了管理逻辑
- 外部代码只通过公开方法操作数据

## 4. Demo 样例

```python
# ======== 学生信息管理系统（简化版） ========

class Student:
    """学生类：存储单个学生的信息"""
    def __init__(self, name, chinese, math, english):
        self.name = name
        self.chinese = chinese
        self.math = math
        self.english = english

    @property
    def total(self):
        """总分"""
        return self.chinese + self.math + self.english

    @property
    def average(self):
        """平均分"""
        return self.total / 3

    def __str__(self):
        return (f"{self.name}: 语文{self.chinese}, 数学{self.math}, "
                f"英语{self.english}, 总分{self.total}, 平均{self.average:.1f}")


class StudentManager:
    """学生管理类：增删改查"""
    def __init__(self):
        self.students = []  # 存储所有学生

    def add(self, student):
        """添加学生"""
        self.students.append(student)

    def remove(self, name):
        """按姓名删除学生"""
        self.students = [s for s in self.students if s.name != name]

    def find(self, name):
        """按姓名查找"""
        for s in self.students:
            if s.name == name:
                return s
        return None

    def list_all(self):
        """列出所有学生"""
        if not self.students:
            print("暂无学生记录")
            return
        for i, s in enumerate(self.students, 1):
            print(f"{i}. {s}")

    def top_student(self):
        """总分最高的学生"""
        if not self.students:
            return None
        return max(self.students, key=lambda s: s.total)


# 使用演示
manager = StudentManager()
manager.add(Student("张三", 85, 92, 78))
manager.add(Student("李四", 75, 88, 90))
manager.add(Student("王五", 92, 86, 95))

print("===== 所有学生 =====")
manager.list_all()

print(f"\n===== 第一名 =====")
top = manager.top_student()
print(top)

print(f"\n===== 查找张三 =====")
stu = manager.find("张三")
if stu:
    print(f"找到: {stu}")
```

```
2. 李四: 语文75, 数学88, 英语90, 总分253, 平均84.3
3. 王五: 语文92, 数学86, 英语95, 总分273, 平均91.0

===== 第一名 =====
王五: 语文92, 数学86, 英语95, 总分273, 平均91.0

===== 查找张三 =====
找到: 张三: 语文85, 数学92, 英语78, 总分255, 平均85.0
```

## 5. 典型例题（附带答案）

### 例题 1

题目：给 Student 类添加 `grade` 属性（property），根据平均分返回等级。

答案：

```python
@property
def grade(self):
    avg = self.average
    if avg >= 90: return "A"
    elif avg >= 80: return "B"
    elif avg >= 70: return "C"
    elif avg >= 60: return "D"
    return "F"
```

### 例题 2

题目：给 StudentManager 添加 `average_all()` 方法，计算所有学生的平均总分。

答案：

```python
def average_all(self):
    if not self.students:
        return 0
    return sum(s.total for s in self.students) / len(self.students)
```

## 6. 易错点

### 易错点 1：实例属性未在 __init__ 中初始化

避免在类级别定义可变属性（如 `students = []`），应在 `__init__` 中初始化。

### 易错点 2：忘记 self

实例方法第一个参数必须是 `self`。调用时 Python 自动传入。

### 易错点 3：方法返回值缺失

修改数据的方法通常返回 `None`；查询方法必须 `return` 结果。

```python
def sort_by_total(self):
    sorted_students = sorted(self.students, key=lambda s: s.total, reverse=True)
```

## 8. 答题总结

- 面向对象编程 = 把数据和操作打包成对象，用类组织代码。
- 典型设计：Model 类（数据）+ Manager 类（业务逻辑）+ 主程序（交互）。
- `@property` 用于计算属性（总分、平均分、等级）。
- 封装让外部只通过公开方法操作数据，内部实现可自由修改。
- 实际项目中，通常一个文件一个类，按功能拆分模块。
- 完整案例的 `demo/8_student_management/` 目录中有更完整的代码实现。
