# 魔术方法

## 1. 知识点定义

**魔术方法（Magic Methods）** 是 Python 中以双下划线开头和结尾的特殊方法（如 `__init__`、`__str__`）。它们让你的类"融入"Python 的语法体系——让自定义对象支持 `print()`、`len()`、`+`、`[]` 等内置操作。

> 小白理解：魔术方法就像万能遥控器——只要你的设备实现了对应的按键协议，就能用标准遥控器控制它，不需要专用遥控器。

## 2. 语法格式

| 魔术方法 | 触发方式 | 作用 |
|---------|---------|------|
| `__init__(self)` | `obj = Class()` | 构造对象 |
| `__str__(self)` | `print(obj)` / `str(obj)` | 用户友好输出 |
| `__repr__(self)` | `repr(obj)` / 直接回车 | 开发者调试输出 |
| `__len__(self)` | `len(obj)` | 支持 len() |
| `__eq__(self, o)` | `a == b` | 相等比较 |
| `__lt__(self, o)` | `a < b` | 排序比较 |
| `__add__(self, o)` | `a + b` | 加法/拼接 |
| `__getitem__(self, k)` | `obj[k]` | 索引访问 |
| `__iter__(self)` | `for x in obj` | 可迭代 |
| `__enter__/__exit__` | `with obj:` | 上下文管理器 |

```python
class Book:
    def __init__(self, title, author, pages):
        self.title, self.author, self.pages = title, author, pages

    def __str__(self):              # print() 时使用
        return f"《{self.title}》作者：{self.author}"

    def __repr__(self):             # 调试输出
        return f"Book('{self.title}', '{self.author}', {self.pages})"

    def __len__(self):              # len()
        return self.pages

    def __eq__(self, other):        # ==
        return self.title == other.title and self.author == other.author
```

## 3. 重点说明

### 3.1 `__str__` vs `__repr__`

| 方法 | 面向 | 目标 | 优先级 |
|------|------|------|:--:|
| `__str__` | 用户 | 易读 | 优先（print 先找它） |
| `__repr__` | 开发者 | 精确 | 备选（`__str__` 未实现时用） |

### 3.2 运算符重载

```python
class Point:
    def __init__(self, x, y): self.x, self.y = x, y
    def __add__(self, other): return Point(self.x+other.x, self.y+other.y)
    def __eq__(self, other): return self.x==other.x and self.y==other.y

p1, p2 = Point(1,2), Point(3,4)
p3 = p1 + p2  # Point(4, 6)
```

## 4. Demo 样例

```python
class Vector:
    def __init__(self, x, y): self.x, self.y = x, y
    def __str__(self): return f"Vector({self.x}, {self.y})"
    def __add__(self, o): return Vector(self.x+o.x, self.y+o.y)
    def __eq__(self, o): return self.x==o.x and self.y==o.y
    def __abs__(self): return (self.x**2+self.y**2)**0.5

v1, v2 = Vector(1,2), Vector(3,4)
print(v1+v2)     # Vector(4, 6)
print(v1==v2)    # False
print(abs(v1))   # 2.236
```

## 5. 典型例题（附带答案）

题目：让 `Student` 对象支持按学号 `==` 比较。

答案：

```python
class Student:
    def __init__(self, id, name): self.id, self.name = id, name
    def __eq__(self, other): return self.id == other.id
```

## 6. 易错点

### 易错点 1：`__str__` 必须返回字符串

错误：返回 int → TypeError。正确：返回 `str` 类型。

### 易错点 2：`__eq__` 未实现时 `==` 比较身份

两个内容相同的对象 `==` 返回 False（默认比较 `is`）。

### 易错点 3：`__init__` 不是"私有"

`__init__` 是魔术方法（双下划线包围），不会被名称改写。

## 7. 课后练习（附带答案）

题目：实现 `Fraction` 类，支持 `+` 和 `==` 运算。

答案：实现 `__add__` 和 `__eq__` 方法，分别处理分数的加法和相等比较。

## 8. 答题总结

- 魔术方法（`__xxx__`）让自定义对象支持 Python 内置语法和操作符。
- 常用：`__str__`/`__repr__`（输出）、`__eq__`（比较）、`__len__`（长度）、`__add__`（加法）。
- `__getitem__`/`__iter__` 让对象像容器；`__enter__/__exit__` 支持 with 语句。
- 只在有意义的场景定义魔术方法，不要为炫技滥用。
