# 方法类型

## 1. 知识点定义

Python 类中有三种方法类型：**实例方法**、**类方法**、**静态方法**。它们各有用途，通过不同的装饰器区分。

| 类型 | 装饰器 | 第一个参数 | 访问权限 |
|------|--------|-----------|---------|
| 实例方法 | 无（默认） | `self`（实例） | 实例属性 + 类属性 |
| 类方法 | `@classmethod` | `cls`（类） | 只能访问类属性 |
| 静态方法 | `@staticmethod` | 无 | 不能直接访问实例/类属性 |

> 小白理解：实例方法 = 员工（操作自己的工位），类方法 = 行政（操作公司的公告板），静态方法 = 外包（和公司有关但不碰公司内部数据）。

## 2. 语法格式

```python
class Student:
    school = "Python学院"  # 类属性

    def __init__(self, name, score):
        self.name = name
        self.score = score

    # 实例方法：操作实例数据，最常用（90%）
    def is_pass(self):
        return self.score >= 60

    # 类方法：操作类级别数据
    @classmethod
    def change_school(cls, new_school):
        cls.school = new_school

    # 静态方法：与类相关但不需要访问实例或类
    @staticmethod
    def is_valid_score(score):
        return 0 <= score <= 100
```

## 3. 重点说明

### 3.1 何时用哪种方法？

| 方法 | 使用场景 |
|------|---------|
| **实例方法** | 需要读写实例属性（`self.xxx`），最常用 |
| **类方法** | 操作类属性、定义备选构造器（如 `from_string`） |
| **静态方法** | 工具函数、校验逻辑，和类逻辑相关但不碰数据 |

### 3.2 @property 把方法变成属性访问

```python
class Circle:
    def __init__(self, radius):
        self._radius = radius

    @property
    def radius(self):        # getter：像属性一样读取
        return self._radius

    @radius.setter
    def radius(self, value): # setter：像属性一样赋值
        if value < 0:
            raise ValueError("半径不能为负")
        self._radius = value

c = Circle(5)
print(c.radius)  # 5 — 像访问属性，不需要 c.radius()
c.radius = 10    # 像赋值属性，自动调 setter
```

## 4. Demo 样例

```python
class Student:
    school = "Python学院"
    count = 0

    def __init__(self, name, score):
        self.name = name
        self.score = score
        Student.count += 1

    def is_pass(self):               # 实例方法
        return self.score >= 60

    @classmethod
    def get_count(cls):              # 类方法
        return f"共创建 {cls.count} 名学生"

    @staticmethod
    def grade(score):                # 静态方法
        if score >= 90: return "A"
        elif score >= 80: return "B"
        elif score >= 60: return "C"
        return "D"

s1 = Student("小明", 85)
s2 = Student("小红", 55)
print(s1.is_pass())          # True
print(Student.get_count())   # 共创建 2 名学生
print(Student.grade(85))     # B（不需要实例）
```

## 5. 典型例题（附带答案）

### 例题 1

题目：解释实例方法、类方法、静态方法的区别。

答案：实例方法操作实例数据（`self`），类方法操作类数据（`cls`），静态方法是独立工具函数。前者无装饰器，后两者分别用 `@classmethod` 和 `@staticmethod`。

### 例题 2

题目：给 `Person` 类添加一个类方法 `from_birth_year`，根据出生年份创建实例。

答案：

```python
class Person:
    def __init__(self, name, age):
        self.name, self.age = name, age
    @classmethod
    def from_birth_year(cls, name, birth_year):
        return cls(name, 2026 - birth_year)
```

## 6. 易错点

### 易错点 1：实例方法忘记 self

`def method():` → 调用时报 TypeError。正确：`def method(self):`

### 易错点 2：类方法和静态方法混淆

类方法需要 `cls` 参数访问类属性；静态方法不需要类/实例数据时使用。

### 易错点 3：@property 使用不当

property 用于计算属性，不要做耗时操作（用户期望属性访问很快）。

## 7. 课后练习（附带答案）

### 练习 1

题目：定义 `Calculator` 类，添加静态方法 `add(a, b)` 和 `multiply(a, b)`。

答案：`@staticmethod` 修饰，直接 `Calculator.add(3, 5)`。

### 练习 2

题目：给 `Student` 类添加 `@property`，计算总分和平均分。

答案：

```python
class Student:
    def __init__(self, chinese, math):
        self.chinese, self.math = chinese, math
    @property
    def total(self): return self.chinese + self.math
    @property
    def average(self): return self.total / 2
```

## 8. 答题总结

- 三种方法：实例方法（`self`，最常用）、类方法（`@classmethod`，操作类属性）、静态方法（`@staticmethod`，独立工具）。
- `@property` 把方法变成属性访问，配合 setter 实现数据校验。
- 90% 的方法都是实例方法；类方法常用于备选构造器；静态方法用于工具函数。
- 实例方法第一个参数必须写 `self`，调用时不需要显式传入。
