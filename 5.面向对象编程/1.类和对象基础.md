# 类和对象基础

## 1. 知识点定义

面向对象编程（OOP）是把**数据**和**操作数据的方法**打包在一起组织代码的方式。类是"图纸"，对象是按图纸造出来的"产品"。

三种编程范式对比：

| 范式 | 核心思想 | 关注点 |
|------|---------|--------|
| 面向过程 | 按步骤写代码，函数调用函数 | 先做什么，再做什么 |
| 面向对象 | 把数据和操作封装成对象 | 有哪些东西，它们能做什么 |
| 函数式 | 用函数组合解决问题 | 数据流入 → 变换 → 流出 |

> 小白理解：
> 类 = 手机的设计图纸（定义了手机有哪些零件和功能）。对象 = 按照图纸生产出来的真实手机（你能拿在手里用的那台）。

## 2. 语法格式

### 2.1 定义类和创建对象

```python
class Dog:
    """狗类"""
    species = "犬科"          # 类属性（所有实例共享）

    def __init__(self, name, age):
        """构造方法：创建对象时自动调用"""
        self.name = name      # 实例属性（每个对象独有）
        self.age = age

    def bark(self):
        """实例方法"""
        print(f"{self.name} 在汪汪叫！")

# 创建对象（实例化）
dog1 = Dog("旺财", 3)
dog2 = Dog("小白", 1)

print(dog1.name)      # 旺财（实例属性）
print(dog2.name)      # 小白
print(Dog.species)    # 犬科（类属性，所有实例共享）
dog1.bark()           # 旺财 在汪汪叫！
```

### 2.2 关键要素

| 要素 | 说明 | 示例 |
|------|------|------|
| `class` | 定义类 | `class Dog:` |
| `__init__` | 构造方法，创建对象时自动调用 | `def __init__(self, name):` |
| `self` | 指向当前实例对象 | 必须是实例方法的第一个参数 |
| 实例属性 | 每个对象独立拥有 | `self.name = name` |
| 类属性 | 所有实例共享 | `species = "犬科"` |

## 3. 重点说明

### 3.1 类属性 vs 实例属性

```python
class Student:
    school = "清华"        # 类属性：所有学生共享

    def __init__(self, name):
        self.name = name   # 实例属性：每个学生不同

s1 = Student("张三")
s2 = Student("李四")
print(s1.school, s2.school)  # 清华 清华（共享）
s1.school = "北大"           # 给 s1 创建了同名的实例属性
print(s1.school, s2.school)  # 北大 清华（s2 不受影响）
```

### 3.2 self 是什么

`self` 就是对象本身。当你调用 `dog1.bark()`，Python 实际执行的是 `Dog.bark(dog1)`，`self` 就是 `dog1`。

### 3.3 Python 3 中所有类默认继承 object

```python
class MyClass:        # Python 3 中等价于 class MyClass(object):
    pass
```

## 4. Demo 样例

```python
class Student:
    """学生类"""
    school = "Python学院"  # 类属性

    def __init__(self, name, age, score=0):
        self.name = name       # 实例属性
        self.age = age
        self.score = score

    def introduce(self):
        """自我介绍"""
        print(f"我叫{self.name}，{self.age}岁，来自{self.school}")

    def is_pass(self):
        """是否及格"""
        return self.score >= 60

# 创建对象
stu1 = Student("小明", 18, 85)
stu2 = Student("小红", 19, 55)

stu1.introduce()   # 我叫小明，18岁，来自Python学院
print(stu1.is_pass())  # True
print(stu2.is_pass())  # False
```

## 5. 典型例题（附带答案）

### 例题 1

题目：定义一个 `Book` 类，属性有书名和作者，方法 `info()` 返回描述字符串。

答案：

```python
class Book:
    def __init__(self, title, author):
        self.title = title
        self.author = author
    def info(self):
        return f"《{self.title}》作者：{self.author}"
```

### 例题 2

题目：以下代码输出什么？

```python
class A:
    count = 0
    def __init__(self):
        A.count += 1
a1, a2 = A(), A()
print(A.count)
```

答案：输出 `2`。`count` 是类属性，所有实例共享，每次 `__init__` 调用 `A.count += 1`。

## 6. 易错点

### 易错点 1：忘记 self 参数

错误：`def bark():` → TypeError
正确：`def bark(self):`

### 易错点 2：实例属性写在类级别

错误：

```python
class Dog:
    name = ""      # 这是类属性，所有狗同名！
```

正确：在 `__init__` 中用 `self.name = name`。

### 易错点 3：通过实例修改类属性

`obj.class_attr = value` 创建的是实例属性，不改变类属性。修改类属性用 `ClassName.attr = value`。

## 7. 课后练习（附带答案）

### 练习 1

题目：定义 `Circle` 类，初始化半径，方法 `area()` 返回面积（π=3.14）。

答案：

```python
class Circle:
    def __init__(self, radius):
        self.radius = radius
    def area(self):
        return 3.14 * self.radius ** 2
```

### 练习 2

题目：定义 `Counter` 类，类属性 `total` 记录创建了多少个实例。

答案：

```python
class Counter:
    total = 0
    def __init__(self):
        Counter.total += 1
```

## 8. 答题总结

- 类用 `class` 定义，对象通过 `类名()` 创建。类是图纸，对象是产品。
- `__init__(self)` 构造方法：创建对象时自动调用，用来初始化属性。
- `self` 指向当前实例，必须是实例方法的第一个参数。
- 实例属性（`self.xxx`）每个对象独有；类属性所有对象共享。
- 不要将可变对象作为类属性的默认值，小心在 `__init__` 中初始化实例属性。
