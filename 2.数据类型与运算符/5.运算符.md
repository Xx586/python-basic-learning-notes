# 运算符

## 1. 知识点定义

运算符是对数据执行操作的符号。Python 提供了丰富的运算符，包括算术、比较、逻辑、赋值、成员、身份、位运算符等。运算符和操作数组成的表达式是编程中最基本的计算单元。

> 小白理解：
> 运算符就是数据的"动词"。数据（名词）加上运算符（动词）就能表达完整的计算意图：`3 + 5`（三加五）、`age >= 18`（年龄是否满十八）。

## 2. 语法格式

### 2.1 算术运算符

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `+` | 加 | `10 + 3` | `13` |
| `-` | 减 | `10 - 3` | `7` |
| `*` | 乘 | `10 * 3` | `30` |
| `/` | 除（结果总是 float） | `10 / 3` | `3.333...` |
| `//` | 整除（向下取整） | `10 // 3` | `3` |
| `%` | 取余 | `10 % 3` | `1` |
| `**` | 乘方 | `10 ** 3` | `1000` |

### 2.2 比较运算符

`==`、`!=`、`>`、`<`、`>=`、`<=` —— 结果总是 `True` 或 `False`。

### 2.3 逻辑运算符（短路求值）

| 运算符 | 说明 | 短路规则 |
|--------|------|---------|
| `and` | 逻辑与 | 左边为 False，右边不执行 |
| `or` | 逻辑或 | 左边为 True，右边不执行 |
| `not` | 逻辑非 | 取反 |

```python
# and/or 返回操作数本身，不一定是 bool！
print(1 and 2 and 3)      # 3（全真，返回最后一个）
print(0 or "" or [])      # []（全假，返回最后一个）
print(0 or 2 or 3)        # 2（返回第一个真值）

# 短路求值实战
data = None
items = data or []         # data 有值就用 data，否则用空列表
```

### 2.4 赋值运算符

`+=`、`-=`、`*=`、`/=`、`//=`、`%=`、`**=`

### 2.5 成员运算符

`in` / `not in` — 判断元素是否在序列中。字典中 `in` 检查的是**键**，不是值。

### 2.6 身份运算符

`is` / `is not` — 判断两个对象是否是**同一个对象**（比较内存地址，不是值！）。

```python
# == 比较值，is 比较身份
a = [1, 2, 3]
b = [1, 2, 3]
print(a == b)     # True（值相同）
print(a is b)     # False（不同对象）

# 判断 None 必须用 is
if x is None:     # ✅ 正确
    pass
```

### 2.7 三元表达式

```python
# 语法：值1 if 条件 else 值2
status = "成年" if age >= 18 else "未成年"
```

### 2.8 运算符优先级（从高到低）

`()` → `**` → `* / // %` → `+ -` → 比较/成员/身份 → `not` → `and` → `or`

> 口诀：拿不准优先级就**加括号**，括号里的总是先算。

## 3. 重点说明

### 3.1 == vs is

| 运算符 | 比较的是什么 | 示例 |
|--------|------------|------|
| `==` | 值是否相等 | `[1,2] == [1,2]` → True |
| `is` | 是否同一个对象（内存地址） | `[1,2] is [1,2]` → False |

判断 None 用 `is`；判断值用 `==`。

### 3.2 整除是向下取整

```python
print(-10 // 3)    # -4（向下取整，不是 -3！）
print(int(-10/3))  # -3（向零取整）
```

### 3.3 可变对象用 `*` 复制时注意

```python
# 危险：内层列表是共享的！
matrix = [[0] * 3] * 3  # 三个行指向同一个子列表
matrix[0][0] = 1
print(matrix)  # [[1,0,0], [1,0,0], [1,0,0]] — 全变了！

# 正确：
matrix = [[0] * 3 for _ in range(3)]
```

## 4. Demo 样例

```python
# ===== 1. 算术运算符 =====
a, b = 10, 3
print(f"{a} + {b} = {a + b}")    # 13
print(f"{a} / {b} = {a / b}")    # 3.333...
print(f"{a} // {b} = {a // b}")   # 3
print(f"{a} % {b} = {a % b}")     # 1
print(f"{a} ** {b} = {a ** b}")   # 1000

# ===== 2. 逻辑运算符和短路求值 =====
user_name = ""  # 用户没输入
display_name = user_name or "匿名用户"  # or 短路：取第一个真值
print(f"显示名称: {display_name}")      # 匿名用户

# 避免除零错误：左边为 False 时不执行右边
count = 0
if count != 0 and 10 / count > 2:
    print("满足")
else:
    print("跳过除法，安全！")

# ===== 3. 成员运算符 =====
fruits = ["苹果", "香蕉", "橘子"]
print("苹果" in fruits)          # True
d = {"name": "张三", "age": 25}
print("name" in d)               # True（检查键！）
print("张三" in d)               # False（不检查值）

# ===== 4. 三元表达式 =====
score = 85
grade = "优秀" if score >= 90 else ("良好" if score >= 80 else "其他")
print(f"等级: {grade}")  # 良好
```

```
显示名称: 匿名用户
跳过除法，安全！
True

False
等级: 良好
```

## 5. 典型例题（附带答案）

### 例题 1

题目：

判断闰年：能被 4 整除但不能被 100 整除，或者能被 400 整除。

答案：

```python
year = 2000
is_leap = (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0)
print(is_leap)  # True
```

### 例题 2

题目：

用 `or` 短路实现：当用户输入为空时使用默认值 `"匿名"`。

答案：

```python
user_input = ""
name = user_input or "匿名"
print(f"你好，{name}！")  # 你好，匿名！
```

## 6. 易错点

### 易错点 1：== vs is 混淆

```python
a, b = [1,2,3], [1,2,3]
print(a == b)   # True（值相同）
print(a is b)   # False（不同对象！）
# 判断 None 用 is，判断值用 ==
```

### 易错点 2：整除负数向下取整

`-10 // 3` = `-4`（不是 `-3`）。

### 易错点 3：and/or 返回操作数本身，不一定是 bool

```python
result = "hello" and 42  # 42，不是 True！
# 想要 bool 结果用 bool()
print(bool("hello" and 42))  # True
```

### 易错点 4：浮点数不要直接用 == 比较

```python
print(0.1 + 0.2 == 0.3)  # False！
import math
print(math.isclose(0.1 + 0.2, 0.3))  # True
```

### 易错点 5：字典中 in 检查的是键，不是值

```python
d = {"name": "张三"}
print("name" in d)   # True（检查键）
print("张三" in d)   # False（不是值！）
```

## 7. 课后练习（附带答案）

### 练习 1

题目：

不用代码，写出以下表达式结果：`(1) 3+4*2`、`(2) not False and True`、`(3) True or False and False`

答案：

```
(1) 3+4*2 = 3+8 = 11（乘法优先）
(2) not False and True = True and True = True（not优先）
(3) True or (False and False) = True or False = True（and优先于or）
```

### 练习 2

题目：

用链式比较和三元表达式，根据成绩判断优良中差。

答案：

```python
score = 85
grade = ("优秀" if score >= 90 else "良好" if 80 <= score < 90
    else "中等" if 70 <= score < 80 else "及格" if score >= 60 else "不及格")
print(f"等级: {grade}")  # 良好
```

### 练习 3

题目：

用位运算判断一个数是否为偶数。

答案：

```python
def is_even(n):
    return (n & 1) == 0  # 末位为0即偶数

print(is_even(10))  # True
print(is_even(11))  # False
```

## 8. 答题总结

- 算术（`+ - * / // % **`）、比较（`== != > <`）、逻辑（`and or not`）、成员（`in`）、身份（`is`）。
- `==` 比较值，`is` 比较是否同一个对象。判断 None 用 `is None`。
- `and`/`or` 有短路求值，返回操作数本身（不一定是 bool）。
- 浮点数不要用 `==` 直接比较，用 `math.isclose()`。
- 拿不准优先级时**加括号**，括号里的总是先算。
- 三元表达式：`值1 if 条件 else 值2`，适合简单的二选一赋值。
