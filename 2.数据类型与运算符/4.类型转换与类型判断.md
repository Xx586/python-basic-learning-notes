# 类型转换与类型判断

## 1. 知识点定义

在实际编程中，经常需要在不同类型之间转换数据。例如：用户输入是字符串需要转成数字做计算，计算结果要嵌入字符串输出，列表需要转成集合去重。

Python 提供了丰富的内置函数用于**类型转换**（显式转换）和**类型判断**（`type()` / `isinstance()`）。

> 小白理解：
> 类型转换就像把食材从一种形态变成另一种——把整根胡萝卜（int）切成丝（str），把面粉做成面团（list → tuple）。本质上还是那些料，但"类型"变了，能做的事情也变了。

## 2. 语法格式

### 2.1 常用类型转换函数

| 目标类型 | 函数 | 示例 | 结果 |
|---------|------|------|------|
| 整数 | `int(x)` | `int("42")`、`int(3.14)` | `42`、`3`（截断，不四舍五入） |
| 浮点数 | `float(x)` | `float("3.14")`、`float(10)` | `3.14`、`10.0` |
| 字符串 | `str(x)` | `str(100)`、`str([1,2])` | `"100"`、`"[1,2]"` |
| 布尔值 | `bool(x)` | `bool(1)`、`bool("")` | `True`、`False` |
| 列表 | `list(x)` | `list("abc")` | `['a','b','c']` |
| 元组 | `tuple(x)` | `tuple([1,2,3])` | `(1, 2, 3)` |
| 集合 | `set(x)` | `set([1,2,2,3])` | `{1, 2, 3}` |

### 2.2 类型判断

```python
# type() — 获取精确类型
print(type(10))           # <class 'int'>
print(type("hello"))      # <class 'str'>

# isinstance() — 检查是否为某类型（支持子类，推荐！）
print(isinstance(10, int))            # True
print(isinstance(True, int))          # True — bool 是 int 的子类
print(isinstance(10, (int, float)))   # True — 检查多种类型用元组
```

### 2.3 布尔转换规则

以下值转为 `False`（其他都为 `True`）：

```python
bool(0), bool(0.0), bool(""), bool([]), bool(()), bool({}), bool(set()), bool(None)
```

### 2.4 安全转换函数

```python
def safe_int(s, default=0):
    """尝试转为 int，失败返回默认值"""
    try:
        return int(s)
    except (ValueError, TypeError):
        return default

print(safe_int("42"))    # 42
print(safe_int("abc"))   # 0
```

## 3. 重点说明

### 3.1 type() vs isinstance()

| 场景 | 推荐 | 原因 |
|------|------|------|
| 检查某类型（含子类） | `isinstance()` | 支持继承，更 Pythonic |
| 必须精确匹配 | `type()` | 严格匹配 |
| 检查 bool 和 int | `isinstance(x, bool)` 先检查！ | bool 是 int 子类 |

### 3.2 int("3.14") 直接报错

```python
# int("3.14")           # ValueError!
num = int(float("3.14"))  # 3 — 先转 float 再转 int
```

### 3.3 隐式类型转换

```python
result = 10 + 3.14       # 13.14 — int + float → float
result = 10 / 3          # 3.333... — 除法结果总是 float
print(True + 1)          # 2 — 布尔在数值运算中当做 0/1
```

## 4. Demo 样例

```python
# ===== 1. 基本类型转换 =====
print("int('42') =", int("42"))          # 42
print("float('3.14') =", float("3.14"))  # 3.14
print("str(100) =", str(100))            # "100"
print("bool(1) =", bool(1), "bool(0) =", bool(0))  # True, False

# ===== 2. 容器转换 =====
print("list('abc') =", list("abc"))              # ['a','b','c']
print("tuple([1,2,3]) =", tuple([1,2,3]))       # (1,2,3)
print("set([1,2,2,3]) =", set([1,2,2,3]))      # {1,2,3} 自动去重

# ===== 3. 类型判断 =====
x = 10
print(f"isinstance(x, int): {isinstance(x, int)}")     # True
print(f"isinstance(True, int): {isinstance(True, int)}") # True
print(f"type(True) == int: {type(True) == int}")        # False

# ===== 4. 用户输入转换 =====
user_input = "3.14"  # 模拟 input 返回的字符串
price = float(user_input)
print(f"价格: {price}, 类型: {type(price)}")  # 价格: 3.14, 类型: <class 'float'>
```

运行结果：


## 5. 典型例题（附带答案）

### 例题 1

题目：

编写函数 `describe(x)`，根据参数类型返回不同描述。注意：bool 要优先判断。

答案：

```python
def describe(x):
    if isinstance(x, bool):           # 先判断 bool（它是 int 的子类）
        return "布尔值"
    elif isinstance(x, int):
        return "整数"
    elif isinstance(x, float):
        return "浮点数"
    elif isinstance(x, str):
        return "字符串"
    elif isinstance(x, list):
        return "列表"
    else:
        return f"其他类型: {type(x).__name__}"

print(describe(True))   # 布尔值
print(describe(10))     # 整数
```

### 例题 2

题目：

写一个安全除法函数 `safe_divide(a, b)`，将参数转为浮点数再除。失败返回 None。

答案：

```python
def safe_divide(a, b):
    try:
        return float(a) / float(b)
    except (ValueError, TypeError, ZeroDivisionError):
        return None

print(safe_divide("10", "2"))  # 5.0
print(safe_divide("abc", "2")) # None
```

## 6. 易错点

### 易错点 1：int("3.14") 直接报错

错误写法：`int("3.14")` → ValueError。正确写法：`int(float("3.14"))`。

### 易错点 2：浮点数截断不是四舍五入

```python
print(int(3.99))   # 3（截断！不是 4）
print(round(3.99)) # 4（四舍五入用 round）
```

### 易错点 3：bool 是 int 的子类

`isinstance(True, int)` 返回 True！判断时先检查 `isinstance(x, bool)` 再检查 `isinstance(x, int)`。

### 易错点 4：isinstance 检多类型用元组

```python
isinstance(x, (int, float))  # ✅ 正确
# isinstance(x, [int, float]) # ❌ TypeError!
```

## 7. 课后练习（附带答案）

### 练习 1

题目：

模拟用户输入两个数字（字符串形式），转成浮点数相加。

答案：

```python
a, b = "3.5", "2.5"  # 模拟 input
result = float(a) + float(b)
print(f"{a} + {b} = {result}")  # 3.5 + 2.5 = 6.0
```

### 练习 2

题目：

列表 `[1,2,2,3,3,3,4]` 去重后转成元组。

答案：

```python
nums = [1, 2, 2, 3, 3, 3, 4]
result = tuple(set(nums))
print(result)  # (1, 2, 3, 4)
```

### 练习 3

题目：

给定列表 `data = [0, 1, "", "hello", [], None, 3.14]`，过滤出真值和假值。

答案：

```python
data = [0, 1, "", "hello", [], None, 3.14]
true_vals = [x for x in data if x]   # 真值
false_vals = [x for x in data if not x]  # 假值
print(f"真值: {true_vals}")   # [1, 'hello', 3.14]
print(f"假值: {false_vals}")  # [0, '', [], None]
```

## 8. 答题总结

- 类型转换用 `int()`/`float()`/`str()`/`bool()`/`list()`/`tuple()`/`set()`。
- 类型判断推荐 `isinstance()`（支持子类），需要精确匹配用 `type()`。
- `int("3.14")` 会报错，需先 `float()` 再 `int()`；`int(3.99)` 截断为 3。
- `bool` 是 `int` 子类，判断顺序很重要：先判断 `isinstance(x, bool)`。
- 空容器（`""`, `[]`, `{}`, `set()`）、`0`、`None` 在布尔上下文中为 `False`。
- 用户输入永远是字符串，做数学计算前必须用 `int()` 或 `float()` 转换。
