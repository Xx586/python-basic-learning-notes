# 基本数据类型

## 1. 知识点定义

数据类型决定了数据在计算机中如何存储、以及可以对数据执行哪些操作。Python 是一种**动态类型语言**，变量不需要声明类型，解释器会根据赋值自动推断类型。

Python 的基本数据类型包括：

- **int（整数）**：没有大小限制（只受内存限制）
- **float（浮点数）**：小数，使用 IEEE 754 双精度标准
- **bool（布尔值）**：只有 `True` 和 `False`，是 int 的子类
- **str（字符串）**：不可变的字符序列
- **NoneType（空值）**：只有一个值 `None`，表示"没有值"

> 小白理解：
> 数据类型就像物品的分类标签。苹果是"水果"，书本是"文具"。计算机也需要知道一个数据是"数字"还是"文字"，因为数字可以做加减法，文字可以拼接，它们能做的事情不一样。

## 2. 语法格式

### 2.1 int（整数）

```python
a = 10          # 直接赋值
b = int(3.14)   # 通过 int() 转换，结果为 3（截断小数）
c = 0xFF        # 十六进制，值为 255
d = 0b1010      # 二进制，值为 10
e = 1_000_000   # 使用下划线增强可读性，值为 1000000
```

常见操作：

```python
x = 10 + 3    # 13 — 加
x = 10 - 3    # 7  — 减
x = 10 * 3    # 30 — 乘
x = 10 / 3    # 3.333... — 除（结果总是 float）
x = 10 // 3   # 3  — 整除
x = 10 % 3    # 1  — 取余
x = 10 ** 3   # 1000 — 乘方

abs(-5)       # 5  — 绝对值
pow(2, 3)     # 8  — 等价于 2 ** 3
divmod(10, 3) # (3, 1) — 同时返回商和余数
```

### 2.2 float（浮点数）

```python
a = 3.14
b = float(10)    # 10.0
c = 1.5e3        # 科学计数法，值为 1500.0
```

### 2.3 bool（布尔值）

`True` 和 `False`（注意**首字母大写**）。bool 是 int 的子类，`True == 1`、`False == 0`。

### 2.4 str（字符串）

```python
s1 = 'hello'     # 单引号
s2 = "world"     # 双引号（和单引号没有区别）
s3 = str(123)    # 通过 str() 转换，结果为 "123"
```

### 2.5 NoneType（空值）

```python
result = None    # 表示"没有值"或"未知"
```

## 3. 重点说明

### 3.1 浮点数精度问题（重要！）

```python
print(0.1 + 0.2)            # 0.30000000000000004，不是精确的 0.3！
print(0.1 + 0.2 == 0.3)     # False！

# 解决方案1：使用 Decimal
from decimal import Decimal
print(Decimal('0.1') + Decimal('0.2'))  # 0.3

# 解决方案2：使用 math.isclose
import math
print(math.isclose(0.1 + 0.2, 0.3))    # True
```

### 3.2 布尔值在运算中当做 0/1

```python
print(True + 1)   # 2
print(False * 100)  # 0
print(isinstance(True, int))  # True — bool 是 int 的子类
```

### 3.3 Python 是动态类型语言

```python
x = 10          # x 是 int
x = "hello"     # x 现在是 str — 变量类型可以改变
```

### 3.4 类型对比

| 类型 | 示例 | 特点 |
|------|------|------|
| int | `42`, `-7`, `0xFF` | 整数，无大小限制 |
| float | `3.14`, `1.5e3` | 小数，双精度 |
| bool | `True`, `False` | 只有两个值，int 子类 |
| str | `"hello"` | 不可变字符序列 |
| NoneType | `None` | 唯一值，表示空 |

## 4. Demo 样例

```python
# -*- coding: utf-8 -*-
"""
基本数据类型演示
"""

# ===== 1. 整数 int =====
print("===== int 演示 =====")
a = 10
b = 3
print(f"{a} + {b} = {a + b}")    # 13
print(f"{a} / {b} = {a / b}")    # 3.333...（除法结果总是 float）
print(f"{a} // {b} = {a // b}")   # 3（整除）
print(f"{a} % {b} = {a % b}")     # 1（取余）
print(f"{a} ** {b} = {a ** b}")   # 1000（乘方）

# ===== 2. 浮点数 float =====
print("\n===== float 演示 =====")
pi = 3.14159
radius = 5
area = pi * radius ** 2  # 计算圆的面积
print(f"半径为 {radius} 的圆，面积约为 {area:.2f}")

# ===== 3. 浮点数精度问题 =====
print("\n===== 浮点数精度 =====")
print(f"0.1 + 0.2 = {0.1 + 0.2}")           # 0.30000000000000004
print(f"0.1 + 0.2 == 0.3: {0.1 + 0.2 == 0.3}")  # False
import math
print(f"用 isclose 比较: {math.isclose(0.1 + 0.2, 0.3)}")  # True

# ===== 4. 布尔值 bool =====
print("\n===== bool 演示 =====")
is_pass = True
print(f"及格了吗？{is_pass}")
print(f"True + 1 = {True + 1}")     # 2（bool 参与运算）
print(f"False * 100 = {False * 100}")  # 0

# ===== 5. NoneType =====
print("\n===== None 演示 =====")
result = None
print(f"result 的值是: {result}")
print(f"result 是 None 吗？{result is None}")  # True
```

## 5. 典型例题（附带答案）

### 例题 1

题目：

以下代码的输出是什么？

```python
print(type(10))
print(type(3.14))
print(type(True))
print(isinstance(True, int))
print(True + False)
```

答案：

```
<class 'int'>
<class 'float'>
<class 'bool'>
True
1
```

解析：`type()` 返回精确类型，`isinstance(True, int)` 为 True 因为 bool 是 int 的子类。`True + False` = 1 + 0 = 1。

### 例题 2

题目：

`0.1 + 0.2 == 0.3` 的结果是什么？为什么？如何正确处理浮点数比较？

答案：

结果是 `False`。因为浮点数在二进制中无法精确表示 0.1 和 0.2，实际计算结果是 `0.30000000000000004`。

正确处理浮点数比较：

```python
import math
print(math.isclose(0.1 + 0.2, 0.3))  # True
```

## 6. 易错点

### 易错点 1：浮点数精度陷阱

错误现象：`0.1 + 0.2 != 0.3`，输出 `False`。

错误原因：浮点数在计算机中用二进制存储，某些十进制小数无法精确表示。

正确做法：使用 `math.isclose()` 进行近似比较。

### 易错点 2：除法的结果总是 float

错误现象：

```python
result = 10 / 2  # 5.0 不是 5！
print(type(result))  # <class 'float'>
```

错误原因：Python 3 中 `/` 运算符的结果总是浮点数。

正确做法：如果希望得到整数，使用 `//`（整除）。

### 易错点 3：bool 是 int 的子类

错误现象：

```python
print(isinstance(True, int))  # True — 容易误判
```

正确做法：如果要区分 bool 和 int，应先检查 bool：

```python
def judge(x):
    if isinstance(x, bool):  # 先检查 bool！
        return "bool"
    elif isinstance(x, int):
        return "int"
```

### 易错点 4：True/False 首字母大写

错误现象：

```python
is_ok = true   # NameError: name 'true' is not defined
```

错误原因：Python 的布尔值是 `True` 和 `False`（首字母大写），不是 `true`/`false`。

正确做法：

```python
is_ok = True   # 首字母大写
```

## 8. 答题总结

- Python 基本数据类型：int（整数）、float（小数）、bool（True/False）、str（字符串）、NoneType（None）。
- Python 是动态类型语言，变量不需要声明类型，可以随时改变。
- `int` 没有大小限制；`float` 存在精度问题，比较时用 `math.isclose()`。
- `bool` 是 `int` 的子类（`True==1`，`False==0`），判断时先用 `isinstance(x, bool)`。
- `/` 的结果总是 float；`//` 才是整除（向下取整）。
- `None` 判断用 `is None`，不要用 `== None`。
