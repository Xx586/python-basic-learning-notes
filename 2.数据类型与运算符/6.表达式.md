# 表达式

## 1. 知识点定义

**表达式**（Expression）是由**值、变量、运算符**组合而成的、能够被 Python 解释器计算出**一个结果**的代码片段。简单说：任何能算出"一个值"的代码都是表达式。

它主要解决的问题是：
- 让程序能进行计算（数学计算、逻辑判断、数据比较等）
- 把计算逻辑写成简洁、可读的形式
- 组合运算符和数据来产生新的数据

在实际写 Python 代码时，表达式无处不在——赋值号右边是表达式、`if` 条件里是表达式、`print` 括号里也可以是表达式。

> 小白理解：
> 表达式就像数学课上的"算式"。`1 + 2` 是一个算式，算出结果是 `3`。在 Python 里，`1 + 2` 就是表达式，算出来的 `3` 是表达式的值。`x > 10` 也是表达式，算出结果是 `True` 或 `False`。

## 2. 语法格式

### 2.1 表达式的基本组成

```python
# 表达式由以下元素组合而成：
# 值（字面量）、变量、运算符、函数调用

# 最简单的表达式：一个值本身
100                 # 表达式的值是 100
"hello"             # 表达式的值是 "hello"
True                # 表达式的值是 True

# 变量也是表达式
name                # 表达式的值是变量 name 中保存的数据

# 运算符组成的表达式
10 + 20             # 算术表达式，值是 30
x > 5               # 比较表达式，值是 True 或 False
a and b             # 逻辑表达式
x in [1, 2, 3]     # 成员判断表达式

# 函数调用也是表达式
len("hello")        # 函数调用表达式，值是 5
input("请输入：")    # 函数调用表达式，值是用户输入的字符串
```

### 2.2 表达式 vs 语句

| 对比维度 | 表达式（Expression） | 语句（Statement） |
|----------|---------------------|-------------------|
| 有无结果值 | **有**，一定能算出一个值 | **没有**，执行一个动作 |
| 能否放在赋值号右边 | ✅ 可以 | ❌ 不可以 |
| 能否放在 print() 里 | ✅ 可以 | ❌ 不可以 |
| 举例 | `1 + 2`, `x * 3`, `len(s)` | `if x > 0:`, `for i in lst:`, `import os` |

```python
# 表达式：有结果值，可以放在赋值号右边
result = 10 + 20          # ✅ 10 + 20 是表达式，值是 30
result = x * 2 + 1        # ✅ 整个右边是表达式
print(len("hello"))       # ✅ len("hello") 是表达式，print 可以用它的值

# 语句：没有结果值，执行一个动作
# x = if True: ...        # ❌ if 是语句，不能放在赋值号右边
# print(for i in lst:)    # ❌ for 也是语句
```

## 3. 重点说明

### 3.1 表达式的分类

| 类别 | 说明 | 示例 | 结果类型 |
|------|------|------|----------|
| **字面量表达式** | 直接写值本身 | `42`, `"hello"`, `True` | 各种类型 |
| **算术表达式** | 数学运算 | `a + b * c`, `x ** 2` | 数字（int/float） |
| **比较表达式** | 比较大小 | `x > 10`, `name == "张三"` | bool |
| **逻辑表达式** | 逻辑组合 | `x > 0 and x < 100` | bool |
| **成员表达式** | 判断是否在里面 | `"a" in "apple"`, `3 in [1,2,3]` | bool |
| **函数调用表达式** | 调用函数 | `len("abc")`, `int("10")` | 取决于函数返回值 |

### 3.2 表达式嵌套

表达式可以嵌套，即一个表达式里面可以包含另一个表达式：

```python
# 嵌套表达式：从内向外计算
result = (a + b) * (c - d)    # 括号内的先算
result = max(a, b) + min(c, d) # 函数调用也是表达式，可以参与运算

# 复杂嵌套示例
x = 10
y = 20
z = (x + y) * 2 > 50 and x != 0
# 计算顺序：
# 1. x + y = 30
# 2. 30 * 2 = 60
# 3. 60 > 50 = True
# 4. x != 0 = True
# 5. True and True = True
```

### 3.3 计算顺序（优先级）

Python 按照运算符优先级来计算表达式：

```text
括号 () → 乘方 ** → 乘除 // */% → 加减 +- → 比较 > < == → 逻辑 not → and → or
```

口诀：**先括号，再乘方，再乘除，再加减，再比较，最后逻辑。**

### 3.4 与其他章节的关系

表达式是一个贯穿整个 Python 学习的概念：

- **第 1 章**：`print(表达式)` 中括号里就是表达式
- **第 2 章**：运算符章节讲各种运算符，本章讲它们如何组合成表达式
- **第 3 章**：`if 表达式:` 条件是一个表达式，结果是 `True` 或 `False`
- **第 4 章**：`return 表达式` 返回表达式的值
- **第 5 章**：方法调用 `obj.method()` 也是表达式

## 4. Demo 样例

### Demo 1：认识各种表达式

```python
# ===== 字面量表达式 =====
print(100)              # 值是 100
print("Python")         # 值是 "Python"

# ===== 算术表达式 =====
a = 10
b = 3
print(a + b)            # 13，加法表达式
print(a / b)            # 3.333...，除法表达式
print(a // b)           # 3，整除表达式
print(a ** b)           # 1000，乘方表达式

# ===== 比较表达式 =====
score = 85
print(score >= 60)      # True，判断是否及格
print(score == 100)     # False，判断是否满分

# ===== 逻辑表达式 =====
age = 20
has_id = True
can_enter = age >= 18 and has_id  # 两个条件都满足
print("可以进入：", can_enter)    # True

# ===== 成员表达式 =====
fruits = ["苹果", "香蕉", "橘子"]
print("苹果" in fruits)    # True，苹果在列表里
print("西瓜" in fruits)    # False，西瓜不在列表里

# ===== 函数调用表达式 =====
text = "Hello Python"
print(len(text))           # 12，len() 返回字符串长度
print(text.upper())        # "HELLO PYTHON"，方法调用也是表达式
```

运行结果：

```text
100
Python
13
3.3333333333333335
3
1000
True
False
可以进入： True
True
False
12
HELLO PYTHON
```

### Demo 2：表达式的组合与嵌套

```python
# 表达式的值可以赋给变量
x = 10           # 右边 10 是表达式
y = x * 2 + 5    # 右边整个是算术表达式，值是 25
print("y =", y)

# 比较表达式的结果赋给变量
is_adult = y >= 18   # 比较表达式，值是 True
print("成年：", is_adult)

# 表达式嵌套：括号里的表达式先计算
result = (10 + 5) * (20 - 8)   # 括号先算：15 * 12 = 180
print("result =", result)

# 表达式用在 print 里
name = "小明"
print("你好，" + name)           # 字符串拼接表达式
print(f"长度是：{len(name)}")    # f-string 里 { } 也是表达式

# 表达式用在条件里（表达式结果为 True/False）
num = 42
if num % 2 == 0:                # 整个是表达式，检查偶数
    print(num, "是偶数")
else:
    print(num, "是奇数")
```

运行结果：

```text
y = 25
成年： True
result = 180
你好，小明
长度是：2
42 是偶数
```

## 5. 典型例题（附带答案）

### 例题 1

题目：

以下哪些是表达式？哪些是语句？

```python
A. 10 + 20
B. x = 10
C. x > 5
D. if x > 5:
E. len("hello")
F. for i in range(10):
```

答案：

| 选项 | 表达式/语句 | 说明 |
|------|------------|------|
| A. `10 + 20` | ✅ 表达式 | 算出结果 30 |
| B. `x = 10` | ❌ 语句 | 赋值语句，不产生结果值 |
| C. `x > 5` | ✅ 表达式 | 算出结果 True 或 False |
| D. `if x > 5:` | ❌ 语句 | if 条件语句本身 |
| E. `len("hello")` | ✅ 表达式 | 算出结果 5 |
| F. `for i in range(10):` | ❌ 语句 | for 循环语句 |

### 例题 2

题目：

请写出以下表达式的计算结果：

```python
a = 15
b = 4

(1) a + b * 2
(2) (a + b) * 2
(3) a // b
(4) a > 10 and b < 5
(5) "Py" + "thon"
```

答案：

```python
(1) 15 + 4 * 2 = 15 + 8 = 23        # 先乘后加
(2) (15 + 4) * 2 = 19 * 2 = 38      # 括号先算
(3) 15 // 4 = 3                      # 整除（向下取整）
(4) True and True = True             # 两边的比较表达式都是 True
(5) "Python"                         # 字符串拼接
```

## 6. 易错点

### 易错点 1：混淆表达式和语句

错误写法：

```python
result = if x > 0:    # SyntaxError，if 是语句不能赋值
```

错误原因：`if` 是语句，没有"返回值"，不能放在赋值号右边。

正确写法：

```python
result = "正数" if x > 0 else "非正数"  # 三元表达式，有返回值
```

### 易错点 2：赋值表达式的位置

错误写法：

```python
print(x = 10)    # TypeError，x = 10 是赋值语句不是表达式
```

错误原因：`print` 括号里需要一个值，但 `x = 10` 是语句，不返回值。

正确写法：

```python
x = 10
print(x)         # 先赋值，再打印
```

### 易错点 3：运算符优先级搞混

错误写法：

```python
result = 10 + 5 * 2    # 你以为：(10+5)*2 = 30
print(result)           # 实际上：10+10 = 20（乘法优先级更高）
```

错误原因：忘了 `*` 比 `+` 先算。

正确写法：

```python
result = (10 + 5) * 2   # 加括号明确优先级
print(result)            # 输出：30
```

> 口诀：拿不准优先级的时候就**加括号**，括号里的总是先算。

## 7. 课后练习（附带答案）

### 练习 1

题目：

判断以下说法是否正确：

1. 表达式一定能计算出一个值。
2. `x = 5` 是一个表达式。
3. `len("abc")` 是一个表达式。
4. 表达式可以放在 `print()` 里面。

答案：

1. ✅ 正确。表达式一定有计算结果。
2. ❌ 错误。`x = 5` 是赋值语句，不返回值。
3. ✅ 正确。函数调用是表达式，返回整数 3。
4. ✅ 正确。`print(len("abc"))` 是可以的，`len("abc")` 的值 3 会传给 `print`。

### 练习 3

题目：

编写程序，计算一元二次方程 `y = x² + 2x + 1` 在 x=3 时的值，并把整个计算过程写成表达式。

答案：

```python
x = 3
y = x ** 2 + 2 * x + 1   # 表达式：3² + 2×3 + 1 = 9 + 6 + 1 = 16
print("当 x =", x, "时，y =", y)
```

运行结果：

```text
当 x = 3 时，y = 16
```

说明：整个右边 `x ** 2 + 2 * x + 1` 是算术表达式，Python 按优先级先算 `x ** 2`（乘方），再算 `2 * x`（乘法），最后算加法。

## 8. 答题总结

- 表达式是由值、变量、运算符、函数调用组成的、能算出**一个结果**的代码片段。
- 表达式和语句的区别：表达式有值，语句没有值。表达式可以放在赋值号右边，语句不行。
- 表达式可以嵌套，计算顺序由优先级和括号决定。
- 表达式的分类：算术表达式、比较表达式、逻辑表达式、成员表达式、函数调用表达式。
- 拿不准优先级时加括号，括号里的总是先算。
