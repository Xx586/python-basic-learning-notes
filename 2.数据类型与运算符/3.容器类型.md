# 容器类型

## 1. 知识点定义

容器类型是 Python 中用来**存放多个数据**的数据结构。Python 有四种核心容器类型：列表（list）、元组（tuple）、字典（dict）、集合（set）。

| 特性 | list | tuple | dict | set |
|------|------|-------|------|-----|
| 写法 | `[]` | `()` | `{}` (键值对) | `{}` / `set()` |
| 有序 | ✅ | ✅ | ✅（3.7+） | ❌ |
| 可变 | ✅ | ❌ | ✅ | ✅ |
| 重复元素 | ✅ | ✅ | 键不允许 | ❌ |
| 索引访问 | `[0]` | `[0]` | `[key]` | ❌ |
| 典型场景 | 有序集合 | 不变的记录 | 键值映射 | 去重、集合运算 |

> 小白理解：
> - **列表** = 购物清单（可以增删改，有顺序）
> - **元组** = 日历上的节日（固定不变的记录）
> - **字典** = 真实的字典（通过"词"查"释义"）
> - **集合** = 钱包里的硬币（不重复，没顺序）

## 2. 语法格式

### 2.1 列表（list）

```python
# 创建
nums = [1, 2, 3]
mixed = [1, "hello", 3.14, True]

# 增删改查
nums.append(4)            # 末尾添加 → [1,2,3,4]
nums.insert(0, 0)         # 指定位置插入 → [0,1,2,3,4]
nums.remove(3)            # 删除第一个匹配值 → [0,1,2,4]
popped = nums.pop()       # 删除并返回末尾 → 4, nums=[0,1,2]
nums[0] = 100             # 直接赋值修改
print(nums[0])            # 索引访问 → 100
print(nums[1:3])          # 切片 → [1, 2]

# 排序
sorted_nums = sorted(nums)               # 返回新列表，原列表不变
nums.sort()                              # 原地排序，直接修改
nums.sort(key=len)                       # 自定义排序规则
```

### 2.2 元组（tuple）

```python
# 创建
t1 = (1, 2, 3)
t2 = 1, 2, 3            # 括号可省略
t3 = (1,)               # 单元素元组必须有逗号！
t4 = (1)                # 这不是元组，是整数 1！

# 不可变性
# t1[0] = 100           # TypeError! 元组不支持修改

# 解包
point = (10, 20)
x, y = point            # x=10, y=20
a, b = b, a             # 一行交换两个变量！
first, *rest = (1,2,3,4)  # first=1, rest=[2,3,4]
```

### 2.3 字典（dict）

```python
# 创建
person = {"name": "张三", "age": 25}

# 增改查删
person["city"] = "北京"              # 新增键值对
person["age"] = 26                   # 修改已有键
print(person["name"])                # 直接访问（键不存在报 KeyError）
print(person.get("city", "未知"))    # 安全访问（不存在返回默认值）
person.pop("city")                   # 删除并返回值
del person["age"]                    # 删除键值对

# 遍历
for key in person:                   # 遍历键
    print(key, person[key])
for key, value in person.items():    # 遍历键值对（推荐）
    print(f"{key} => {value}")
```

### 2.4 集合（set）

```python
# 创建（注意：空集合用 set()，{} 是空字典）
s1 = {1, 2, 3}
s2 = set([1, 2, 2, 3, 3])  # {1, 2, 3}，自动去重
empty = set()               # 空集合

# 增删
s1.add(4)                   # 添加
s1.discard(3)               # 安全删除（不存在不报错）

# 集合运算
a, b = {1, 2, 3, 4}, {3, 4, 5, 6}
print(a & b)   # {3, 4} — 交集
print(a | b)   # {1, 2, 3, 4, 5, 6} — 并集
print(a - b)   # {1, 2} — 差集
```

## 3. 重点说明

### 3.1 如何选择容器？

- **要按顺序放一组数据，且有增删改查** → list
- **一组不希望被修改的常量**（如星期名、颜色值） → tuple
- **通过某个键快速查找对应值** → dict
- **要去重、或做交集/并集运算** → set

### 3.2 列表是可变对象——小心"共享引用"

```python
a = [1, 2, 3]
b = a             # b 和 a 指向同一个列表！
b.append(4)
print(a)          # [1, 2, 3, 4] — a 也被改了！

# 正确：用 copy() 或 [:] 创建副本
b = a.copy()
b = a[:]
```

## 4. Demo 样例

```python
# ===== 列表演示 =====
fruits = ["苹果", "香蕉"]
fruits.append("橘子")                  # 添加元素
fruits.insert(1, "葡萄")              # 在索引1插入
print(f"列表: {fruits}")              # ['苹果', '葡萄', '香蕉', '橘子']

# ===== 元组演示 =====
point = (3, 4)
x, y = point                          # 解包
print(f"坐标: x={x}, y={y}")

# ===== 字典演示 =====
student = {"name": "小明", "score": 85}
student["grade"] = "A"                # 新增
print(f"学生: {student}")
print(f"姓名: {student.get('name', '未知')}")

# 遍历字典
for key, value in student.items():
    print(f"  {key}: {value}")

# ===== 集合演示 =====
nums = [1, 2, 2, 3, 3, 3, 4]
unique = list(set(nums))              # 去重
print(f"去重后: {unique}")            # [1, 2, 3, 4]

a, b = {1, 2, 3}, {2, 3, 4}
print(f"交集: {a & b}")               # {2, 3}
print(f"并集: {a | b}")               # {1, 2, 3, 4}
```

## 5. 典型例题（附带答案）

### 例题 1

题目：

统计字符串 `"apple banana apple orange banana apple"` 中每个单词出现的次数，用字典存储。

答案：

```python
text = "apple banana apple orange banana apple"
words = text.split()
freq = {}
for word in words:
    freq[word] = freq.get(word, 0) + 1
print(freq)  # {'apple': 3, 'banana': 2, 'orange': 1}
```

### 例题 2

题目：

用元组解包交换两个变量 a=100, b=200 的值。

答案：

```python
a, b = 100, 200
a, b = b, a
print(a, b)  # 200 100
```

## 6. 易错点

### 易错点 1：列表共享引用

错误写法：`b = a`（两个变量指向同一个列表），改 b 会影响 a。

正确写法：`b = a.copy()` 或 `b = a[:]`。

### 易错点 2：单元素元组忘写逗号

```python
t = (1)     # int！不是元组！
t = (1,)    # 这才是元组
```

### 易错点 3：空集合用 `set()` 而不是 `{}`

```python
s = {}       # 空字典！
s = set()    # 空集合
```

### 易错点 4：字典的键必须是不可变类型

```python
# d = {[1, 2]: "value"}  # TypeError! 列表不可哈希
d = {(1, 2): "value"}    # 元组可以作为键
```

## 7. 课后练习（附带答案）

### 练习 1

题目：

给定列表 `nums = [4, 2, 2, 8, 3, 3, 1, 5]`，得到一个去重后按降序排列的列表。

答案：

```python
nums = [4, 2, 2, 8, 3, 3, 1, 5]
result = sorted(set(nums), reverse=True)
print(result)  # [8, 5, 4, 3, 2, 1]
```

### 练习 2

题目：

找两个列表 `a=[1,2,3,4,5]` 和 `b=[4,5,6,7,8]` 的共同元素。

答案：

```python
a, b = [1, 2, 3, 4, 5], [4, 5, 6, 7, 8]
common = list(set(a) & set(b))
print(common)  # [4, 5]
```

### 练习 3

题目：

用字典存储学生信息，完成增删改查操作。

答案：

```python
students = {"001": {"name": "张三", "score": 85}}
students["003"] = {"name": "王五", "score": 78}  # 增
del students["001"]                               # 删
students.get("002", {}).get("score", "不存在")     # 查
```

## 8. 答题总结

- 四大容器：list（可变有序）、tuple（不可变有序）、dict（键值对）、set（无序不重复）。
- list 最常用，dict 第二常用，tuple 适合存不变数据，set 用来去重和集合运算。
- 列表是可变对象，`b = a` 是共享引用，要用 `copy()` 创建独立副本。
- 单元素元组必须有逗号 `(1,)`；空集合用 `set()` 而非 `{}`。
- 字典用 `get(key, default)` 安全访问，避免 KeyError。
