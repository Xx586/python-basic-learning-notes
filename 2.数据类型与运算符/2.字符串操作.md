# 字符串操作

## 1. 知识点定义

Python 中的字符串 `str` 是**不可变**（immutable）的**字符序列**。一旦创建，就不能原地修改——所有字符串方法都会**返回新字符串**，原字符串不变。

字符串是 Python 中用得最多的数据类型。处理用户输入、读取文件内容、Web 页面文本、日志分析等，都离不开字符串操作。

> 小白理解：
> 字符串就像写在纸上的字——你不能把纸上的"错"字擦掉改成"对"字，只能换一张纸重新写。Python 的字符串也是同理：所有"修改"操作实际上是创建了一个新字符串。

## 2. 语法格式

### 2.1 字符串创建

```python
s1 = 'hello'                 # 单引号
s2 = "world"                 # 双引号（和单引号没有区别）
s3 = '''多行
字符串'''                    # 三引号（保留换行）
s4 = r"C:\Users\name"        # 原始字符串，\ 不转义
s5 = str(123)                # 通过 str() 转换，结果为 "123"
```

### 2.2 索引

字符串中每个字符都有位置编号，从 **0** 开始：

```python
s = "Python"
# 正向索引： 0    1    2    3    4    5
# 字符：     P    y    t    h    o    n
# 反向索引：-6   -5   -4   -3   -2   -1

print(s[0])    # 'P'
print(s[-1])   # 'n'
```

### 2.3 切片

语法：`s[start:end:step]`

```python
s = "Hello World"

print(s[0:5])      # 'Hello'（取索引 0 到 4）
print(s[6:])       # 'World'（省略 end，到末尾）
print(s[::2])      # 'HloWrd'（每隔一个取一个）
print(s[::-1])     # 'dlroW olleH'（反转字符串，经典技巧！）
```

> 切片不会越界：`s[0:100]` 不会报错，返回整个字符串。

### 2.4 f-string 格式化（推荐）

```python
name = "小明"
age = 25
print(f"我叫{name}，今年{age}岁")       # 基本用法
print(f"明年我{age + 1}岁")             # 表达式
print(f"价格: {19.999:.2f}")            # 保留两位小数
print(f"{'左对齐':<10}|")               # 对齐
```

### 2.5 .format() 方法

```python
print("我叫{}，今年{}岁".format("小明", 25))
print("我叫{name}，今年{age}岁".format(name="小明", age=25))
```

### 2.6 常用字符串方法

| 方法 | 作用 | 示例 |
|------|------|------|
| `split(",")` | 分割 → 列表 | `"a,b,c".split(",")` → `['a','b','c']` |
| `join(seq)` | 列表 → 字符串 | `",".join(['a','b'])` → `"a,b"` |
| `strip()` | 去两端空白 | `" hi ".strip()` → `"hi"` |
| `replace(old, new)` | 替换（返回新串） | `"hello".replace("h","H")` → `"Hello"` |
| `find("x")` | 查找子串索引 | `"abc".find("b")` → `1`（找不到返回 -1） |
| `startswith("x")` | 判断前缀 | `"file.pdf".endswith(".pdf")` → `True` |
| `upper()` / `lower()` | 大小写转换 | `"Hi".lower()` → `"hi"` |
| `count("x")` | 计数 | `"banana".count("a")` → `3` |

## 3. 重点说明

### 3.1 字符串是不可变类型

```python
s = "hello"
s.replace("h", "H")  # 返回值被丢弃了！
print(s)             # 还是 "hello"！

# 正确做法：接收返回值
s = s.replace("h", "H")
print(s)             # "Hello"
```

### 3.2 切片索引不会越界，单元素索引会

```python
s = "hello"
print(s[0:100])   # 'hello'（不报错）
print(s[100])     # IndexError!
```

### 3.3 大量字符串拼接用 join，不用 +

```python
# 不推荐（每次 + 都创建新字符串）
result = ""
for i in range(1000):
    result += str(i)

# 推荐（使用 join 一次完成）
parts = []
for i in range(1000):
    parts.append(str(i))
result = "".join(parts)
```

## 4. Demo 样例

```python
# 字符串操作综合演示
s = "  Hello, Python World!  "

# 1. 去除首尾空格
cleaned = s.strip()
print(f"去除空格后: '{cleaned}'")  # 'Hello, Python World!'

# 2. 大小写转换
print(f"大写: {cleaned.upper()}")    # 'HELLO, PYTHON WORLD!'
print(f"小写: {cleaned.lower()}")    # 'hello, python world!'

# 3. 分割与连接
words = cleaned.split()  # 按空白分割
print(f"分割结果: {words}")  # ['Hello,', 'Python', 'World!']
joined = " - ".join(words)
print(f"连接结果: {joined}")  # 'Hello, - Python - World!'

# 4. 查找与替换
print(f"'Python' 的位置: {cleaned.find('Python')}")  # 7
new_s = cleaned.replace("World", "编程")
print(f"替换后: {new_s}")  # 'Hello, Python 编程!'

# 5. 切片操作
text = "Python编程"
print(f"前6个字符: {text[:6]}")      # 'Python'
print(f"反转: {text[::-1]}")         # '程编nohtyP'

# 6. f-string 格式化
name = "小明"
score = 95.567
print(f"姓名: {name}, 分数: {score:.1f}")  # 姓名: 小明, 分数: 95.6
```

运行结果：

```
去除空格后: 'Hello, Python World!'
大写: 'HELLO, PYTHON WORLD!'
小写: 'hello, python world!'
分割结果: ['Hello,', 'Python', 'World!']
连接结果: 'Hello, - Python - World!'
'Python' 的位置: 7
替换后: 'Hello, Python 编程!'
前6个字符: 'Python'
反转: '程编nohtyP'
姓名: 小明, 分数: 95.6
```

## 5. 典型例题（附带答案）

### 例题 1

题目：

给定邮箱地址 `"user@example.com"`，请提取出 `@` 后面的域名部分。

答案：

```python
email = "user@example.com"

# 方法1：使用 split
domain = email.split("@")[1]
print(domain)  # example.com

# 方法2：使用 find + 切片
at_index = email.find("@")
domain = email[at_index + 1:]
print(domain)  # example.com
```

### 例题 2

题目：

判断字符串是否为回文串（正读反读一样），忽略大小写和空格。例如 `"A man a plan a canal Panama"` 是回文串。

答案：

```python
def is_palindrome(s):
    cleaned = s.lower().replace(" ", "")
    return cleaned == cleaned[::-1]

print(is_palindrome("A man a plan a canal Panama"))  # True
print(is_palindrome("hello"))  # False
```

## 6. 易错点

### 易错点 1：replace 不修改原字符串

错误现象：

```python
s = "hello"
s.replace("h", "H")
print(s)  # 还是 "hello"！
```

错误原因：字符串是不可变类型，`replace` 返回新字符串。

正确做法：`s = s.replace("h", "H")`

### 易错点 2：split 和 join 方向不要搞反

```python
# split：字符串 → 列表
"a,b,c".split(",")         # ['a', 'b', 'c']

# join：列表 → 字符串，调用者是分隔符！
",".join(['a', 'b', 'c'])  # 'a,b,c'
# ['a', 'b', 'c'].join(",")  # 错误！
```

### 易错点 3：`+` 拼接大量字符串效率低

正确做法：大量拼接时使用 `"".join(list)`。

### 练习 2

题目：

用 f-string 格式化输出：姓名张三，语文92.5，数学88.0，英语95.5，平均分保留两位小数。

答案：

```python
name = "张三"
chinese, math, english = 92.5, 88.0, 95.5
average = (chinese + math + english) / 3
print(f"姓名：{name}，平均分：{average:.2f}")  # 平均分：92.00
```

### 练习 3

题目：

有一串用户输入 `"  Hello,   World!  "`，去掉首尾空格，将多个连续空格合并为一个，转为大写。

答案：

```python
s = "  Hello,   World!  "
s = s.strip()          # 去首尾空格
words = s.split()      # split 默认按任意空白分割
s = " ".join(words)    # 用单个空格重新连接
s = s.upper()          # 转大写
print(s)  # HELLO, WORLD!
```

## 8. 答题总结

- 字符串是**不可变**类型，所有"修改"操作都返回新字符串，必须接收返回值。
- 索引从 0 开始，`s[::-1]` 可反转字符串；切片不会越界，但单元素索引会。
- f-string（`f"{变量}"`）是最推荐的格式化方式，简洁直观。
- `split()` 和 `join()` 是最常用的两个字符串方法，方向相反。
- `strip()`（去空白）、`replace()`（替换）、`find()`（查找）、`startswith()/endswith()`（判断前后缀）。
- 大量拼接用 `join`，不用 `+`（每次 `+` 创建新对象，效率低）。
