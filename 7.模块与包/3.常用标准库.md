# 常用标准库

## 1. 知识点定义

Python 标准库是 Python 安装时自带的模块集合，不需要额外安装就能直接 `import` 使用。它们覆盖了系统交互、日期时间、数学运算、随机数、数据结构、迭代器等方方面面。

> 小白理解：
> Python 标准库就像买手机时自带的 App——电话、短信、相机这些基础功能都预装好了，不用去应用商店下载。标准库就是 Python "预装"的工具箱，打开就能用。

它主要解决的问题是：
- 无需安装第三方库就能完成大部分常见编程任务
- 提供经过充分测试的可靠工具
- 统一了常见的编程接口（跨平台兼容）

常用的标准库模块包括：`os`、`sys`、`datetime`、`time`、`math`、`random`、`collections`、`itertools`、`functools`。

## 2. 语法格式

### 2.1 os 和 sys —— 系统交互

```python
import os
import sys

# === os 模块 ===
os.getcwd()                        # 获取当前工作目录
os.path.join('folder', 'file.txt') # 跨平台拼接路径
os.path.exists('/etc/hosts')       # 判断路径是否存在
os.path.isfile('/etc/hosts')       # 判断是否为文件
os.path.isdir('/etc')              # 判断是否为目录
os.listdir('.')                    # 列出当前目录内容
os.makedirs('a/b/c')               # 递归创建多层目录

# === sys 模块 ===
sys.argv                           # 命令行参数列表
sys.path                           # 模块搜索路径
sys.version                        # Python 版本信息
sys.exit(0)                        # 退出程序，0=正常退出
```

### 2.2 datetime 和 time —— 日期时间

```python
from datetime import datetime, timedelta
import time

# === datetime ===
now = datetime.now()                     # 当前日期时间
now.strftime("%Y-%m-%d %H:%M:%S")        # 格式化输出: "2026-06-09 14:30:00"
datetime.strptime("2026-06-09", "%Y-%m-%d") # 字符串解析为 datetime

tomorrow = now + timedelta(days=1)       # 日期加减
yesterday = now - timedelta(days=1)

# === time ===
time.time()                              # 当前时间戳（秒）
time.sleep(2)                            # 休眠 2 秒
```

### 2.4 random —— 随机数

```python
import random

random.randint(1, 100)       # [1, 100] 随机整数
random.choice(['A', 'B'])    # 随机选一个元素
random.sample(range(100), 5) # 随机选 5 个不重复的
random.shuffle(my_list)      # 原地打乱列表
random.seed(42)              # 固定随机种子，让结果可复现
```

### 2.5 collections —— 增强容器

```python
from collections import Counter, defaultdict, namedtuple, deque

# Counter：统计元素出现次数
c = Counter(['a', 'b', 'a', 'c', 'a'])
c.most_common(2)             # [('a', 3), ('b', 1)]

# defaultdict：带默认值的字典
dd = defaultdict(list)       # 访问不存在的 key 自动创建空列表
dd['fruits'].append('苹果')

# namedtuple：具名元组（像轻量级类）
Point = namedtuple('Point', ['x', 'y'])
p = Point(10, 20)
print(p.x, p.y)              # 10 20

# deque：双端队列（两端操作都是 O(1)）
dq = deque([1, 2, 3])
dq.appendleft(0)             # 左侧添加
dq.pop()                     # 右侧弹出
```

### 2.6 itertools —— 高效迭代器

```python
import itertools

# 排列组合
itertools.permutations('ABC', 2)    # 排列: AB, AC, BA, BC, CA, CB
itertools.combinations('ABC', 2)    # 组合: AB, AC, BC
itertools.product('AB', '12')       # 笛卡尔积: A1, A2, B1, B2

# 串联和切片
itertools.chain([1, 2], [3, 4])     # [1, 2, 3, 4]
itertools.islice(range(100), 5, 15) # 惰性切片

# 无限迭代器（需要 break 控制）
itertools.count(10, 2)              # 10, 12, 14, ...
itertools.cycle('ABC')              # A, B, C, A, B, C, ...
```

### 2.7 functools —— 高阶函数工具

```python
from functools import partial, lru_cache, wraps, reduce

# partial：固定部分参数
square = partial(pow, exp=2)
square(5)                    # 25

# lru_cache：缓存函数结果（记忆化）
@lru_cache(maxsize=128)
def fib(n):
    if n < 2: return n
    return fib(n-1) + fib(n-2)

# reduce：累积归约
reduce(lambda x, y: x + y, [1, 2, 3, 4, 5])  # 15
```

## 3. 重点说明

### 3.1 os 和 sys 的区别

| 模块 | 用途 | 典型场景 |
|------|------|---------|
| `os` | 与**操作系统**交互 | 文件路径、目录操作、环境变量 |
| `sys` | 与 **Python 解释器**交互 | 命令行参数、模块搜索路径、退出程序 |

### 3.2 datetime 和 time 的区别

| 模块 | 用途 | 典型函数 |
|------|------|---------|
| `datetime` | 日期时间的**表示和运算** | `datetime.now()`, `timedelta`, `strftime` |
| `time` | 时间戳、**休眠**、性能计时 | `time.time()`, `time.sleep()`, `time.perf_counter()` |

### 3.3 常用模块速查表

| 模块 | 核心用途 | 必会函数 |
|------|---------|---------|
| `os` | 系统交互、路径操作 | `getcwd()`, `path.join()`, `path.exists()`, `listdir()` |
| `sys` | 解释器交互 | `argv`, `path`, `exit()` |
| `datetime` | 日期时间处理 | `now()`, `strftime()`, `strptime()`, `timedelta()` |
| `time` | 时间戳、休眠 | `time()`, `sleep()`, `perf_counter()` |
| `math` | 数学运算 | `sqrt()`, `pi`, `ceil()`, `floor()`, `factorial()` |
| `random` | 随机数 | `randint()`, `choice()`, `sample()`, `shuffle()` |
| `collections` | 增强数据结构 | `Counter`, `defaultdict`, `namedtuple`, `deque` |
| `itertools` | 迭代器工具 | `permutations`, `combinations`, `chain`, `count` |
| `functools` | 高阶函数 | `partial`, `lru_cache`, `wraps`, `reduce` |

## 5. 典型例题（附带答案）

### 例题 1

**题目：**

用 `sys.argv` 实现一个命令行计算器，支持以下用法：

```bash
$ python calc.py add 10 5    # 输出: 15
$ python calc.py mul 6 7     # 输出: 42
```

**答案：**

```python
import sys

def main():
    if len(sys.argv) != 4:
        print("用法: python calc.py <操作> <数字1> <数字2>")
        print("支持操作: add, sub, mul, div")
        sys.exit(1)

    op = sys.argv[1]
    a = float(sys.argv[2])
    b = float(sys.argv[3])

    ops = {
        'add': lambda x, y: x + y,
        'sub': lambda x, y: x - y,
        'mul': lambda x, y: x * y,
        'div': lambda x, y: x / y if y != 0 else '错误: 除数不能为0',
    }

    result = ops.get(op, lambda x, y: f"不支持的操作: {op}")(a, b)
    print(result)

if __name__ == '__main__':
    main()
```

**解析：** `sys.argv` 是一个列表，`sys.argv[0]` 是脚本名，`sys.argv[1]` 开始是用户传的参数。使用前应先检查 `len(sys.argv)`，避免索引越界。

### 例题 2

**题目：**

用 `Counter` 统计一段文本中每个单词的出现次数，输出频率最高的 5 个单词。再用 `defaultdict` 实现一个按首字母分组的词表。

**答案：**

```python
from collections import Counter, defaultdict

text = """
Python is an interpreted high-level programming language
Python is dynamically typed and garbage-collected
Python supports multiple programming paradigms
including structured object-oriented and functional programming
""".lower()

# Counter 统计词频
words = text.split()
word_count = Counter(words)

print("Top 5 词汇:")
for word, count in word_count.most_common(5):
    print(f"  {word}: {count}")

# defaultdict 按首字母分组
grouped = defaultdict(list)
for word in set(words):            # set 去重
    grouped[word[0]].append(word)

print("\n按首字母分组:")
for letter in sorted(grouped):
    print(f"  {letter}: {', '.join(sorted(grouped[letter]))}")
```

**解析：** `Counter` 的 `most_common(n)` 方法直接返回频率最高的 n 个元素。`defaultdict(list)` 访问不存在的键时自动创建空列表。

## 6. 易错点

### 易错点 1：sys.argv 索引越界

错误写法：
```python
import sys
name = sys.argv[1]   # 用户没传参数时 IndexError!
```

错误原因：用户可能不传参数，直接访问 `sys.argv[1]` 会越界。

正确写法：
```python
import sys
if len(sys.argv) > 1:
    name = sys.argv[1]
else:
    print("请提供一个参数")
```

### 易错点 2：datetime.strptime 格式不匹配

错误写法：
```python
from datetime import datetime
dt = datetime.strptime("2026-05-07", "%Y/%m/%d")  # 格式不匹配！
```

错误原因：字符串用 `-` 分隔，但格式写的是 `/`。

正确写法：
```python
dt = datetime.strptime("2026-05-07", "%Y-%m-%d")  # 格式与字符串一致
```

### 易错点 3：random.seed() 放错位置

错误写法：
```python
random.seed(42)
print(random.randint(1, 100))
random.seed(42)          # 每次调用前都 reset seed → 每次同一个数！
print(random.randint(1, 100))
```

错误原因：`seed()` 在程序开头调用一次即可，不需要每次随机前都设置。

正确写法：
```python
random.seed(42)          # 只在开头调用一次
print(random.randint(1, 100))  # 82
print(random.randint(1, 100))  # 另一个随机数
```

### 易错点 4：lru_cache 使用可变参数

错误写法：
```python
from functools import lru_cache

@lru_cache
def process(data):
    return sum(data)

process([1, 2, 3])  # TypeError: unhashable type: 'list'
```

错误原因：`lru_cache` 要求所有参数可哈希，列表、字典不可哈希。

正确写法：
```python
@lru_cache
def process(data):
    return sum(data)

process(tuple([1, 2, 3]))  # 转为元组（不可变，可哈希）
```

### 易错点 5：itertools.groupby 不先排序

错误写法：
```python
import itertools
data = [('b', 3), ('a', 1), ('b', 4), ('a', 2)]
for key, group in itertools.groupby(data, key=lambda x: x[0]):
    print(key, list(group))
# 输出两组的 'a' 和两组的 'b'，没有正确合并！
```

错误原因：`groupby` 只合并相邻的相同 key，不排序会导致相同 key 分散在多组。

正确写法：
```python
data.sort(key=lambda x: x[0])   # 先排序
for key, group in itertools.groupby(data, key=lambda x: x[0]):
    print(key, list(group))     # 现在正确合并了
```

## 7. 课后练习（附带答案）

### 练习 1

**题目：**

写一个函数 `days_until(target_date_str)`，接收格式为 `"2026-12-25"` 的字符串，返回距离该日期还有多少天。日期已过则返回负数。再写一个函数计算两个日期间有多少个工作日（周一至周五）。

**答案：**

```python
from datetime import datetime, timedelta

def days_until(target_date_str):
    """计算距离目标日期还有多少天"""
    target = datetime.strptime(target_date_str, "%Y-%m-%d")
    now = datetime.now()
    return (target - now).days

def workdays_between(date_str1, date_str2):
    """计算两个日期间的工作日数（周一至周五）"""
    d1 = datetime.strptime(date_str1, "%Y-%m-%d")
    d2 = datetime.strptime(date_str2, "%Y-%m-%d")
    if d1 > d2:
        d1, d2 = d2, d1              # 确保 d1 <= d2

    count = 0
    current = d1
    while current <= d2:
        if current.weekday() < 5:    # 0-4 是周一到周五
            count += 1
        current += timedelta(days=1)
    return count

print(f"距离元旦还有 {days_until('2027-01-01')} 天")
print(f"工作日数: {workdays_between('2026-06-01', '2026-06-07')}")
```

**说明：** `weekday()` 返回 0（周一）到 6（周日），工作日判断条件是 `< 5`。

### 练习 2

**题目：**

创建一个列表 `prizes = ['一等奖', '二等奖', '三等奖', '参与奖', '参与奖', '参与奖']`，有 6 个奖品但只有 5 个人。用 `random.sample` 随机抽出 5 个不同的人获奖，随机分配奖品。用 `random.seed(42)` 固定种子验证结果可复现。

**答案：**

```python
import random

prizes = ['一等奖', '二等奖', '三等奖', '参与奖', '参与奖', '参与奖']
names = ['张三', '李四', '王五', '赵六', '钱七']

random.shuffle(prizes)           # 洗牌奖品

```

**说明：** `random.seed(42)` 让随机序列固定，每次运行结果一致，方便调试和验证。

### 练习 3

**题目：**

用 `@lru_cache` 装饰 `fibonacci(n)` 函数，对比缓存前后计算 `fibonacci(35)` 的耗时差异。

**答案：**

```python
import time
from functools import lru_cache

# 无缓存版本
def fib_no_cache(n):
    if n < 2:
        return n
    return fib_no_cache(n - 1) + fib_no_cache(n - 2)

# 有缓存版本
@lru_cache(maxsize=None)          # None 表示不限制缓存大小
def fib_with_cache(n):
    if n < 2:
        return n
    return fib_with_cache(n - 1) + fib_with_cache(n - 2)

# 对比耗时
start = time.perf_counter()
fib_no_cache(35)
print(f"无缓存耗时: {time.perf_counter() - start:.4f} 秒")

start = time.perf_counter()
fib_with_cache(35)
print(f"有缓存耗时: {time.perf_counter() - start:.6f} 秒")

print(f"缓存信息: {fib_with_cache.cache_info()}")
# 有缓存比无缓存快几个数量级！
```

**说明：** 无缓存时 `fib(35)` 需要递归调用约 3000 万次，有缓存后只需要约 36 次，性能天差地别。

## 8. 答题总结

- Python 标准库是自带的模块集合，无需安装，`import` 后直接用。
- `os` 用于操作系统交互（路径、目录），`sys` 用于解释器交互（命令行参数、退出）。
- `datetime` 处理日期时间的表示和运算，`time` 处理时间戳、休眠和性能计时。
- `math` 提供数学常量和函数，`random` 提供随机数生成。
- `collections` 提供增强数据结构：`Counter`（计数器）、`defaultdict`（默认字典）、`namedtuple`（具名元组）、`deque`（双端队列）。
- `itertools` 提供高效迭代器：排列组合、chain、islice、无限迭代器等。
- `functools` 提供高阶函数工具：`partial`（偏函数）、`lru_cache`（缓存）、`reduce`（归约）。
- `lru_cache` 要求参数可哈希，`groupby` 需要先排序，`seed()` 只需调用一次。
