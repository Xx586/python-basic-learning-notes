# input 输入

## 1. 知识点定义

`input` 是 Python 中用于**从键盘获取用户输入**的函数。当程序运行到 `input()` 时，会停下来等待用户在键盘上输入内容并按回车键，然后把用户输入的内容作为**字符串**返回给程序。

可以这样理解：如果 `print` 是程序对你"说话"，那 `input` 就是你对着程序"说话"。它让程序不再是单向输出，而是可以和用户进行互动。

它主要解决的问题是：
- 让程序能接收用户的数据（用户名、密码、数字、选项等）
- 实现交互式程序（问答、计算器、菜单选择等）
- 让同一份代码可以根据不同输入产生不同结果

在实际写 Python 代码时，`input` 常用于编写命令行交互程序、收集用户数据、实现菜单导航等。

> 小白理解：
> 想象你在餐厅点菜，`input` 就是服务员问"您想吃什么？"，然后等你回答。你告诉服务员后，服务员把你的回答带回厨房（程序）继续处理。

---

## 2. 语法格式

```python
# 基本语法：等待用户输入，把输入内容保存到变量中
变量名 = input("提示文字")
```

说明：
- `input()` 括号里可以写一段提示文字（字符串），告诉用户该输入什么。
- 用户输入的内容会以**字符串（str）** 类型返回。
- 返回的结果必须用变量接住，否则输入的内容就丢失了。

```python
# 最简单的用法：不写提示文字
name = input()  # 程序停住等你输入，输入的 <内容></内容>保存到 name 变量

# 推荐用法：加上提示文字
name = input("请输入你的名字：")  # 屏幕上会显示"请输入你的名字："，然后等你打字

# 如果需要一个整数，需要另外转换
age = input("请输入你的年龄：")   # age 是字符串类型，比如 "18"
age = int(age)                    # 转换成整数 18
# 或者一步到位：
age = int(input("请输入你的年龄："))
```

## 3. 重点说明

### 3.1 input 的返回值永远是字符串

这是初学者最容易忘记的一点。不管用户输入的是数字、字母还是标点符号，`input` 函数返回的值**一定是 `str` 类型**。

```python
num = input("请输入一个数字：")  # 用户输入 100
print(type(num))                 # 输出：<class 'str'>，是字符串 "100" 不是数字 100
```

如果你要做数学计算，必须先转换类型：

```python
num = int(input("请输入一个数字："))   # 输入后立刻转成整数
price = float(input("请输入价格："))   # 输入后立刻转成小数
```

### 3.2 input 和 print 配合使用

`input` 和 `print` 是最常见的搭档：

| 函数 | 方向 | 作用 |
|------|------|------|
| `print` | 程序 → 用户 | 把信息展示给用户看 |
| `input` | 用户 → 程序 | 把用户的输入交给程序处理 |

典型模式：先 `print` 提示 → 再 `input` 接收 → 处理数据 → 再 `print` 结果。

### 3.3 input 的提示信息

提示信息可以帮助用户知道要输入什么，是一个好的编程习惯：

```python
# 不好的写法：用户不知道该输入啥
x = input()

# 好的写法：有明确的提示
username = input("请输入用户名：")
password = input("请输入密码：")
```

### 3.4 什么时候用 input？

- **收集用户信息**：姓名、年龄、密码等
- **交互式计算器**：用户输入数字，程序计算结果
- **菜单选择**：用户输入选项编号，程序执行对应功能
- **问答测试**：程序出题，用户输入答案，程序判断对错

## 4. Demo 样例

### Demo 1：简单的问候程序

```python
# 获取用户输入的姓名
name = input("请输入你的名字：")  # 屏幕上显示提示，等待用户输入并回车

# 获取用户输入的年龄
age = input("请输入你的年龄：")   # 注意：input 返回的是字符串类型

# 用 print 把输入的内容显示出来
print("你好，", name, "！")       # 输出问候语
print("你今年", age, "岁。")       # age 还是字符串，直接输出没有问题

# 展示 age 的类型
print("age 变量的类型是：", type(age))  # 输出 <class 'str'> 证明是字符串
```

运行结果：

```text
请输入你的名字：小明
请输入你的年龄：18
你好， 小明 ！
你今年 18 岁。
age 变量的类型是： <class 'str'>
```

### Demo 2：简单的加法计算器

```python
# 要求用户输入两个数字，计算它们的和
print("===== 简单加法计算器 =====")

# 用 int() 把 input 返回的字符串转成整数
num1 = int(input("请输入第一个数字："))  # 输入后立刻转整数
num2 = int(input("请输入第二个数字："))  # 输入后立刻转整数

# 计算两个数的和
result = num1 + num2                    # 整数相加

# 输出结果
print(num1, "+", num2, "=", result)     # 用逗号隔开多个输出内容
# 也可以使用 sep 参数让输出更紧凑
print(num1, "+", num2, "=", result, sep="")
```

运行结果：

```text
===== 简单加法计算器 =====
请输入第一个数字：15
请输入第二个数字：27
15 + 27 = 42
15+27=42
```

> 注意：如果用户输入的不是纯数字（比如输入了字母），`int()` 转换会报错 `ValueError`。这在后面学异常处理时会讲到。

## 5. 典型例题（附带答案）

### 例题 1

题目：

编写程序，让用户输入姓名和年龄，然后程序输出一句话："XXX 同学，你明年就 XX 岁了。"（其中明年年龄 = 当前年龄 + 1）。

答案：

```python
name = input("请输入你的姓名：")
age = int(input("请输入你的年龄："))   # 一定要转成整数才能做加法
next_year_age = age + 1
print(name, "同学，你明年就", next_year_age, "岁了。")
```

解析：

关键点：`input` 返回的是字符串，必须用 `int()` 转成整数后才能进行 `+1` 的数学运算。否则 `"18" + 1` 会报错。

### 例题 2

题目：

编写程序，让用户输入语文、数学、英语三门课的成绩，计算总分和平均分并输出。

答案：

```python
chinese = float(input("请输入语文成绩："))
math = float(input("请输入数学成绩："))
english = float(input("请输入英语成绩："))

total = chinese + math + english      # 计算总分
average = total / 3                    # 计算平均分

print("总分：", total)
print("平均分：", average)
```

解析：

这里用 `float()` 而不是 `int()`，因为成绩可能带小数（比如 87.5）。`total / 3` 得到的是浮点数结果。

## 6. 易错点

### 易错点 1：忘记 input 返回的是字符串

错误写法：

```python
num = input("请输入数字：")  # 用户输入 10
print(num + 5)               # 报错！TypeError: can only concatenate str (not "int") to str
```

错误原因：`num` 是字符串 `"10"`，不能用 `+` 和整数 `5` 做加法。

正确写法：

```python
num = int(input("请输入数字："))  # 输入后立刻转整数
print(num + 5)                    # 输出 15
```

### 易错点 2：input 的结果没有用变量保存

错误写法：

```python
input("请输入你的名字：")  # 输入完就丢失了，程序拿不到
print("你好", ???)          # 没法引用输入的内容
```

错误原因：`input` 返回的值必须用一个变量去接，否则就没了。

正确写法：

```python
name = input("请输入你的名字：")  # 用 name 变量保存输入结果
print("你好", name)
```

### 易错点 3：提示文字中的拼写问题

错误写法：

```python
name = input（"请输入名字："）  # 中文括号，报语法错误
name = input("请输入名字：')    # 引号不匹配（前面双引号，后面单引号）
```

错误原因：Python 要求成对的引号类型必须一致，括号必须是英文半角。

正确写法：

```python
name = input("请输入名字：")   # 英文括号 + 成对的英文双引号
```

## 7. 课后练习（附带答案）

### 练习 1

题目：

编写程序，让用户输入自己的名字，然后程序输出："XXX，欢迎来到 Python 的世界！"

答案：

```python
name = input("请输入你的名字：")
print(name, "，欢迎来到 Python 的世界！")
```

说明：这是 input 最基本的用法——接收输入、保存到变量、用 print 输出。

---

### 练习 2

题目：

编写一个矩形面积计算器。让用户输入矩形的长和宽，计算并输出面积。

```text
请输入矩形的长：10
请输入矩形的宽：5
矩形的面积是：50
```

答案：

```python
length = float(input("请输入矩形的长："))
width = float(input("请输入矩形的宽："))
area = length * width
print("矩形的面积是：", area)
```

说明：这里用 `float()` 而不是 `int()`，因为用户可能输入带小数的长和宽。

---

### 练习 3

题目：

编写程序，让用户输入自己的出生年份，然后程序计算并输出该用户 2026 年时的年龄。

答案：

```python
birth_year = int(input("请输入你的出生年份："))
age = 2026 - birth_year
print("你在2026年的年龄是：", age, "岁")
```

说明：`input` 返回字符串，必须用 `int()` 转换后才能做减法运算。

## 8. 答题总结

- `input` 用于从键盘接收用户输入，程序会停下来等待用户输入并按回车。
- `input` 的返回值**永远是字符串类型**，要做数学计算必须先用 `int()` 或 `float()` 转换。
- `input("提示文字")` 中的提示文字让用户知道该输入什么，是好习惯。
- `input` 的返回值必须用变量保存，否则输入就丢失了。
- 常见错误：忘记类型转换、提示用了中文标点、引号不匹配、没有变量保存结果。
