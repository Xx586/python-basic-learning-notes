# 变量初识

## 1. 知识点定义

**变量**是程序中用来**保存数据**的一个"命名空间"。你可以把它想象成给数据贴了一个标签，通过标签就能找到并使用这个数据。

它主要解决的问题是：
- 让程序能记住数据（用户输入、计算结果、中间状态等）
- 让代码可以通过变量名反复使用同一个数据，而不需要重复书写
- 让程序更灵活——修改变量的值就能改变程序的行为

在实际写 Python 代码时，变量无处不在。从保存用户姓名到存储计算结果，没有变量程序几乎什么也做不了。

> 小白理解：
> 变量就像一个带名字的收纳盒。你可以把数据（数字、文字等）放进盒子里，给盒子贴上标签（变量名）。以后要用的时候，喊标签名字就能拿到盒子里的东西。而且你可以随时更换盒子里的东西。

---

## 2. 语法格式

```python
# 基本语法：变量名 = 值
变量名 = 值
```

说明：等号 `=` 在 Python 中叫**赋值运算符**，意思是把右边的值"放入"左边的变量中。它不是数学里的"等于"。

```python
# 定义变量并赋值
name = "小明"       # 把字符串"小明"保存到 name 变量中
age = 18            # 把整数 18 保存到 age 变量中
score = 95.5        # 把小数 95.5 保存到 score 变量中
is_passed = True    # 把布尔值 True 保存到 is_passed 变量中

# 使用变量：通过变量名访问保存的数据
print(name)         # 输出：小明
print(age)          # 输出：18
print(score)        # 输出：95.5

# 修改变量的值：重新赋值即可
age = 19            # age 原来的值是 18，现在改成 19
print(age)          # 输出：19

# 变量的值可以参与计算
a = 10
b = 20
c = a + b           # c 的值为 30
print(c)            # 输出：30

# 多个变量同时赋值
x, y, z = 1, 2, 3   # x=1, y=2, z=3
```

## 3. 重点说明

### 3.1 Python 变量不需要声明类型

和 Java、C 语言不同，Python 定义变量时不需要指定数据类型，解释器会自动判断：

```python
# Java 需要这样写：int age = 18;
# Python 只需要：
age = 18  # Python 自动知道这是整数
```

这种特性称为**动态类型**，让 Python 代码写起来更快更灵活。

### 3.2 变量必须先赋值才能使用

如果你用了一个从未赋值的变量，Python 会报错 `NameError`：

```python
print(score)  # 报错！NameError: name 'score' is not defined
# 因为 score 变量还没被创建
```

### 3.3 变量名区分大小写

```python
name = "小明"
Name = "小红"
NAME = "小刚"

print(name)  # 输出：小明
print(Name)  # 输出：小红
print(NAME)  # 输出：小刚
# name、Name、NAME 是三个不同的变量
```

### 3.4 变量重新赋值会覆盖原来的值

```python
x = 100
print(x)  # 输出：100
x = 200   # 重新赋值
print(x)  # 输出：200，原来的 100 被覆盖了
```

### 3.5 变量的值之间可以互相传递

```python
a = 10
b = a      # b 得到 a 的值的副本
print(b)   # 输出：10
a = 20     # 修改 a，不会影响 b
print(b)   # 仍然输出：10
```

## 4. Demo 样例

### Demo 1：变量的基本操作

```python
# 定义不同类型的变量
student_name = "张三"       # 字符串类型变量，保存学生姓名
student_age = 20            # 整数类型变量，保存学生年龄
student_score = 87.5        # 浮点数类型变量，保存学生成绩
is_graduated = False        # 布尔类型变量，表示是否毕业

# 输出所有变量
print("学生信息：")
print("姓名：", student_name)
print("年龄：", student_age)
print("成绩：", student_score)
print("已毕业：", is_graduated)

# 修改变量值
student_age = 21            # 过了一年，年龄加一
student_score = 92.0        # 补考后成绩提高
print("\n更新后的信息：")     # \n 是换行符，在输出结果中空一行
print("年龄：", student_age)
print("成绩：", student_score)
```

运行结果：

```text
学生信息：
姓名： 张三
年龄： 20
成绩： 87.5
已毕业： False

更新后的信息：
年龄： 21
成绩： 92.0
```

### Demo 2：变量参与运算

```python
# 用变量做购物计算
item_name = "笔记本"        # 商品名称
unit_price = 25.5           # 单价
quantity = 3                # 数量

total = unit_price * quantity  # 用变量参与计算：总价 = 单价 × 数量

print("商品：", item_name)
print("单价：", unit_price, "元")
print("数量：", quantity)
print("总价：", total, "元")
```

运行结果：

```text
商品： 笔记本
单价： 25.5 元
数量： 3
总价： 76.5 元
```

## 5. 典型例题（附带答案）

### 例题 1

题目：

以下代码中，变量 `x` 最终的值是多少？

```python
x = 5
y = x
x = 10
```

选项：A. 5　　B. 10　　C. 报错

答案：

B. 10

解析：

- 第1行：`x` 被赋值为 5。
- 第2行：`y` 得到 `x` 当前的值（5），`x` 不受影响。
- 第3行：`x` 被重新赋值为 10，原来 5 被覆盖。
- 所以最终 `x` 是 10，`y` 是 5。

### 例题 2

题目：

请定义一个变量 `temperature` 保存今天的气温 26.5，再定义一个变量 `weather` 保存天气状况 "晴"，最后用 `print` 输出："今天的天气是晴，气温26.5℃"。

答案：

```python
temperature = 26.5
weather = "晴"
print("今天的天气是", weather, "，气温", temperature, "℃")
```

解析：

- 气温是小数，用 `float` 类型（直接写小数即可）。
- 天气是文字，用字符串类型（需要加引号）。
- 用逗号把多个内容拼接起来输出。

## 6. 易错点

### 易错点 1：使用未赋值的变量

错误写法：

```python
print(score)  # NameError: name 'score' is not defined
```

错误原因：`score` 变量从未被赋值过，Python 中变量必须先赋值才能使用。

正确写法：

```python
score = 85    # 先赋值
print(score)  # 再使用，输出 85
```

### 易错点 2：把变量名和字符串搞混

错误写法：

```python
name = "小明"
print("name")  # 输出：name（四个字母），不是"小明"
```

错误原因：加了引号就是字符串，不是变量。

正确写法：

```python
name = "小明"
print(name)    # 输出：小明
```

### 易错点 3：变量名拼写不一致

错误写法：

```python
student_name = "张三"
print(studentName)   # NameError，Python 是大小写敏感的
print(studen_name)   # NameError，少写了一个 t
```

错误原因：变量名必须前后完全一致，包括大小写。

正确写法：

```python
student_name = "张三"
print(student_name)  # 正确！拼写和大小写完全一致
```

### 易错点 4：中文变量名带来的潜在问题

Python 3 支持中文变量名，但**不建议使用**：

```python
年龄 = 18    # 能运行，但不推荐
print(年龄)  # 可能在某些环境下显示异常，也不符合国际编码规范
```

建议使用英文拼音或英文单词命名。

## 7. 课后练习（附带答案）

### 练习 1

题目：

定义三个变量，分别保存你的姓名、年龄和所在城市，然后用 `print` 一次性输出。

答案：

```python
name = "小明"
age = 20
city = "北京"
print("我叫", name, "，今年", age, "岁，来自", city)
```

说明：用逗号拼接变量和文字，这是最简单的输出方式。

### 练习 2

题目：

下面代码有错误，请找出来并改正：

```python
name = 小明
age = "18"
print(姓名)
```

答案：

错误有2处：
1. `小明` 没加引号，Python 会把它当成变量名，但不存在这个变量，应该改成 `"小明"`。
2. `print(姓名)` 中的 `姓名` 变量不存在，应该改成 `print(name)`。

正确代码：

```python
name = "小明"
age = "18"
print(name)
```

说明：字符串必须加引号，变量名必须和定义时完全一致。

### 练习 3

题目：

已知苹果每斤 8 元，香蕉每斤 5 元。小明买了 3 斤苹果和 2 斤香蕉。请用变量计算并输出他需要付多少钱。

答案：

```python
apple_price = 8       # 苹果单价
banana_price = 5      # 香蕉单价
apple_weight = 3      # 苹果重量
banana_weight = 2     # 香蕉重量

total = apple_price * apple_weight + banana_price * banana_weight
print("小明需要付", total, "元")
```

说明：用变量分别保存单价和重量，最后用乘法加法算出总价。

## 8. 答题总结

- 变量是用来保存数据的"容器"，通过变量名可以随时访问和修改数据。
- Python 定义变量不需要声明类型，`变量名 = 值` 即可。
- 变量必须先赋值再使用，否则报 `NameError`。
- 变量名区分大小写，`name`、`Name`、`NAME` 是三个不同的变量。
- 重新给变量赋值会覆盖原来的值。
- 常见错误：使用未定义变量、变量名拼写不一致、字符串忘加引号、变量名多加了引号。
