# 关键字

## 1. 知识点定义

**关键字**（Keyword）是 Python 语言**内置的、有特殊含义的保留词**。这些词已经被 Python 占用，用来定义语言的语法规则，比如 `if`、`for`、`def`、`class` 等。

它主要解决的问题是：
- 构成 Python 语法的基础——没有关键字，Python 代码就无法被正确理解
- 避免命名冲突——知道哪些词不能用作变量名，就不会写出奇怪的 bug

在实际写 Python 代码时，你每定义一个变量、一个函数、一个类，都要确保名字不是关键字。

> 小白理解：
> 关键字就像是"交通标志牌"。你在路上看到"停"的牌子，就知道必须停下来——这个"停"字已经被交通规则占用，你不能自己再定义"停"是什么意思。Python 的关键字也是同理，`if`、`for` 这些词已经被 Python 占用了，你不能用它们给变量起名。

---

## 2. 语法格式

Python 提供了一种方法可以查看当前版本有哪些关键字：

```python
# 查看 Python 的关键字列表
import keyword         # 导入关键字模块
print(keyword.kwlist)  # 打印所有关键字的列表
```

Python 3 中常见的 35 个关键字如下：

```text
False      await      else       import     pass
None       break      except     in         raise
True       class      finally    is         return
and        continue   for        lambda     try
as         def        from       nonlocal   while
assert     del        global     not        with
async      elif       if         or         yield
```

> 说明：`True`、`False`、`None` 在 Python 3 中也是关键字，不能用作变量名。

## 3. 重点说明

### 3.1 关键字不能用作变量名

这是最重要的规则：

```python
# 错误写法！
if = 10        # 报错！if 是关键字
class = "一班"  # 报错！class 是关键字
for = 5        # 报错！for 是关键字
```

### 3.2 常见关键字分类

按用途对关键字进行分类，更容易理解和记忆：

| 分类 | 关键字 | 用途 |
|------|--------|------|
| 条件判断 | `if`, `elif`, `else` | 条件分支 |
| 循环 | `for`, `while`, `break`, `continue` | 循环控制 |
| 函数 | `def`, `return`, `lambda`, `yield` | 函数定义 |
| 类 | `class` | 类定义 |
| 逻辑运算 | `and`, `or`, `not` | 逻辑运算 |
| 变量作用域 | `global`, `nonlocal` | 作用域声明 |
| 异常处理 | `try`, `except`, `finally`, `raise`, `assert` | 异常处理 |
| 导入模块 | `import`, `from`, `as` | 模块导入 |
| 上下文管理 | `with` | with 语句 |
| 特殊值 | `True`, `False`, `None` | 布尔值和空值 |
| 其他 | `in`, `is`, `del`, `pass`, `async`, `await` | 其他语法 |

### 3.3 如何判断一个词是不是关键字？

最简单的办法是在 Python 中测试：

```python
import keyword

print(keyword.iskeyword("if"))     # True，是关键字
print(keyword.iskeyword("name"))   # False，不是关键字，可以用作变量名
print(keyword.iskeyword("class"))  # True，是关键字
print(keyword.iskeyword("score"))  # False，不是关键字
```

### 3.4 容易误用的关键字

初学者最容易不小心用作变量名的关键字：

| 关键字 | 为什么容易误用 | 替代命名建议 |
|--------|---------------|-------------|
| `list` | 想给一个列表起名 | `my_list`、`items`、`data_list` |
| `dict` | 想给一个字典起名 | `my_dict`、`info_dict`、`data` |
| `str` | 想给一个字符串起名 | `text`、`my_str`、`content` |
| `int` | 想给一个整数起名 | `num`、`count`、`number` |
| `sum` | 想给总和起名 | `total`、`sum_result` |
| `type` | 想给类型起名 | `data_type`、`kind` |
| `id` | 想给编号起名 | `user_id`、`stu_id`、`item_id` |
| `class` | 想给班级起名 | `class_name`、`cls_name` |

> `list`、`dict`、`str`、`int`、`sum`、`type`、`id` 这些虽然是 Python 内置函数/类型名，严格说不算关键字，但用作变量名会覆盖（遮蔽）原有功能，造成后续代码出错，所以同样**不建议用作变量名**。

## 4. Demo 样例

### Demo 1：输出所有关键字

```python
import keyword  # 导入 keyword 模块，这个模块用来处理关键字相关功能

# 获取所有关键字的列表
all_keywords = keyword.kwlist  # kwlist 是 keyword 模块提供的关键字列表

print("Python 3 的全部关键字（共", len(all_keywords), "个）：")
print(all_keywords)  # 输出完整列表
```

运行结果：

```text
Python 3 的全部关键字（共 35 个）：
['False', 'None', 'True', 'and', 'as', 'assert', 'async', 'await', 'break', 'class', 'continue', 'def', 'del', 'elif', 'else', 'except', 'finally', 'for', 'from', 'global', 'if', 'import', 'in', 'is', 'lambda', 'nonlocal', 'not', 'or', 'pass', 'raise', 'return', 'try', 'while', 'with', 'yield']
```

### Demo 2：判断一个词是不是关键字

```python
import keyword  # 导入关键字模块

# 测试几个词是否是关键字
test_words = ["if", "name", "class", "score", "True", "hello"]

for word in test_words:  # 遍历列表中的每一个词
    if keyword.iskeyword(word):  # iskeyword 函数判断一个词是不是关键字
        print(word, "→ 是关键字，不能用作变量名")
    else:
        print(word, "→ 不是关键字，可以用作变量名")
```

运行结果：

```text
if → 是关键字，不能用作变量名
name → 不是关键字，可以用作变量名
class → 是关键字，不能用作变量名
score → 不是关键字，可以用作变量名
True → 是关键字，不能用作变量名
hello → 不是关键字，可以用作变量名
```

## 5. 典型例题（附带答案）

### 例题 1

题目：

以下哪些是合法的 Python 变量名？哪些不是？分析原因。

```text
A. name
B. 2name
C. class
D. my_score
E. True
```

答案：

| 选项 | 是否合法 | 原因 |
|------|----------|------|
| A. `name` | ✅ 合法 | 普通英文单词，不是关键字 |
| B. `2name` | ❌ 不合法 | 数字不能作为变量名的开头 |
| C. `class` | ❌ 不合法 | `class` 是 Python 关键字 |
| D. `my_score` | ✅ 合法 | 字母 + 下划线，不是关键字 |
| E. `True` | ❌ 不合法 | `True` 是 Python 关键字 |

解析：

合法的变量名规则：
1. 只能包含字母、数字和下划线。
2. 不能以数字开头。
3. 不能是 Python 关键字。
4. 区分大小写。

### 例题 2

题目：

小明写了一段代码，运行时报错了。请找出错误并改正。

```python
class = "三年一班"
print(class)
```

答案：

错误原因：`class` 是 Python 中用于定义类的关键字，不能用作变量名。

正确代码：

```python
class_name = "三年一班"  # 改成 class_name 即可
print(class_name)
```

## 6. 易错点

### 易错点 1：用关键字做变量名

错误写法：

```python
if = 10          # SyntaxError: invalid syntax
class = "一班"    # SyntaxError: invalid syntax
```

错误原因：`if`、`class` 是 Python 关键字，有特殊用途，不能用作变量名。

正确写法：

```python
if_flag = 10        # 加上修饰词成为新名字
class_name = "一班"  # 加上后缀
```

### 易错点 2：用内置函数名做变量名

错误写法：

```python
list = [1, 2, 3]  # 虽然不会报错，但覆盖了内置的 list 函数
# 后面再使用 list() 就会出错
nums = list("hello")  # TypeError: 'list' object is not callable
```

Error原因：把内置函数 `list` 覆盖成了一个普通变量，后续代码中 `list()` 不能再用了。

正确写法：

```python
my_list = [1, 2, 3]  # 避免用内置函数名做变量名
```

### 易错点 3：混淆 True/False 和 true/false

错误写法：

```python
is_ok = true    # NameError: name 'true' is not defined
```

错误原因：Python 的布尔值是 `True` 和 `False`（首字母大写），不是 `true` 和 `false`。

正确写法：

```python
is_ok = True    # 正确，首字母大写
```

## 8. 答题总结

- Python 关键字是被语言占用的保留词，有特殊含义，不能用作变量名。
- Python 3 有 35 个关键字，常见的有 `if`、`for`、`while`、`def`、`class`、`True`、`False`、`None` 等。
- 可以用 `import keyword` 后调用 `keyword.kwlist` 和 `keyword.iskeyword()` 来查看和判断。
- 除了关键字，Python 内置函数名（如 `list`、`dict`、`sum`）也不建议用作变量名。
- 变量名规则：只能包含字母、数字、下划线，不能以数字开头，不能是关键字。
