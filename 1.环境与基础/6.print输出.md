# print 输出

## 1. 知识点定义

`print` 是 Python 中最常用的**输出函数**，它的作用是把文字、数字、变量等内容显示到屏幕上，让我们能看到程序运行的结果。

可以这样理解：程序在计算机内部做了很多计算，但如果不输出，我们什么也看不到。`print` 就像是程序和我们之间的"对话窗口"，程序通过 `print` 把结果告诉我们。

它主要解决的问题是：
- 让程序运行结果可见
- 帮我们调试代码（输出中间变量，看运行到哪一步了）
- 给用户显示提示信息和最终结果

在实际写 Python 代码时，`print` 几乎无处不在——从最简单的 "Hello World" 到复杂程序的日志输出，都离不开它。

> 小白理解：
> 把 `print` 想象成程序在对你"喊话"。程序算完一道数学题，通过 `print` 把答案告诉坐在屏幕前的你。

---

## 2. 语法格式

```python
# 基本语法：输出一个或多个内容到屏幕
print(内容)
```

说明：`print` 后面跟一对小括号 `()`，括号里放你想输出的内容。

```python
# 输出一段文字（字符串）
print("Hello World")

# 输出数字
print(100)

# 输出变量
name = "小明"
print(name)

# 一次输出多个内容，用逗号隔开
print("我叫", "小明", "今年", 18, "岁")

# sep 参数：多个内容之间的分隔符，默认是空格
print("2024", "06", "09", sep="-")  # 输出: 2024-06-09

# end 参数：输出结束后追加的内容，默认是换行
print("第一行", end="---")
print("第二行")  # 输出: 第一行---第二行
```

---

## 3. 重点说明

### 3.1 什么时候用 print？

- **显示计算结果**：算完一道题，把答案输出到屏幕。
- **程序运行提示**：告诉用户"正在处理中，请稍候"。
- **调试 bug**：在代码关键位置插入 `print`，看看变量值对不对、程序有没有走到这一步。
- **格式化输出**：把数据按照特定格式输出，比如表格、菜单、成绩单等。

### 3.2 print 输出不同类型的区别

| 内容类型 | 写法 | 输出结果 | 注意 |
|----------|------|----------|------|
| 字符串 | `print("hello")` | `hello` | 需要加引号 |
| 数字 | `print(100)` | `100` | 不需要加引号 |
| 变量 | `print(name)` | 变量的值 | 变量名不要加引号 |
| 多个内容 | `print("a", 1)` | `a 1` | 自动用空格隔开 |

### 3.3 初学者最容易混淆的地方

- **引号要不要加**：输出文字必须加引号，输出变量名不能加引号。
- **逗号和中英文**：Python 代码中的逗号、括号都必须是英文符号。
- **print 的拼写**：是 `print` 不是 `pirnt` 或 `prnt`。

### 3.4 sep 和 end 参数

| 参数 | 作用 | 默认值 | 示例 |
|------|------|--------|------|
| `sep` | 多个输出内容之间的分隔符 | 空格 `" "` | `print(1,2,3, sep=",")` → `1,2,3` |
| `end` | 输出结束后的追加内容 | 换行 `"\n"` | `print("hi", end="!")` → `hi!` |

## 4. Demo 样例

### Demo 1：基本输出演示

```python
# 输出不同类型的值
print("===== 基本输出演示 =====")  # 输出一个标题
print("Hello Python!")            # 输出字符串，引号内的文字会原样打印
print(2024)                       # 输出整数，数字不需要加引号
print(3.14)                       # 输出小数（浮点数）
print(True)                       # 输出布尔值

# 定义变量，并用 print 输出变量
student_name = "张三"             # 把名字保存在变量里
student_age = 20                  # 把年龄保存在变量里
print(student_name)               # 输出变量值，打印的是"张三"
print(student_age)                # 输出变量值，打印的是 20

# 一次输出多个内容，用逗号隔开
print("姓名：", student_name, "，年龄：", student_age, "岁")
# 输出：姓名： 张三 ，年龄： 20 岁（注意逗号会自动变成空格）
```

运行结果：

```text
===== 基本输出演示 =====
Hello Python!
2024
3.14
True
张三
20
姓名： 张三 ，年龄： 20 岁
```

### Demo 2：sep 和 end 参数演示

```python
# sep 参数：指定多个内容之间的分隔符
print("2024", "06", "09", sep="-")   # 用短横线分隔，输出日期格式
print("张三", "李四", "王五", sep="、")  # 用顿号分隔，输出名单
print("apple", "banana", "orange", sep=" | ")  # 用竖线分隔

# end 参数：指定输出结束后的内容，默认是换行
print("第一句", end="，")   # 结尾加逗号，不换行
print("第二句", end="。")   # 结尾加句号，不换行
print()                      # 输出一个空行（换行）
print("上面有个空行")

# sep 和 end 可以一起使用
print("A", "B", "C", sep="→", end=" 结束")
print()  # 换行
```

运行结果：

```text
2024-06-09
张三、李四、王五
apple | banana | orange
第一句，第二句。

上面有个空行
A→B→C 结束
```

> 小技巧：`print()` 括号里什么都不写，就输出一个空行，常用来让输出更美观。

## 5. 典型例题（附带答案）

### 例题 1

题目：

请用 `print` 函数在屏幕上输出以下内容（要求一次调用 `print` 完成）：

```text
我叫小明，今年18岁，来自北京。
```

答案：

```python
name = "小明"
age = 18
city = "北京"
print("我叫", name, "，今年", age, "岁，来自", city, "。")
```

解析：

先用变量保存姓名、年龄、城市三个信息，然后用 `print` 一次性输出。变量名不要加引号，文字内容需要加引号用逗号隔开。

### 例题 2

题目：

请用 `print` 的 `sep` 和 `end` 参数，输出下面的内容：

```text
2024-01-15
2024-01-16
2024-01-17
```

要求使用 `sep="-"` 和 `end="\n"`。

答案：

```python
# 用 sep="-" 把年月日连接起来，end 默认就是换行所以可以不写
print("2024", "01", "15", sep="-")
print("2024", "01", "16", sep="-")
print("2024", "01", "17", sep="-")
```

解析：

`sep="-"` 让三个部分之间用短横线连接。`end` 默认就是换行符 `"\n"`，所以每个 `print` 结束后自动换行。

## 6. 易错点

### 易错点 1：输出变量时加了引号

错误写法：

```python
name = "小明"
print("name")  # 输出的是 name 这4个字母，不是小明
```

错误原因：加引号后 Python 把它当成字符串文字，而不是变量名。

正确写法：

```python
name = "小明"
print(name)  # 输出：小明
```

### 易错点 2：输出文字时忘了加引号

错误写法：

```python
print(Hello)  # 报错：NameError: name 'Hello' is not defined
```

错误原因：Python 把 `Hello` 当成变量名，但这个变量没有定义过。

正确写法：

```python
print("Hello")  # 输出：Hello
```

### 易错点 3：print 拼写错误

错误写法：

```python
pirnt("hello")  # 报错：NameError: name 'pirnt' is not defined
Prnt("hello")   # 报错：NameError: name 'Prnt' is not defined
PRINT("hello")  # 报错：NameError: name 'PRINT' is not defined
```

错误原因：Python 是大小写敏感的语言，`print` 必须全部小写，而且拼写必须正确。

正确写法：

```python
print("hello")  # 输出：hello
```

### 易错点 4：用了中文标点符号

错误写法：

```python
print（"你好"）    # 括号是中文全角括号
print("你好"，"世界")  # 逗号是中文逗号
```

错误原因：Python 只认识英文标点符号。中文括号、中文逗号都会报语法错误。

正确写法：

```python
print("你好")         # 英文括号
print("你好", "世界")  # 英文逗号
```

### 练习 2

题目：

请使用 `sep` 参数，让三个 `print` 输出以下内容：

```text
苹果-香蕉-橘子
苹果-香蕉-橘子
苹果-香蕉-橘子
```

答案：

```python
print("苹果", "香蕉", "橘子", sep="-")
print("苹果", "香蕉", "橘子", sep="-")
print("苹果", "香蕉", "橘子", sep="-")
```

说明：`sep="-"` 让每个 `print` 中的三样水果之间用短横线隔开。

---

### 练习 3

题目：

请使用 `end` 参数，用三个 `print` 分别输出 "我"、"爱"、"Python"，让它们显示在同一行，中间没有空格，最后输出一个感叹号。

预期输出：

```text
我爱Python！
```

答案：

```python
print("我", end="")    # end="" 表示输出后不换行，也不加任何字符
print("爱", end="")    # 继续不换行
print("Python", end="！")  # 结尾加感叹号
```

说明：`end=""` 设为空字符串，就可以让多个 `print` 的输出连在一行。

## 8. 答题总结

- `print` 是 Python 最常用的输出函数，用于在屏幕上显示内容。
- 输出文字（字符串）必须加引号，输出变量不加引号，输出数字不加引号。
- 多个内容用逗号隔开，`sep` 可自定义分隔符，`end` 可自定义结尾。
- 初学者最常见错误：print 拼写错、中文标点、输出变量时多加了引号、大小写写错。
- `print()` 是调试代码最朴素有效的方法——在代码中途插入 `print(变量名)` 就能看到变量当前的值。
